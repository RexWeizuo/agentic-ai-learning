# 背词出题规范

## 出题格式

每个词只给：**单词 + 提示**（词根拆解/例句/搭配）

## 禁止事项

1. **禁止 AskUserQuestion 多选选项** — 选项直接泄露释义，用户不用回忆。该 tool 在背词场景完全禁用
2. **禁止在提示里包含中文释义** — 词根拆解只拆前缀+词根+后缀，不加"→ 意思"
3. **提示不能包含答案** — 例句里不能直接暗示中文意思
4. **重测也不能用选项** — 即使是口头巩固，也不能用选项让用户选

## 正确做法

- 词根拆解：只给 "前缀(X)+词根(Y)+后缀(Z)"，不加箭头和结论
- 例句：给英文句子，不给中文翻译
- 搭配：给英文搭配，如 "abound ___"（留空）

## 评级与纠错

5. **答错自动降级** — 用户说对了但给了高分→不管；用户明显说错了→自动按错误程度评级（明显错=1，接近=2），不要原样保留过高评级。
6. **不会的词给例句** — 评级 1 或用户明确说"不会"的词，下一轮重测前先给出英文例句帮回忆。
7. **基础词快速过** — 像 teacher, earth, knowledge 这种直接跳过不深究，评级 4/5 即可。

## 挑词流程

- 每次新词用 `python pick_words.py 50` 一键完成，不要用子代理/Agent tool 做这件事。
- 脚本已写好：`workspace/siyuan-vocabulary/pick_words.py`
