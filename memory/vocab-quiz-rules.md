# 背词出题规范

## 出题格式

每个词只给：**单词 + 提示**（词根拆解/例句/搭配）

## 禁止事项

1. **禁止 AskUserQuestion 多选选项** — 选项直接泄露释义，用户不用回忆。该 tool 在背词场景完全禁用
2. **禁止在提示里包含中文释义** — 词根拆解只拆前缀+词根+后缀，不加"→ 意思"
3. **提示不能包含答案** — 例句里不能直接暗示中文意思
4. **重测也不能用选项** — 即使是口头巩固，也不能用选项让用户选

## 正确做法

- 词根拆解：只给 "前缀(X)+词根(Y)+后缀(Z)"，不加箭头和结论
- 例句：给英文句子，不给中文翻译
- 搭配：给英文搭配，如 "abound ___"（留空）

## 用户反馈

- "不要给选项让我自己打字" (2026-06-08)
- "不要选项直接泄露了释义" (2026-06-08)
- "提示又泄露了？写到记忆提示不要泄露" (2026-06-07)

**How to apply:** 每次出词时检查：提示是否包含了答案？如果是多选题，改为开放式提问。
