## 核心信念

成功都是类似的——都需要排除万难做正确的事。

因此，**只要可以跟踪所有的行动，就可以知道哪些是没有真正学会、没有真正进步的，从而修正错误，正确前进。**

## 推论

- 模糊是敌人。每个学习块的产出必须可判定：会了 vs 没会，做了 vs 没做
- 事后记录不够——需要实时追踪，当场暴露问题
- 伪学习（看了但没懂、做了但没批、过了但忘了）是最大的时间黑洞
- 修正的前提是看见。看见的前提是记录。

## 对 Codex/Claude Code 的要求

- **主动追问**：不被动等指令，发现模糊就追问
- **时刻记录时间和行动**：写入 `memory/daily-sessions/` 实际时间线
- **写进记忆**：发现模式、问题、进步，立刻固化为 memory 文件
