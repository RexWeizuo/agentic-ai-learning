### 错题详解

**Q4 · PDU/SDU 关系**
> 问：关于PDU和SDU关系正确的是？

- A ❌：3-PDU = 3-SDU + 3-PCI，而 3-SDU = **4-PDU**，不是 4-SDU
- B ✅：数据链路层是唯一首部+尾部都有的层
- C ❌：物理层 PDU 是比特流，不是帧
- D ❌：应用层 SDU 不能直接当传输层 PDU，中间隔了应用层头部封装

**Q6 · 文件传输属于哪层**
> 两端用户传输文件，应属于OSI的（）处理。

学生选 C（传输层）❌。正确答案 D（应用层）。

原因：传输层只管把字节流可靠送到——不关心这些字节是文件、网页还是邮件。**文件**的定义和访问方式是应用层的事（FTP/HTTP）。

**Q20 · 相邻节点流量控制**
> 实现两个相邻节点间流量控制功能的是？

学生选 C（网络层）❌。正确答案 B（数据链路层）。

关键区别：
- 链路层：**相邻**节点间（一跳），帧级别的流量控制
- 网络层：端到端，做的是**拥塞控制**不是流量控制
- 传输层：端到端流量控制（如 TCP 滑动窗口）

### 错题详解

**Q21 · 分布式处理 ≠ 负载均衡**
> 任务分配给空闲计算机 → 体现什么？

学生选 D（负载均衡）❌。正确答案 C（分布式处理）。

区别：负载均衡是多台服务器均匀分担同类请求。**分布式处理**是把一个复杂任务拆给不同机器共同完成——题干"复杂任务分配给其他空闲计算机"正合此意。

**Q22 · 通信子网 = 网络层+数据链路层+物理层**
> 属于通信子网的是？I.网络层 II.物理层 III.应用层 IV.传输层

学生选 D（I,II,IV）❌。正确答案 A（I,II）。

通信子网 = 下三层（物理层+数据链路层+网络层）。传输层是承上启下的桥，不属于通信子网也不属于资源子网。

**Q23 · 总线拓扑实时性差**
> 关于拓扑结构描述正确的是？

学生猜 A ❌。正确答案 B。

- A：星形是集中式，节点之间不分布处理
- B ✅：总线共享一根线，有碰撞，不能保证实时
- C：树形可以扩展
- D：网状成本**高**（每条节点多条连线）

**Q26 · 时延带宽积**
> 传播速度 2×10⁸ m/s，200km，10Mb/s → 正在传播的比特数？

学生计算：200/200000×10Mb/s=2000 ❌。

传播时延 = 200km / 2×10⁵km/s = 0.001s = 1ms

时延带宽积 = 0.001 × 10×10⁶ = **10000** → C

错误原因：200/200000 = 0.001，不是 0.0002。

**Q29 · 报文交换 vs 分组交换（2013）**
> 8Mb 报文，10kb 分组，两段 10Mb/s 链路。两种方式总时间？

学生选 C（1600, 800）❌，以为分组交换=800ms。正确答案 D（1600, 801）。

- 报文交换：8Mb/10Mb/s × 2 = 1600ms ✅
- 分组交换：800个分组，流水线——最后一个分组 A发完=800ms，路由器转发=1ms → 801ms 到 B

分组交换不是 800ms！流水线是最后一个包到达时间，不是第一个。

**Q30 · 分组交换总时延（2023）**
> 1MB 文件，1000B 分组，100Mbps，时延带宽积=1000bits

学生算 1025 个分组 ❌。M=10⁶，1MB/1000B = 1000 个（不是 1024）。

发送时延每包 = 0.08ms，传播时延 = 0.01ms

总 = 0.18 + 999×0.08 = **80.10ms → D**（不是 80.09）
