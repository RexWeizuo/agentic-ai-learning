## Ch.01 期间学习（2026-06-08）

> 背景：选市场时引出加密货币概念，逐步深入

| 概念 | 一句话 | 追问/纠正 |
|------|--------|----------|
| 加密货币是什么 | 纯数字的钱，没有央行/银行，靠密码学+全球计算机维护账本 | |
| 为什么能交易 | 有人愿意买、有人愿意卖，和股票逻辑一样。区别：股票背后是公司，加密货币背后是共识 | |
| 7×24 交易 | 没有实体交易所，全球计算机永远开着 | |
| 能买日常东西吗 | 几乎不能——价格太不稳定。用途：投机、跨境转账、智能合约平台 | "只是有钱人的玩具吗？" → 不，阿根廷抗通胀、尼日利亚收款、被制裁国家刚需 |
| 以太坊跑程序 | 程序部署后没人能改/删/关，全球计算机运行，每步要付 gas 费防死循环 | "微信也是程序啊" → 区别：微信腾讯能封，以太坊没人能关 |
| USDT 稳定币 | Tether 发，声称 1:1 锚美元，长期拒审计——对手方风险 | "会不会卷钱跑路？" → 重复博弈>一次性博弈，跑路收益<持续赚手续费 |
| 挖矿 | 一万台电脑抢答数学题，先对的拿奖励，其他人白干。没合同没雇主——强调竞争+随机奖励 | "为什么叫挖矿不叫出租电脑？" |
| 质押 | 押 ETH 换记账权。掉线/作恶→押金被扣。可凑份子但信任合约代码 | "押的钱不会被拿走吗？" |
| NFT | 技术上没毛病（数字唯一编号），但 290 万买推特→现在值 0.2 美元。稀缺性≠价值 | "为什么是炒作？" |
| 涨跌停 | A 股一天最多涨/跌 10%，触到就停。目的防恐慌踩踏。港股无涨跌停 | |
| 蓝筹股 | 大市值、业绩稳、流动性好（茅台/工行）。赌场术语——蓝色筹码面值最高 | "大蓝筹都不是科技公司啊" |
| A 股分层 | 主板(大蓝筹)→科创板(688,硬科技,允未盈利)→创业板(300,成长型) | "中小创业公司也能上市吗" |
| 外商投资负面清单 | 列出禁止外资独资进入的行业。互联网内容服务在清单上——政府态度：舆论阵地不能让外资说了算 | "外资不信任中国互联网吗？" |
| VIE 架构 | 开曼上市壳(外资能买)→合同控制→中国运营公司(持牌照)。阿里/腾讯/百度/拼多多全此架构。非逃税，是法律夹缝中的生存策略 | "阿里为什么注册在开曼？外资百分百控股吗？" |
| 港股 | 腾讯/美团/小米上不了 A 股（盈利要求/VIE）→ 走香港。港股通 50 万门槛，T+0，无涨跌停 | "港股是什么？为什么有些企业不在A股？" |

## Ch.02 期间学习（2026-06-09）

> 背景：讨论量化回测指标时引出

| 概念 | 一句话 | 追问/纠正 |
|------|--------|----------|
| 夏普比率 | (年化收益-无风险利率)/波动率。0.5=不值得，1.0=及格，2.0+=优秀 | |
| 最大回撤 | 峰顶到谷底的最大跌幅。比收益率更影响"能否拿住"——跌 60% → 10 万剩 4 万 → 大部分人扛不住跑路 | "为什么最大回撤比收益率更关键？" |
