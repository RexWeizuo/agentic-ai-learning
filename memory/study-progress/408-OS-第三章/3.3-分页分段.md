## 概念纠正

### 分页一维 vs 分段二维

- **分页**：程序员看到的是一个地址数字（0x12345），页号+偏移是 MMU 硬件自动切分的，对程序员透明 → **一维**
- **分段**：程序员显式指定 (段名, 偏移)，如 `mov [ds:100]` → **二维**
- 关键区别：**谁来拆分**。硬件自动拆 = 一维；程序员显式指定 = 二维。

### 分页的动态重定位

分页系统中，页表替代了重定位寄存器，但本质相同——执行时才做地址转换。
