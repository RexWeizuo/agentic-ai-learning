### 比值法 · (n/n+1)^n → 1/e 未识别
> 日期：6/20
> 题：Σ a^n·n!/n^n (a>0)，比值结果为 a/e·(n/n+1)^n
> 卡点：(n/n+1)^n = 1/(1+1/n)^n → 1/e，这个极限形式没反应出来
> 
> **追问（6/20）**：为什么 (1+1/n)^n 单调递增？→ AM-GM：n 个 (1+1/n) + 1 个 1，均值不等式即得 (1+1/(n+1))^(n+1) ≥ (1+1/n)^n。单调递增+有上界→极限存在→e。
> 结论：a<e 收敛，a>e 发散
> 
> a=e 时比值=1 失效 → 斯特林公式 n!~√(2πn)(n/e)^n → 通项~√(2πn)→∞ → **发散**

## 新增卡点（6/20 习题）

### Q1(1) · 等比级数直接算
> 题：Σ (3^n+(-2)^n)/6^n
> 卡点：可以拆成 Σ(1/2)^n + Σ(-1/3)^n，两个都是等比级数，直接套公式。没往这方向想。

### Q? · 1/ln(n+1) 和调和级数比较
> 题：Σ 1/ln(n+1)
> 卡点：没想出来和调和级数比。构造 f(x)=x-ln(x+1)，导数>0+f(1)>0 → n>ln(n+1) → 1/n < 1/ln(n+1)。调和级数发散→更大的 1/ln(n+1) 必发散。

### Leibniz 单调性验证 · ln n / n
> 日期：6/20
> 题：Σ(-1)^(n-1) ln n / n
> 卡点：绝对值级数发散（ln n/n > 1/n），需 Leibniz。单调减没写出来——令 f(x)=ln x/x，f'(x)=(1-ln x)/x^2，x>e 时负→递减。看解析才补上。

### Leibniz 单调性 · ln(n/(n+1))
> 日期：6/20
> 题：Σ(-1)^(n-1) ln(n/(n+1))
> 卡点：单调减的判断偷看解析了。|通项| = -ln(n/(n+1)) = ln((n+1)/n) = ln(1+1/n)，1/n↓ → ln(1+1/n)↓。取绝对值后多一步变形。

### 任意项级数判敛 · 做题顺序
> 日期：6/20
> 规则：任意项级数判敛三步走——
> ① 通项→0？→ 不趋零直接发散
> ② 趋零则试**绝对值级数** → 比值/比较法判 → 收敛=绝对收敛
> ③ 绝对值发散才走 Leibniz 判**条件收敛**

### 比值法 · 下标取错 n/n-1 → n+1/n
> 题：Σ n^(n+1)/(n+1)!
> 卡点：自作聪明用 n 项/n-1 项，比值法应该取 **a_{n+1}/a_n**，对通项代入 n+1 再除。

### arctan n 放缩 · 下限取 n=1
> 题：级数 Σ arctan n / n^?
> 卡点：想到了 arctan n → π/2 但没注意单调性——arctan 单调增，n≥1 时 arctan n ≥ arctan 1 = π/4。放缩需要用下限不是上限。

### Q1(2) · ln(n/(n+1)) 裂项相消
> 题：Σ ln(n/(n+1))
> 卡点：ln(n/(n+1)) = ln n - ln(n+1)，前 n 项和 S_n = ln1 - ln(n+1) = -ln(n+1) → -∞，发散。裂项消去法没反应过来。

## 新增卡点（6/20）

### 积分法 · Σ 1/(n(ln n)^p) 需讨论 p
> 题：Σ 1/(n(ln n)^p)，p>0
> 卡点：积分法推导跳过了，看解析才知道需讨论 p。令 u=ln x，∫ dx/(x(ln x)^p) = ∫ du/u^p
> 结论：p>1 收敛，p≤1 发散（和 p 级数同模版，收敛门槛不变）
