## Q17 · 实对称相似充要条件 ⭐

> A = [[1,a,1],[a,b,a],[1,a,1]] 与 B = diag(2,b,0) 相似的充要条件为（ ）。
> (A) a=0, b=2  (B) a=0, b任意  (C) a=2, b=0  (D) a=2, b任意
> 答案：B

**核心知识点**：实对称矩阵相似 ⇔ 特征值相同。
因为实对称必可对角化（几何重数=代数重数），特征值同 → 对角化后矩阵同 → 相似。

**五步解法**（逐个验证，不解三次方程）：

1. 识别 A 为实对称 → 特征值必须 = B 的特征值 2, b, 0
2. λ=0：A 第1行=第3行 → |A|=0 → 0是特征值 ✅（不卡参数）
3. λ=2：|A-2E| = 4a² = 0 → a = 0
4. λ=b（a=0代入）：|A-bE| 中间行全0 → 行列式恒0 → b任意
5. 答案：B

**关键计算**：|A-2E| 的行变换 → 第1行+第3行 → (0,2a,0) → 按第1行展开 → 4a²=0。

## Q23 · 线性方程组 + 正交对角化 ⭐

> A = [[1,1,a],[1,a,1],[a,1,1]]，β=(1,1,-2)ᵀ，Ax=β 有解但不唯一。
> (1) 求 a；(2) 求正交矩阵 Q 使 QᵀAQ 为对角矩阵。
> 答案：(1) a=-2

**关键陷阱**：|A|=0 解得 a=1 或 a=-2。

**a=1 为什么被排除**？
代入 A：三行全是 (1,1,1)。方程组变成：
```
x₁+x₂+x₃ = 1    ← 第1行
x₁+x₂+x₃ = 1    ← 第2行
x₁+x₂+x₃ = -2   ← 第3行：左边一样，右边是-2
```
同一表达式同时=1和=-2 → **矛盾，无解**。不符合"有解但不唯一"。

a=-2 时 r(A)=r(A,β)=2<3，无穷多解 ✅。

## Q24 · 正交性证明（"插桥法"）⭐

> 设 A 实对称，λ₁≠λ₂，证明 α₁ᵀα₂ = 0。

**核心技巧**：插入 A 作为桥——α₁ᵀAα₂ 同时连接 A、α₁、α₂。

```
第一次（A 作用在 α₂）：α₁ᵀAα₂ = λ₂(α₁ᵀα₂)
第二次（Aᵀ=A 翻到左边）：α₁ᵀAα₂ = λ₁(α₁ᵀα₂)

λ₂(α₁ᵀα₂) = λ₁(α₁ᵀα₂) → (λ₂-λ₁)(α₁ᵀα₂) = 0
λ₁≠λ₂ → α₁ᵀα₂ = 0。∎
```

## 实对称正交对角化 · 几何意义

**椭球模型**：
- 实对称矩阵 = 沿相互垂直方向的拉伸/压缩（纯变形，无旋转）
- 特征向量 = 椭球主轴方向（天然正交 = α₁ᵀα₂=0 的几何含义）
- 特征值 = 各方向的拉伸倍数
- 重特征值 = 截面是圆（任意方向都是特征向量，需 Schmidt 挑正交）
- 非对称矩阵 = 有旋转+剪切 → 特征向量不正交，可能有复数特征值
