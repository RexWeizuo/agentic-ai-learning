## 改进方案（6/8周复盘产出）

### 1. 休息超时预警

- **小休息**：15min 不回来 → Codex/Claude Code 追问"在干嘛"
- **大午休/晚休**：2h 不回来 → 追问

### 2. 手机控制

- **学习时**：手机放包里（不是桌上，不是裤兜）
- **睡前**（23:00后）：手机放客厅，进卧室不带手机
  - 替代品：和 Agent session 聊量化/Agent 学习（低强度，比刷视频有意义）
- **睡不着的处理**：干躺，不开灯，不拿手机。哪怕躺一小时也比刷手机好
- **起床后**：第一件事不能是手机

### 3. 每周一回顾机制

- **时间**：周一上午或午睡后，雷打不动 3 小时
- **方式**：Codex/Claude Code 主导，不是自己翻
  - 翻上周时间线（追问空档都去了哪）
  - 翻错题（互动式复习，不是看）
  - 定本周重点（不超过 3 件）
  - 写回顾文件

### 4. 双 Session 分工

| Session | 职责 |
|---------|------|
| 主 session（本项目） | 学习生活管理 + 301/408 学习 + 时间追踪 |
| Agent session | 量化 Agent 开发 + AI 学习 + 求职准备 |

Agent session 的产出也记录到本项目 memory，两条线共享记忆。

### 5. 身体

- 腰突：每天 10min 腰椎康复动作（猫牛式、婴儿式、鸟狗式）
- 打球：可投篮、避免对抗
- 50-10 的"10"：必须真站起来走动

### 6. 睡眠硬控

| 时间 | 动作 |
|------|------|
| 23:30 | 如果在 → 提醒"为什么没睡" |
| 24:00 | 手机放客厅 |
