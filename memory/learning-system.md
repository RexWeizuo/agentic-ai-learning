## 每日节奏：50-10-汇报

- **50min 深度块** → **10min 真休息** → 汇报 → 下一块
- 一天 6 个深度块 + 1 个弹性缓冲 ≈ 5-5.5h 有效学习
- 每次打开 Codex/Claude Code = 学习块开始
- 每次汇报 = 记录实际进展（不是事后补）

## 休息原则

真休息 ≠ 玩手机。真休息：站起来走、望远处、喝水吃水果、闭眼停机、写一句困惑、向 Codex 汇报。

## 周节奏

| 周一 | 上午休息，下午启动，无新课（启动日） |
| 周二 | 满负荷，最硬内容 |
| 周三 | 正常 |
| 周四 | **主动降载 4 块**（预设轻量日，承认历史滑坡规律） |
| 周五 | 正常 |
| 周六 | 可加量 |
| 周日 | 5 块 + 周回顾，16:00 收工 |

## 汇报协议

三种汇报类型：
- **完成汇报**："XX done，50min，有个点模糊"
- **休息汇报**："休息 15min，14:20 继续"
- **调整汇报**："比预想难，只做了 5/10，多用 30min"

## Codex/Claude Code 角色

从"事后记录员" → **实时学习搭档**：
- 会话打开 = 学习块开始，主动确认当前任务
- 每次汇报 = 记录到 `memory/daily-sessions/` 实际时间线
- 收尾 = 总结状态 + 预告明天任务
- 监控疲劳：[[pulse-check]] 连续低分触发警告
