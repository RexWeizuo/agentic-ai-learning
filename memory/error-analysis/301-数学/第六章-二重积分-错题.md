# 第六章 二重积分 错题

## 题1 — 比较 I1=(x+y)^2 与 I2=(x+y)^3 在D={x≥0,y≥0,x+y≤1}上的大小
- **错误类型**: 概念缺失
- **原因**: 不会比较二重积分大小。需要利用：在积分区域上，若 f(x,y) ≤ g(x,y) 处处成立，则积分不等号同向。
- **关键**: D上 0 ≤ x+y ≤ 1，所以 (x+y)^3 ≤ (x+y)^2，因此 I2 ≤ I1

## 题2 — 比较 I1=sin^2(x+y) 与 I2=(x+y)^2，D为任意有界闭区域
- **错误类型**: 概念缺失
- **原因**: 同上。sin^2 t ≤ t^2 对所有 t 成立（因为 |sin t| ≤ |t|），所以 I1 ≤ I2

## 题3 — 极限 lim(r→0+) (1/πr^2) ∬_{Dr} f(x,y) dxdy
- **错误类型**: 概念缺失
- **原因**: 忘了二重积分的中值定理。∬_{Dr} f = f(ξ,η)·πr^2 → 极限 = f(x0,y0)

## 题5 — ∬_D (xy² + e^{x+2y}) dσ, D=[-1,1]×[-1,1]
- **错误类型**: 待补充
- **关键**: 对称性。xy² 对 x 是奇函数 → 积分为0。e^{x+2y}=e^x·e^{2y} → 分离为两个定积分乘积。

## 题6 — ∬_D xy·e^{xy²} dσ, D=[0,1]×[0,1]
- **错误类型**: 待补充
- **关键**: 先对 y 积分，令 u=xy² → du=2xy dy → xy dy = du/2

## 题18 — ∬_D √(4-x²-y²) dσ, D={x²+y²≤2x}
- **错误类型**: 待补充
- **解析**: 
  极坐标：x²+y²≤2x → r≤2cosθ, θ∈[-π/2, π/2]
  ∬ √(4-r²)·r dr dθ
  u=4-r² → r dr = -du/2
  ∫ r√(4-r²) dr = -(1/3)(4-r²)^{3/2}
  代入上下限后：被积函数含 |sinθ|³，利用对称性 2∫_0^{π/2}
  最终 = 8(3π-4)/9

## 题20 — ∬_D xy dσ, D={x²+y²≥1, x²+y²≤2x, x≥0, y≥0}
- **错误类型**: 待补充
- **关键**: 
  区域是月牙形：单位圆外 + 圆(x-1)²+y²≤1内 + 第一象限
  极坐标：r∈[1, 2cosθ], θ∈[0, π/3]（cosθ≥1/2）
  ∬ r³cosθsinθ dr dθ，换元 u=cosθ
  最终 = 9/16
