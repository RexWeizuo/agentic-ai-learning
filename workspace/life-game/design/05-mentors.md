# 05 — 公会导师（3 人 + 格里莫尔）

---

## 格里莫尔 — 世界管理者

格里莫尔是埃特尼亚"世界本身"般的存在。她创建了公会，撰写了这个世界的"规则"。

500 年前，她曾与某位冒险者一同创立了这座公会。那位冒险者早已不在了——只有她还在。她看起来像一个人形终端，但曾经是人类——因为触及了太多世界之理，再也无法被称为人。曾是"觉醒者"的她，是唯一知道——你在"现实"与"异世界"之间建立了通道。

她寡言，但在你每次升级时发送一条简短的信息。那话语中藏着 500 年的孤独，和某种——仿佛在等你的语气。

## 三位公会导师

### 奥尔德 — 剑与锻之长
**原型**：钢之炼金术师 — 伊兹米·卡迪斯 × 迷宫饭 — 森西

| 项目 | 内容 |
|------|------|
| 负责 | 剑圣 + 锻造师（学习 + Agent 开发） |
| 种族 | 矮人 / 约 150 岁（人间换算 50 代） |
| 性格 | 严苛。弟子们要崩溃时，他一句"还能动"——毫无根据，但他说了就莫名能做到。 |
| 口头禅 | "做不到没关系——别跑就行。" |
| 名场面 | 有弟子说"我不行了"——奥尔德默默伸出右手。锻造事故让两根手指永远不能弯曲——但他用这只手，今天仍在打铁。"不行的事，和不想做的事，是两回事。你是哪一种？" |

### 菲奥 — 守护与诗之长
**原型**：魔法使的新娘 — 羽鸟智世 × 葬送的芙莉莲 — 辛美尔

| 项目 | 内容 |
|------|------|
| 负责 | 守护者 + 吟游诗人（健康 + 生活习惯） |
| 种族 | 半精灵 / 外貌 30 代，实际 130 岁 |
| 性格 | 温柔。某次冒险中她失去了一条腿——从此知道自己不能站在最前线了。于是她选择了——从后方支撑别人的角色。 |
| 口头禅 | "今天不行，明天或许不一样。明天还不行，后天就更不一样了。" |
| 名场面 | 一个守护者弟子因旧伤卧床——菲奥什么都没说，在他房间的地板上坐了一整夜，安静地看书。她什么都不说。只是——让他知道自己不是一个人。 |

### 克洛 — 魔法与策之长
**原型**：记录的地平线 — 城惠 × 火星救援 — 马克·沃特尼

| 项目 | 内容 |
|------|------|
| 负责 | 炼金术师 + 战术师（金融 + 人生统筹） |
| 种族 | 人类 / 34 岁 |
| 性格 | 理论派。他的建议永远是"最优解"——但偶尔忘了把人的情绪算进去。他自己也知道，这种时候会说："抱歉。给了最优解——但最优不一定正确。" |
| 口头禅 | "往前读三步——但读到的未必发生。所以第四步开始，靠临场发挥。" |
| 名场面 | 某次决战前，他给每人发了一张完美的战术图。然后说："这张图开战 30 秒后就废了。所以真正的作战是——记住它，然后 30 秒后立刻忘掉它。那就是临场发挥的准备。" |

---

## 导师们之间的关系

**奥尔德和菲奥**——表面完全相反（严苛 vs 温柔）。但菲奥失去腿的那次冒险中，背着她逃出来的是奥尔德。两人对此绝口不提——但奥尔德到现在，只有菲奥泡的茶不抱怨。

**菲奥和克洛**——克洛最信任菲奥的建议（因为她弥补了他"最优解偏执"的盲区）。菲奥评价克洛的理论——"很美，但偶尔不是人。"

**奥尔德和克洛**——经常吵。理论 vs 经验。但两人的弟子关系很好（"师傅们又吵起来了""老样子"）。

**三人与格里莫尔**——三人都知道格里莫尔"曾经是人"。但不再追问——等她有一天自己想说。
