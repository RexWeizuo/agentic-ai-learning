# 02 — 职业 · 技能 · 称号 · 进化

## 六大职业

### ⚔️ 剑圣 — 学习·知识领域
**主属性**：INT / **副属性**：SPI

追求剑道的极致之人。每一斩都倾注全部——如同解题时的专注力。修行之地是"古之图书馆"——在禁书锁链缠绕的最深处，一本一本解开封印。

| 位阶 | 称号 | 技能 |
|------|------|------|
| 见习 | 剑之徒 | **一闪**：MP 15——下一击命中率 +20%（集中学习） |
| 修行 | 剑术使 | **破邪之剑**：MP 25——击败的敌人魂魄被吸收，转化为永久经验加成（错题纠正后强化记忆） |
| 冒险 | 魔剑士 | **剑圣之间**：MP 40——展开 25 分钟的时间缓流结界（25 分钟超级专注模式） |
| 专家 | 剑圣 | **残心**：被动——与曾击败的敌人再战时经验 1.5 倍（复习加成） |
| 大师 | 剑圣候补 | **天眼**：MP 80——自动分析一周的战斗记录，生成弱点报告 |
| 贤者 | 真理之剑 | **概念显化**：觉醒技——将掌握的剑技"传授"给 NPC，贡献值 2 倍 |

### 🔨 锻造师 — Agent/AI 开发领域
**主属性**：CRT / **副属性**：INT

火山中心锻造神殿里，他们锤炼矿石、铭刻符文。代码即符文，调试即去除杂质，完成的项目即神器的诞生。

| 位阶 | 称号 | 技能 |
|------|------|------|
| 见习 | 锻造之徒 | **矿石感知**：MP 15——初看矿脉（代码/文档）时辨别价值的准确率 +30% |
| 修行 | 符文工匠 | **素材精炼**：MP 20——从已有矿石中提取纯粹成分（从参考代码中提取可复用模式） |
| 冒险 | 武具工匠 | **符文铭刻**：每完成一件武器即铭刻一枚符文，集齐符文触发套装效果 |
| 专家 | 名匠 | **神之熔炉**：被动——长时间锻造不再出现低级失误 |
| 大师 | 神器锻造 | **碎星**：每完成一件神器（完整项目），CRT 大幅提升且获得称号 |
| 贤者 | 世界锻造 | **新世界创造**：自创框架/工具→影响所有 NPC 的锻造效率 |

| 冒险 | 城塞骑士 | **防壁展开**：持续防御训练后自动发动的护盾 |

## 技能进化

```
主动技能使用次数达到阈值 → 进化

示例：一闪（命中率 +20%）
  → 使用 30 次 → 一闪·改（+25%）
  → 使用 80 次 + INT ≥ 20 → 连闪（+30%，可连续使用 2 次）
  → 使用 150 次 + INT ≥ 40 → 绝·一闪（+40%，MP 消耗减半）

示例：破邪之剑（错题纠正→永久加成）
  → 使用 20 次 → 破邪之剑·改（永久加成 +50%）
  → 使用 60 次 + INT ≥ 25 → 噬魂之剑（纠正时自动强化相关概念）
```
