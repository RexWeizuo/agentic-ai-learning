# 04 — 世界机制

> 你在现实中做了什么 → 埃特尼亚就发生什么。转换原理连你自己也说不清。但它确实在发生。

---

## 日常转换规则

### 学习 → 魔物讨伐

```
现实：机器学习 3 道题，全部正确
      ↓
异世界：在废弃矿洞中遭遇了三只岩蜥。
      第一只一剑斩下，第二只稍费了些功夫，
      第三只时你甚至有余裕尝试新的剑招。

获得：INT 经验 +30，贡献值 +150，灵魂碎片 +1
NPC 反应：加尔德"今天手感不错啊。出手比平时快。"
```

### 错题订正 → 治愈伤口再战

```
现实：昨天做错的题，今天重新做对了
      ↓
异世界：昨天留下深伤口的森林魔兽——再次相遇。
      这次你已看穿了它的弱点。一击毙命。
      它的魂魄被净化，化作你的力量。

获得：INT 经验 +45（复习加成 1.5 倍），贡献值 +200
NPC 反应：伊修卡"…你打败了曾经输过的对手。这是屠龙之术的前兆。"
```

### 腰突康复训练 → 修复城墙壁垒

```
现实：做了 10 分钟的康复动作
      ↓
异世界：在废弃要塞，你修补了一小段崩塌的城墙。
      石料沉重，震得筋骨发酸——但城墙确实比以前稳固了一些。

获得：VIT 经验 +20，守护者职业经验加成
NPC 反应：泰拉"这修补的痕迹——很细致。没有人教你，你却能做到。"
```

### 刷太久手机 → 被魔雾吞噬

```
现实：不知不觉刷了 2 小时手机
      ↓
异世界：旅途半途，浓雾袭来。视野一片惨白。
      方向感消失。等你走出来——发现本该走的路程一步未动。
      两小时，就这么被雾吃掉了。

惩罚：SP 临时上限 -10
NPC 反应：加尔德"魔雾啊…谁都遇到过。要紧的是雾散了之后，
         别走错方向。"
```

### 高质量睡眠 → 旅店安眠

```
现实：睡了 7.5 小时好觉
      ↓
异世界：路边旅店，难得一个安全的夜晚。
      壁炉的火噼啪作响，隔壁传来谁的鼾声。
      晨曦唤醒你——盔甲上的伤痕淡了许多。

恢复：HP 全满，SP 全满，MP +50%
NPC 反应：辛"…昨晚我在旅店周围巡逻。什么都没来。
         偶尔——也该有这样的夜晚。"
```

## 副本

副本是"现实中的目标"异世界化后的形态。一旦进入，必须在限定现实时间内完成。

### 副本生成

```
触发器：现实中判断"需要复习某章"
      → 异世界中副本入口浮现

示例：封印回廊（线代 Ch5 错题回顾）
  难度：B
  时限：3 天
  层数：5（每一道错题成为独立的一层）
  每层 Boss：错题异世界化后的敌人
  最深处宝藏：称号"封印解除者"，灵魂碎片 ×5
  失败条件：3 天内未完成全部订正
  失败后果：副本"深化"——同样的题目下次变成更强敌人
```

### 副本类型

| 副本名 | 触发条件 | 难度 | 可组队 NPC |
|--------|---------|------|-----------|
| **试炼回廊** | 新章开始 | C~S | 伊修卡（剑圣指导）、泽法尔（记忆的提示） |
| **反噬深渊** | 错题累积 5+ | B~A | 诺克斯（失败的专家）、凡（时间回溯的智慧） |
| **专注狭间** | 90 分钟连续学习挑战 | B | 辛（封雷的专注）、米拉（深海听觉） |
| **锻造神熔炉** | Agent 项目启动 | A~S | 泰拉（魔像的坚韧）、泽法尔（对未知的适应） |
| **星海大决战** | 模拟考试 | S~SS | 加尔德（老兵）、凡（决战亲历者） |
| **心之迷宫** | 动力下降/持续倦怠 | 特殊 | 伊格尼斯（不再祈祷者的陪伴） |

## 钓鱼

```
河边垂下钓竿——
会钓上什么，无从知晓。这是轻松复习和偶发性发现的异世界化。

· 银鳞小鱼 → 小复习成功 → INT EXP +5
· 古代记忆碎片 → 遗忘知识的再发现 → INT EXP +20
· 生锈的戒指 → 什么也没有（偶尔空竿——钓鱼本就是如此）
· 黄金鲤鱼 → 稀有发现 → 灵魂碎片 +1
· 朽木枯枝 → 真的什么也没有（但 RES EXP +3——"等待"也是经验）
```

**可邀约钓鱼的 NPC**：米拉（熟悉水生物）、加尔德（莫名擅长）、泰拉（可以一动不动等待）

## 一天的节奏

```
【黎明】旅店醒来。检查昨夜恢复量。
       今日天色（天气 = 当天运势的轻微 Buff/Debuff）

【早晨】公会公告板确认任务（今日任务 = 现实计划）
       决定今天磨哪把剑（选择主攻科目）

【白昼】自由行动——魔物讨伐（学习）、副本（复习）、
       锻造（项目）、钓鱼（轻松复习）、市场（规划）

【黄昏】向公会报告今日战果（自动经验结算）
       查看排名变动

【夜晚】酒馆与 NPC 闲谈（羁绊事件触发）
       检查壁垒修复（康复训练确认）

【深夜】回旅店。炉火前整理装备（一日反思）。
       准备好明日的探索，入睡。
```
