|------|------|
| 种族/年龄 | 人类女 / 22 岁（被龙诅咒 4 年） |
| 职业/等级 | 剑圣 Lv.26 |
| 外貌 | 右手背到颈侧蔓延着黑色鳞片。右手尚是人手，但指尖已开始化为钩爪。深兜帽遮着脸，不愿与人目光相接。 |
| 背景 | 4 年前，她故乡的村庄被龙袭击。全村遇难——只有她活了下来。龙没有杀她，而是刻下了"诅咒"：她将逐渐化为龙——鳞片覆盖全身，失去理智，去袭击下一个村庄。她成为剑圣不是为了屠龙——是为了在自己完全变成龙之前，找到一个能杀死自己的人。 |
| 困境 | 鳞片正缓慢扩散，还剩几年。越变强——越加速龙化。变强等于加速自己的死亡——这个悖论是她每天醒来面对的第一件事。 |
| 与你关系 | 初期——她在"观察"你。你能"凭空变强"这件事让她警觉——她怀疑你身上有"龙以外的东西"。中期——她问你"如果我变成龙了……"。本是玩笑。你的认真回答让她沉默了很久。后期——她不再寻找"能杀死自己的人"。转而和你一起研究——诅咒能否被逆转。 |

**羁绊技能**：
- Lv.3：**龙鳞加护**——致命伤害一次 50% 减免
- Lv.5：**诅咒的悖论**——变强的副作用（加速恶化）被削弱 30%

**原型**：NieR:Automata — 2B，Frieren — 辛美尔，咒术回战 — 虎杖悠仁
