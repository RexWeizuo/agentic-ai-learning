3、解析  
(1) $A \bar{B} \bar{C}$ .  
(2) $A B \bar{C}$ .  
(3) $A B C$ .  
(4) $A + B + C$ .  
(5) $A B + A C + B C$ 或 $A B \bar{C} \cup A \bar{B} C \cup \bar{A} B C \cup A B C$ .  
(6) $A \bar{B} \bar{C} + \bar{A} B \bar{C} + \bar{A} \bar{B} C$ .  
(7) $A \bar{B} \bar{C} + \bar{A} B \bar{C} + \bar{A} \bar{B} C + \overline{A B C}$ 或 $\overline{A B} + \overline{A C} + \overline{B C}$ . （注：此处根据图像笔迹推测，(7)左侧似为前几项的扩展，右侧为等价形式）

小雀点睛：本题主要考查大家将叙述性语言转化为事件关系的能力，有两个易错点需注意。  
(1) $\overline{A B}$ 与 $\bar{A} \bar{B}$ 易混淆，$\overline{A B} = \overline{A \cap B} = \bar{A} \cup \bar{B} = \bar{A} + \bar{B}$，$\bar{A} \bar{B} = \bar{A} \cap \bar{B} = \overline{A \cup B} = \overline{A + B}$ 。

## 第 290 页

(2) 只有 A 发生意味着 A 发生且 B, C 都不发生，所以不要只写 A，它只能表示 A 发生，并没有限制 B, C，而要写 $A \bar{B} \bar{C}$。
