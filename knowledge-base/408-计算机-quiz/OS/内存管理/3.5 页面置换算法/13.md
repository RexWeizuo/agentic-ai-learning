# 3.5 页面置换算法 · 题13

13.【2022年统考真题】下列选项中，不会影响系统缺页率的是（）。

A.页置换算法  
B.工作集的大小  
C.进程的数量  
D.页缓冲队列的长度

## 解析

- A 页置换算法：直接影响命中率。
- B 工作集大小：决定了缺页频率的基准。
- C 进程数量：多进程竞争内存，影响各进程可用页框数。
- D 页缓冲队列：用于**加速缺页处理**（暂存换出页、预读等），影响处理**速度**而非缺页**率**。

选 D。
