# 3.5 页面置换算法 · 题4

4. 在请求分页存储管理中，若进程访问的页面不在主存，且主存中没有可用的空闲块时，系统正确的处理顺序为（）$^{[2]}$。

A. 决定淘汰页，页面调出，缺页中断，页面调入  
B. 决定淘汰页，页面调入，缺页中断，页面调出  
C. 缺页中断，决定淘汰页，页面调出，页面调入  
D. 缺页中断，决定淘汰页，页面调入，页面调出

## 解析

缺页处理正确流程：① 发生缺页中断 → ② 决定淘汰哪一页（页面置换算法选 victim）→ ③ 若被淘汰页被修改过，将其**调出**到外存 → ④ 将所需页面**调入**内存。因此顺序为：缺页中断 → 决定淘汰页 → 页面调出 → 页面调入。选 C。
