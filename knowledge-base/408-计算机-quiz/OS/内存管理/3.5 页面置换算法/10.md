# 3.5 页面置换算法 · 题10

10.【2020年统考真题】下列因素中，影响请求分页系统有效（平均）访存时间的是（）。

I .缺页率  
II .磁盘读写时间  
III .内存访问时间  
IV .执行缺页处理程序的CPU时间

A.仅II、III  
B.仅I、IV  
C.仅I、III、IV  
D.I、II、III和IV

## 解析

有效访存时间公式：**EAT = (1-p)×Ma + p×T_fault**

其中 p = 缺页率(I)，T_fault = 磁盘读写时间(II) + 缺页处理 CPU 时间(IV)，Ma = 内存访问时间(III)。

四项因素**全部**影响 EAT。选 D。
