# 3.5 页面置换算法 · 题3

3. 考虑页面置换算法，系统有m个物理块供调度，初始时全空，页面引用串长度为p，包含了n个  
   不同的页号。无论用什么算法，缺页次数不会少于（）$^{[1]}$。

A. m  
B. p  
C. n  
D. min(m, n)

## 解析

引用串中有 n 个不同页号，每个页号**首次被访问时必然缺页**（初始全空）。无论什么算法、分配多少物理块，这 n 个不同页面至少各调入一次，所以缺页次数**不少于 n**。选 C。
