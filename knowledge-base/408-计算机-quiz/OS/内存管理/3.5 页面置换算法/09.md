# 3.5 页面置换算法 · 题9

9.【2019年统考真题】某系统采用LRU页置换算法和局部置换策略，若系统为进程P预分配了4个页框，进程P访问页号的序列为 $0, 1, 2, 7, 0, 5, 3, 5, 0, 2, 7, 6$，则进程访问上述页的过程中，产生页置换的总次数是（）。

A.3  
B.4  
C.5  
D.6

## 解析

LRU + 4 页框，序列：0,1,2,7,0,5,3,5,0,2,7,6。

- 0,1,2,7 初始装入（4 次缺页，无置换）
- 0 命中
- 5：淘汰最久未用的 1 → **置换 1 次**
- 3：淘汰最久未用的 2 → **置换 2 次**
- 5,0 命中
- 2：淘汰最久未用的 7 → **置换 3 次**
- 7：淘汰最久未用的 5 → **置换 4 次**
- 6：淘汰最久未用的 3 → **置换 5 次**

共 **5 次页置换**。选 C。
