# 3.5 页面置换算法 · 题12

12.【2022年统考真题】某进程访问的页b不在内存中，导致产生缺页异常，该缺页异常处理过程中不一定包含的操作是（）。

A.淘汰内存中的页  
B.建立页号与页框号的对应关系  
C.将页b从外存读入内存  
D.修改页表中页b对应的存在位

## 解析

缺页处理流程：① 陷入缺页中断 → ② 检查有无空闲页框 → **若有空闲**，直接从外存调入页 b，建立页表映射，置存在位=1 → 返回。**只有无空闲页框时才需要淘汰**。因此"淘汰内存中的页"是**不一定**发生的操作。选 A。
