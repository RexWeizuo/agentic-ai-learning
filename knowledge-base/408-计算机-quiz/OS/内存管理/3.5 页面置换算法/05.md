# 3.5 页面置换算法 · 题5

5. 假设系统为某进程分配了三个物理块，考虑以下页面号引用串：5, 0, 1, 2, 0, 3, 0, 4, 2, 3, 0, 3, 2, 1, 2, 0, 1, 5, 0, 1，若采用最佳页面置换算法，则发生（）次页面置换。

A. 8  
B. 7  
C. 6  
D. 5

## 解析

OPT（最佳置换）算法，3 页框。逐次追踪（标记 R 为发生页面置换）：

| 访问 | 5 | 0 | 1 | 2 | 0 | 3 | 0 | 4 | 2 | 3 | 0 | 3 | 2 | 1 | 2 | 0 | 1 | 5 | 0 | 1 |
|------|---|---|---|---|---|---|---|---|---|---|---|---|---|---|---|---|---|---|---|---|
| 页框 | 5 | 5,0 | 5,0,1 | R0,1,2 | ✓ | R1,2,3 | ✓ | R2,3,4 | ✓ | ✓ | R3,4,0 | ✓ | ✓ | R4,0,1 | ✓ | ✓ | ✓ | R0,1,5 | ✓ | ✓ |

- 前三项 5,0,1 为初始装入，无置换
- 2：淘汰未来最远的 5（下次位置#17），R1
- 3：淘汰未来最远的 1（位置#19），R2
- 4：淘汰未来最远的 0（位置#10），R3
- 0：淘汰永不使用的 4，R4
- 1：淘汰永不使用的 3，R5
- 5：淘汰永不使用的 2，R6

共 **6 次页面置换**。选 C。
