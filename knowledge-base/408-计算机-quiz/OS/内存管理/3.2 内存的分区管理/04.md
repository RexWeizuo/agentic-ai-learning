# 3.2 内存的分区管理 · 题4

4. 在可变分区存储管理方案中, 硬件需要一对专用寄存器实现重定位, ( ) 用于实现越界判断。III  
   A. 物理地址寄存器  
   B. 基地址寄存器  
   C. 界限地址寄存器  
   D. 逻辑地址寄存器

## 解析

**题目：**  可变分区存储管理中，硬件需要一对专用寄存器实现重定位，（ ）用于实现越界判断。

**答案：C. 界限地址寄存器**

**解析：**  动态重定位需要两个硬件寄存器：

- **基址寄存器（Base Register）** ：存放进程的起始物理地址，用于地址转换（逻辑地址 + 基址 → 物理地址）
- **界限寄存器（Limit Register）** ：存放进程的长度（或结束地址），**每次访存时检查物理地址是否越界**

因此越界判断由**界限地址寄存器**完成。
