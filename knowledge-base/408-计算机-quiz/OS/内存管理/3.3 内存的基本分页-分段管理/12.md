# 3.3 内存的基本分页-分段管理 · 题12

12. 【2021】在采用二级页表的分页系统中, CPU 页表基址寄存器中的内容是 ()。IV II

A. 当前进程的一级页表的起始虚拟地址  
B. 当前进程的一级页表的起始物理地址  
C. 当前进程的二级页表的起始虚拟地址  
D. 当前进程的二级页表的起始物理地址

## 解析

CPU 页表基址寄存器存放的是**一级页表（页目录）的起始物理地址**。MMU 用该物理地址访问一级页表，找到二级页表的物理地址，再查二级页表得到最终页框号。所有页表寻址均使用物理地址。选 B。
