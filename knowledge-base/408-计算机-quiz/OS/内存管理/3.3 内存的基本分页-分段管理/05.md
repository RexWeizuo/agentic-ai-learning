# 3.3 内存的基本分页-分段管理 · 题5

5. 【南京大学 842 2013 年】在一个基本分页存储管理系统中，页表内容如下图所示。若页的大小为 4KB，则地址转换机构将逻辑地址 0 转换成的物理地址为（）。I

|页号|块号|
| ------| ------|
|0|2|
|1|1|
|2|6|
|3|3|
|4|7|

A.8192  
B.4096  
C.2048  
D.1024

## 解析

**题目：**  页大小 4KB，逻辑地址 0 转换成的物理地址为（ ）。

**答案：A. 8192**

**解析：**

页大小 4KB = 4096B。

```
页号 P = 逻辑地址 / 页大小 = 0 / 4096 = 0
页内偏移 d = 逻辑地址 % 页大小 = 0 % 4096 = 0
```

查页表：页号 0 → 块号 2

```
物理地址 = 块号 × 页大小 + 页内偏移
         = 2 × 4096 + 0
         = 8192
```

从十进制角度直观验证：8192 = 8K，即第 2 块的起始地址。
