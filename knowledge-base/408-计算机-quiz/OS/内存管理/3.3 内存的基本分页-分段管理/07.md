# 3.3 内存的基本分页-分段管理 · 题7

7. 【南京大学 842 2014】采用分段存储管理的系统, 若地址用 24 位表示, 其中 8 位表示段号,则允许每段的最大长度是 () 。I，

A. $2^{16}$  
B. $2^{24}$  
C. $2^{28}$  
D. $2^{32}$

## 解析

**题目：**  地址 24 位，其中 8 位表示段号，每段最大长度是（ ）。

**答案：A. 2¹⁶**

**解析：**

```
地址 24 位 = 段号（8位）+ 段内偏移（16位）
段内偏移 16 位 → 最多表示 2^16 个地址
按字节编址 → 每段最大 2^16 B = 64KB
```
