# 3.3 内存的基本分页-分段管理 · 题6

6. 在基本分页管理系统中，分页是由（）完成的。

A. 程序员  
B. 硬件  
C. 编译软件  
D. 以上说法均不对

OS完成的？

分页存储管理的实现是**操作系统和硬件共同配合**完成的：

- 逻辑空间的页面划分 → 操作系统
- 物理内存的页框管理 → 操作系统
- 地址转换（查页表、拼接物理地址）→ 硬件（MMU）
- 页表和页表寄存器 → 操作系统管理，硬件使用

## 解析

**题目：**  在基本分页管理系统中，分页是由（ ）完成的。

**答案：D. 以上说法均不对**

**解析：**

分页存储管理的实现是**操作系统和硬件共同配合**完成的：

- 逻辑空间的页面划分 → 操作系统
- 物理内存的页框管理 → 操作系统
- 地址转换（查页表、拼接物理地址）→ 硬件（MMU）
- 页表和页表寄存器 → 操作系统管理，硬件使用

A 错误：对程序员透明，程序员不参与分页。  
B 不完整：硬件只负责地址转换部分。  
C 错误：编译软件不参与分页，它只生成逻辑地址。

因此既不是程序员，也不是单纯硬件或编译软件，"以上说法均不对"。
