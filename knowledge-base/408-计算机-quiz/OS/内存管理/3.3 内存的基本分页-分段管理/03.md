# 3.3 内存的基本分页-分段管理 · 题3

3. 基本分页式存储管理的主要特点是（）。III  
   A. 要求处理缺页中断  
   B. 要求扩充主存容量  
   C. 不要求作业装入到主存的连续区域  
   D. 不要求作业全部同时装入主存

## 解析

**题目：**  基本分页式存储管理的主要特点是（ ）。

**答案：C. 不要求作业装入到主存的连续区域**

**解析：**

- A 错误：缺页中断是**请求分页（虚拟存储）** 才有的，基本分页要求全部页面装入内存。
- B 错误：基本分页不涉及虚拟存储器扩充主存容量。
- C ✅：基本分页通过页表实现离散分配，逻辑上连续的页面可以映射到物理上不连续的页框。
- D 错误：基本分页**要求全部装入**，请求分页才允许部分装入。
