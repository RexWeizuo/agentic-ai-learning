# 3.3 内存的基本分页-分段管理 · 题2

2. 较好地解决外部碎片问题的存储管理方法是（）。II  
   I. 分页管理和段页式管理  
   II. 请求分页管理  
   III. 动态分区  
   IV. 分段管理  
   A. I  
   B. I、II  
   C. I、III  
   D. I、II、III、IV

   动态分区3.2 请求分页3.4 是什么

## 解析

**题目：**  较好地解决外部碎片问题的存储管理方法是（ ）。

**答案：B. I、II**

**解析：**

- **I. 分页管理和段页式管理** ✅：物理内存划分为固定大小的页框，任何页都能装入任何页框，不存在"空闲总量够但不连续无法分配"的问题，无外部碎片。
- **II. 请求分页管理** ✅：建立在分页基础上，同样无外部碎片。
- **III. 动态分区** ❌：经典的外部碎片来源，空闲区不连续，文档中需要靠"紧凑"消除。
- **IV. 分段管理** ❌：段大小可变，分配和回收会产生外部碎片（类似动态分区）。

**注意：**  分页和段页式解决了外部碎片，但牺牲是引入了**内部碎片**（最后一页用不满）。
