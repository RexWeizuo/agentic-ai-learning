# 3.1 程序装入的基本概念 · 题4

4. 以下关于链接的说法, 正确的是（）。III  
   A. 链接结束后, 装入模块里的地址是各自独立的  
   B. 链接只有静态链接这一种方式  
   C. 链接是将多个目标模块合并成一个可执行文件时重新给各个模块编址的过程  
   D. 链接就是将可执行文件装入内存的过程

## 解析

**题目**：以下关于链接的说法，正确的是（）。  
A. 链接结束后，装入模块里的地址是各自独立的  
B. 链接只有静态链接这一种方式  
C. 链接是将多个目标模块合并成一个可执行文件时重新给各个模块编址的过程  
D. 链接就是将可执行文件装入内存的过程

**答案**：C  
**解析**：

- C正确，**链接**的核心工作就是**拼接模块并统一编址**，消除模块间的地址冲突，形成完整的装入模块。
- A错误，链接后装入模块里的地址是**统一**的，不能独立。
- B错误，链接方式有**静态链接、装入时动态链接、运行时动态链接**三种。
- D错误，将可执行文件装入内存的过程叫**装入**，不是链接。
