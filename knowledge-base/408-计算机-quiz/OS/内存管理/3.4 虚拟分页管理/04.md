# 3.4 虚拟分页管理 · 题4

4. 有关虚拟存储器的叙述中正确的是 ()。II

A. 要求作业运行前, 必须全部装入内存, 且在运行中必须常驻内存  
B. 要求作业运行前, 不必全部装入内存, 且在运行中不必常驻内存  
C. 要求作业运行前, 不必全部装入内存, 但在运行中必须常驻内存  
D. 要求作业运行前, 必须全部装入内存, 且在运行中不必常驻内存

## 解析

**题目：**  有关虚拟存储器的叙述中正确的是（ ）。

**答案：B. 要求作业运行前，不必全部装入内存，且在运行中不必常驻内存**

**解析：**

文档列出了虚拟存储器的三个特征：多次性、对换性、虚拟性。

- **多次性**：不必一次性全部装入 → "运行前不必全部装入" ✅
- **对换性**：运行时可以换入换出，不必常驻 → "运行中不必常驻内存" ✅

```
传统存储管理：一次性全部装入 + 运行时常驻内存
虚拟存储管理：分多次装入    + 运行时可以换出
```
