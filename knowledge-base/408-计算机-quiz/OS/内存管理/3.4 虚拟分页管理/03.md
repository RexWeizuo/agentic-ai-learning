# 3.4 虚拟分页管理 · 题3

3. 【电子科技大学 820 2014 年】为使虚拟存储管理系统具有良好的性能, 应用程序应具备的特征是 ( )。II

A. 程序模块化程度高, 由许多小模块组成  
B. 程序应具备良好的局部性特征  
C. 程序的 I/O 操作较少  
D. 程序实际大小应小于实际物理内存容量

## 解析

**题目：**  为使虚拟存储管理系统具有良好的性能，应用程序应具备的特征是（ ）。

**答案：B. 程序应具备良好的局部性特征**

**解析：**

虚拟存储的核心理论基础就是**程序局部性原理**（文档第一节大篇幅讲解）：

> "基于程序访问的局部性原理，不必把整个程序都装入到内存中，而只要将当前需要执行的那部分页面或段装入到内存，就可以让程序开始执行。"

- 时间局部性：最近访问的很可能再次访问 → 已在内存的不易被换出
- 空间局部性：邻近地址很可能被访问 → 预调页高效

具备良好局部性的程序，缺页率低，虚拟存储性能好。

|选项|判断|理由|
| ---------------------| :----: | ------------------------------------------|
|A. 模块化程度高|❌|这是覆盖技术的要求，与虚拟存储局部性无关|
|B. 局部性特征好|✅|虚拟存储的理论基础|
|C. I/O 少|❌|无关|
|D. 程序小于物理内存|❌|如果是这样就不需要虚拟存储了|
