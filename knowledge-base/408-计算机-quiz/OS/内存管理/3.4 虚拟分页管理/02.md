# 3.4 虚拟分页管理 · 题2

2. 虚拟地址空间大小 () ${ }^{[2]}$ 。II

A. 由内存和外存容量决定  
B. 由计算机的地址结构决定  
C. 仅由外存决定  
D. 仅由物理内存决定

## 解析

**题目：**  虚拟地址空间大小（ ）。

**答案：B. 由计算机的地址结构决定**

**解析：**

文档原文 + 练习：

> "逻辑地址空间一般都比较大，其大小由系统的地址结构决定。"  
> "设主存容量为 1MB，外存容量为 400MB，计算机系统的地址寄存器为 32 位，那么进程的虚拟地址空间大小为 → **2³² B**"

虚拟地址空间大小 = 2<sup>地址位数，与物理内存、外存大小无关。32 位系统就是 4GB，64 位系统就是 2</sup>64 B。
