## 一、临界资源与临界区

> 1. 以下（ ）属于临界资源。
>    A. 磁盘存储介质      B. 公用队列      C. 私用数据      D. 可重入的程序代码
>    答案：B

> 2. 有两个进程 P1 和 P2，它们共享一个变量 n，程序代码如下。当两个进程并发执行时的输出结果有哪几种可能性？
>    （见：一、基本概念.md → 并发执行流分析 → 该例题）

> 3. 有两个并发进程 P1 和 P2，其程序代码如下：
>
> ```c
> P1(){
>     x=1;    //A1
>     y=2;
>     z=x+y;
>     print z;    //A2
> }
> P2(){
>     x=-3;   //B1
>     c=x*x;
>     print c;    //B2
> }
> ```
>
> 可能打印出的 z 值有（ ），可能打印出的 c 值有（ ）（其中 x 为 P1，P2 的共享变量）
> A. z=1,-3; c=-1;9      B. z=-1,3; c=1,9      C. z=-1,3,1; c=9      D. z=3; c=1,9
> 答案：B

> 4. 在 Peterson 算法中，turn 变量的作用是（ ）。
