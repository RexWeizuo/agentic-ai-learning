# 习题

## 第 342 页

第五章 大数定律和中心极限定理

1. 设随机变量 $X_1, X_2, \cdots, X_n$ 相互独立，且都服从正态分布 $N(\mu, \sigma^2)$，其中 $\sigma^2 \neq 0$，令 $\overline{X} = \frac{1}{n} \sum_{i=1}^{n} X_i$。则根据切比雪夫不等式估计 $P\{\left|\overline{X}-\mu\right|<2\sigma\}$ $\geq$ ______。
2. 设随机变量 $X$ 和 $Y$ 的数学期望分别为 -2 和 2，方差分别为 1 和 4，$X$ 和 $Y$ 的相关系数为 -0.5，则根据切比雪夫不等式估计 $P\{|X+Y|$ $\geq 6\} \leq$ ______。
3. 随机从数集 $\{1,2,3,4,5\}$ 中有放回地取出 $n$ 个数 $X_1, X_2, \cdots, X_n$，则当 $n \rightarrow \infty$ 时，$\frac{1}{n} \sum_{i=1}^{n} X_i$ 依概率收敛于 ______，$\frac{1}{n} \sum_{i=1}^{n} X_i^2$ 依概率收敛于 ______。
4. 设随机变量 $X_1, X_2, \cdots, X_n$ 相互独立，且都在 $[-1,1]$ 上服从均匀分布，则 $\lim _{n \rightarrow \infty} P\left\{\frac{n}{\sqrt{n}} X_i \leq \sqrt{n}\right\}=$ ( )。

(A) $\Phi(0)$

(B) $\Phi(1)$

(C) $\Phi(\sqrt{3})$

(D) $\Phi(2)$

注：$\Phi(x)$ 为标准正态分布函数。
