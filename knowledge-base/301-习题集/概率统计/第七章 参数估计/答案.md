    即 $\hat{\theta}$ 依概率收敛于 $\theta$, 故 $\hat{\theta} = \frac{2}{3}\bar{X}$ 是 $\theta$ 的一致估计量.
18. 解析  
    已知 $\sigma^2$ 的 $\mu$ 的置信区间为 $[\bar{x} - \frac{\sigma}{\sqrt{n}} u_{\alpha/2}, \bar{x} + \frac{\sigma}{\sqrt{n}} u_{\alpha/2}]$,  
    将 $\bar{x} = 5, \sigma = 0.9, n = 9, u_{\alpha/2} = u_{0.025} = 1.96$ 代入可知,  
    $\mu$ 的置信水平为 0.95 的置信区间为 $[4.412, 5.588]$.
