# 习题

## 第 285 页

第一章 随机事件和概率

1、设事件 $A, B$ 满足 $A \cup B = B$，则下列选项错误的是（ ）。  
(A) $A \subset B$  
(B) $\bar{B} \subset \bar{A}$  
(C) $A\bar{B} = \phi$  
(D) $\bar{A}B = \phi$

2、设 $A, B$ 为任意两事件，则下列选项错误的是（ ）。  
(A) 若 $AB = \phi$，则 $\bar{A}, \bar{B}$ 可能不相容  
(B) 若 $AB \neq \phi$，则 $\bar{A}, \bar{B}$ 可能相容  
(C) 若 $AB = \phi$，则 $\bar{A}, \bar{B}$ 可能相容  
(D) 若 $AB \neq \phi$，则 $\bar{A}, \bar{B}$ 一定不相容

3、设 $A, B, C$ 为三个事件，用 $A, B, C$ 的运算关系表示下列各事件：  
(1) 只有 $A$ 发生；  
(2) $A, B$ 都发生，而 $C$ 不发生；  
(3) $A, B, C$ 同时发生；  
(4) $A, B, C$ 至少有一个发生；  
(5) $A, B, C$ 至少有两个发生；  
(6) $A, B, C$ 只有一个发生；  
(7) $A, B, C$ 不多于一个发生。

4、设 $A, B$ 为任意两事件，且 $A \subset B$，$P(B) > 0$，则下列选项必然成立的是（ ）。  
(A) $P(A) < P(A|B)$  
(B) $P(A) \le P(A|B)$  
(C) $P(A) > P(A|B)$  
(D) $P(A) \ge P(A|B)$

5、设事件 $A, B$ 满足 $0 < P(A) < 1, 0 < P(B) < 1$, 且 $P(B|A) = 1$, 则（）.  
(A) $P(A|B) = 1$       (B) $P(\bar{A}|B) = 1$  
(C) $P(A|\bar{B}) = 0$   (D) $P(\bar{A}|\bar{B}) = 0$

6、设事件 $A, B$ 满足 $0 < P(A) < 1, 0 < P(B) < 1, P(A|B) + P(\bar{A}|\bar{B}) = 1$, 则（）.  
(A) 事件 $A$ 和事件 $B$ 不相容  
(B) 事件 $A$ 和事件 $B$ 互相对立  
(C) 事件 $A$ 和事件 $B$ 相互独立  
(D) 事件 $A$ 和事件 $B$ 不独立

7、设 $A, B, C$ 三个事件两两独立，则 $A, B, C$ 相互独立的充要条件是（）.  
(A) $A$ 和 $BC$ 独立          (B) $AB$ 和 $A \cup C$ 独立  
(C) $AB$ 和 $AC$ 独立          (D) $A \cup B$ 和 $A \cup C$ 独立

8、设事件 $A, B$ 相互独立，且 $P(A) = 0.4$, 则 $P(\bar{A}|\bar{B}) =$ \_\_\_\_\_.

9、设事件 $A, B$ 满足 $P(A) = 0.5, P(A+B) = 0.8$, 且 $A, B$ 互不相容，则 $P(B) =$ \_\_\_\_\_.

10、设事件 $A$ 满足 $P(A) = 0.8, P(A-B) = 0.1$, 则 $P(\overline{AB}) =$ \_\_\_\_\_.

11、设事件 $A, B$ 满足 $P(AB) = P(\bar{A}\bar{B}), P(A) = k$, 则 $P(B) =$ \_\_\_\_\_.

## 第 287 页

12、设事件 $A, B, C$ 满足 $A \supset B, A \supset C$, 且 $P(A) = 0.9, P(\bar{B} \cup \bar{C}) = 0.8$, 则 $P(A - BC) =$ \_\_\_\_\_.

13、设事件 $A, B$ 满足 $P(A) = 0.4, P(B) = 0.3, P(A+B) = 0.6$, 则 $P(A\bar{B}) =$ \_\_\_.

14、设事件 $A, B, C$ 满足 $P(A) = P(B) = P(C) = \frac{1}{4}, P(AB) = 0, P(AC) = P(BC) = \frac{1}{16}$, 则 $A, B, C$ 全不发生的概率为 \_\_\_\_\_.

15、设事件 $A, B$ 满足 $P(A) = \frac{1}{4}, P(A|B) = \frac{1}{2}, P(B|A) = \frac{1}{3}$, 则 $P(A \cup B) =$ \_\_\_.

16、设 100 件产品中，有 60 件正品，40 件次品，事件 $A = \{ 3\text{件均为次品} \}$, 事件 $B = \{ 3\text{件中两正品一次品} \}$.  
(1) 采取放回抽样方式，从产品中一次取 1 件，任取 3 件，求 $P(A)$ 和 $P(B)$；  
(2) 采取不放回抽样方式，从产品中一次取 1 件，任取 3 件，求 $P(A)$ 和 $P(B)$；  
(3) 从产品中一次性取 3 件，求 $P(A)$ 和 $P(B)$.

17、某人向同一目标重复射击，每次命中目标的概率均为 $P$, 则此人第 $k (k \ge 3)$ 次射击恰好第 3 次命中目标的概率为 \_\_\_\_\_.

18、某人向同一目标独立地进行 4 次射击，每次命中目标的概率均相等，若至少命中一次的概率为 $\frac{80}{81}$, 则此人最多命中 2 次的概率为 \_\_\_\_\_.

## 第 288 页

19、在区间 $[0, 1]$ 上任取两个数，则事件 $A = \{ \text{两数之和小于等于} \frac{1}{2} \}$ 的概率为 \_\_\_\_\_.

20、在区间 $[0, 1]$ 上任取两个数 $x$ 和 $y$, 则事件 $A = \{ |x-y| \le \frac{1}{2} \}$ 的概率为 \_\_\_\_\_.

21、有 3 个箱子，第一个箱子中有 4 个黑球和 1 个白球，第二个箱子中有 3 个黑球和 3 个白球，第三个箱子中有 3 个黑球和 5 个白球。现在先随机地取 1 个箱子，再从这个箱子中随机取出 1 个球。  
(1) 求取出的球是白球的概率；  
(2) 若已知取出的球是白球，求它来自第二个箱子的概率。
