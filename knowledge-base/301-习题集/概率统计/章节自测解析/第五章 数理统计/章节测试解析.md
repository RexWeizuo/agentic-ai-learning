(1)【7分】设总体$X$服从参数为$\lambda$的指数分布，$X_1, X_2, \cdots, X_n$为来自总体$X$的简单随机样本，$\overline{X}$为样本均值，则
$$E\left[\sum_{i=1}^{n}\left(X_{i}-\overline{X}\right)^{2}\right]=\quad(\quad).$$
(A) $\frac{n}{\lambda}$
(B) $\frac{n}{\lambda^2}$
(C) $\frac{n-1}{\lambda}$
(D) $\frac{n-1}{\lambda^2}$
【答案】(D)
【解析】
因为总体$X$服从参数为$\lambda$的指数分布，故$DX=\frac{1}{\lambda^2}$，记样本方差$S^2=\frac{1}{n-1}\sum_{i=1}^{n}(X_i-\overline{X})^2$，则$E(S^2)=DX=\frac{1}{\lambda^2}$。
因此$E\left[\sum_{i=1}^{n}\left(X_{i}-\overline{X}\right)^{2}\right]=E\left[(n-1)\cdot\frac{1}{n-1}\sum_{i=1}^{n}\left(X_{i}-\overline{X}\right)^{2}\right]=E\left[(n-1)\cdot S^2\right]=(n-1)E(S^2)=\frac{n-1}{\lambda^2}$。答案选(D)。

(2)【7分】设$X_1, X_2, \cdots, X_n$是来自总体$X \sim P(\lambda)$的简单随机样本，统计量$T=\frac{1}{n}\sum_{i=1}^{n}X_i(X_i-1)$的数学期望$E(T)$为
$(\quad)$。
(A) $\lambda^2$
(B) $\lambda(\lambda-1)$
(C) $\lambda^2-1$
(D) $\lambda$
【答案】(A)
【解析】
$$ET=E\left[\frac{1}{n}\sum_{i=1}^{n}X_i(X_i-1)\right]=\frac{1}{n}\sum_{i=1}^{n}E\left(X_i^2-X_i\right)=\frac{1}{n}\sum_{i=1}^{n}\left(EX_i^2-EX_i\right)$$
$$=\frac{1}{n}\sum_{i=1}^{n}\left[\left(EX_i\right)^2+DX_i-EX_i\right]=\frac{1}{n}\sum_{i=1}^{n}\left[\lambda^2+\lambda-\lambda\right]=\frac{1}{n}\cdot n\cdot \lambda^2=\lambda^2.$$
因此，答案选(A)。

(8)【8分】设随机变量$X \sim t(n)$，$Y \sim F(1,n)$，常数$C$满足$P\{X > C\}=0.6$，则$P\{Y > C^2\}=$______.
【答案】0.8
【解析】
因$X \sim t(n)$，故存在相互独立的$U \sim N(0,1)$，$V \sim \chi^2(n)$，使得$X=\frac{U}{\sqrt{V/n}}$，考虑$X^2=\frac{U^2}{V/n}$，因为$U^2 \sim \chi^2(1)$，
$U,V$独立，故$X^2=\frac{U^2}{V/n}=\frac{U^2/1}{V/n} \sim F(1,n)$.
由于$t(n)$分布的概率密度函数是偶函数，而$P\{X > C\}=0.6 > 0.5$，故$C < 0$.所以，
$P\{Y > C^2\}=P\{X > -C\}+P\{X < C\}=2P\{X < C\}=2[1-P\{X \ge C\}]=2[1-P\{X > C\}]$
$=2(1-0.6)=0.8$.答案为0.8.

(9)【8分】设$X_1, X_2, \cdots, X_n$为来自正态总体$X \sim N(\mu_1, \sigma^2)$的一个简单随机样本，记$\overline{X}=\frac{1}{n}\sum_{i=1}^n X_i$，
$S^2=\frac{1}{n-1}\sum_{i=1}^n (X_i-\overline{X})^2, T=\overline{X}^2-\frac{1}{n}S^2$，求$E(T)=$______.
【答案】$\mu^2$
【解析】
$ET=E\left(\overline{X}^2-\frac{1}{n}S^2\right)=E\overline{X}^2-\frac{1}{n}ES^2=D\overline{X}+(E\overline{X})^2-\frac{1}{n}ES^2=\frac{\sigma^2}{n}+\mu^2-\frac{\sigma^2}{n}=\mu^2$.

(10)【8分】设$X_1, X_2, \cdots, X_n$是来自总体$X \sim N(\mu, \sigma^2)$的简单随机样本，记样本方差为$S^2$，则$DS^2=$______.
【答案】$\frac{2}{n-1}\sigma^4$
【解析】
因为样本方差满足$\frac{(n-1)S^2}{\sigma^2} \sim \chi^2(n-1)$，则有$D\left(\frac{(n-1)S^2}{\sigma^2}\right)=2(n-1)=\frac{(n-1)^2}{\sigma^4}DS^2$，
所以$DS^2=\frac{2}{n-1}\sigma^4$.答案为$\frac{2}{n-1}\sigma^4$.

(11)【15分】设总体$X$服从正态分布$N(0,1)$，$X_1, X_2, \cdots, X_n$是来自总体$X$的简单随机样本，$\overline{X}$为样本均值，
设$Y_i = X_i + X_{i-1} - 2\overline{X} (i=2,3,\cdots,n)$. 求：
(1) $Y_i$的期望$EY_i$和方差$DY_i (i=2,3,\cdots,n)$；
(2) $Y_2$与$Y_3$的协方差$Cov(Y_2, Y_3)$.
【解析】
(1) 因$X_1, X_2, \cdots, X_n$独立同服从$N(0,1)$分布，故$EX_i = E\overline{X} = 0, DX_i = 1, D\overline{X} = \frac{1}{n}$.
所以$EY_i = E\left(X_i + X_{i-1} - 2\overline{X}\right) = EX_i + EX_{i-1} - 2E\overline{X} = 0$
$D\left(X_i + X_{i-1} - 2\overline{X}\right) = D\left(X_i + X_{i-1} - \frac{2}{n}X_1 - \frac{2}{n}X_2 - ... - \frac{2}{n}X_n\right)$
$= D\left(-\frac{2}{n}X_1 - \frac{2}{n}X_2 - ... + \frac{n-2}{n}X_{i-1} + \frac{n-2}{n}X_i - ... - \frac{2}{n}X_n\right)$
$= \frac{4}{n^2}DX_1 + \frac{4}{n^2}DX_2 + ... + \frac{(n-2)^2}{n^2}DX_{i-1} + \frac{(n-2)^2}{n^2}DX_i + ... + \frac{4}{n^2}DX_n$
$= \frac{4}{n^2} \cdot (n-2) + \frac{(n-2)^2}{n^2} \cdot 2 = \frac{2n-4}{n}$.
(2) $Cov(Y_2, Y_3) = Cov\left(X_2 + X_1 - 2\overline{X}, X_3 + X_2 - 2\overline{X}\right)$
$= Cov(X_2, X_3) + Cov(X_2, X_2) - 2Cov\left(X_2, \overline{X}\right) + Cov(X_1, X_3) + Cov(X_1, X_2) - 2Cov\left(X_1, \overline{X}\right)$
$- 2Cov\left(\overline{X}, X_3\right) - 2Cov\left(\overline{X}, X_2\right) + 4Cov\left(\overline{X}, \overline{X}\right)$
① 因为$X_1, X_2, X_3$相互独立，所以$Cov(X_2, X_3) = 0, Cov(X_1, X_3) = 0, Cov(X_1, X_2) = 0$.
② $Cov(X_i, \overline{X}) = Cov\left(X_i, \frac{1}{n}X_1 + ... + \frac{1}{n}X_i + ... + \frac{1}{n}X_n\right)$
$= \frac{1}{n}Cov(X_i, X_1) + ... + \frac{1}{n}Cov(X_i, X_i) + ... + \frac{1}{n}Cov(X_i, X_n)$
由于$X_1, X_2, \cdots, X_n$相互独立，因此有$Cov(X_i, X_j) = 0, j \neq i$.
则$Cov(X_i, \overline{X}) = \frac{1}{n}Cov(X_i, X_i) = \frac{1}{n}DX_i = \frac{1}{n}$.
故$Cov\left(X_2, \overline{X}\right) = Cov\left(X_1, \overline{X}\right) = Cov\left(\overline{X}, X_3\right) = Cov\left(\overline{X}, X_2\right) = \frac{1}{n}$.
