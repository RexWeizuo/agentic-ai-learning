# 习题

## 第 374 页

第八章 假设检验

1. 假设检验中的第一类错误指的是（ ）。  
   (A) $H_0$ 为真，检验结果是拒绝 $H_0$。  
   (B) $H_0$ 为假，检验结果是接受 $H_0$。  
   (C) $H_1$ 为真，检验结果是拒绝 $H_1$。  
   (D) $H_1$ 为假，检验结果是接受 $H_1$。
2. 在正态总体 $N(\mu, \sigma^2)$ 的均值 $\mu$ 检验中，若 $\sigma^2$ 已知，则检验统计量为______；若 $\sigma^2$ 未知，则检验统计量为______。
3. 某手表厂生产的男表表壳在正常情况下，其直径（单位：mm）服从正态分布 $N(20, 1)$。在某天的生产过程中，随机抽查4只表壳，测得直径分别为19.5，19.8，20，20.5。问在显著性水平 $\alpha = 0.05$ 下，这天生产的表壳直径的均值是否正常？  
   参考数据：$t_{0.05}(3) = 3.182, U_{0.025} = 1.96$。
4. 对正态总体 $N(\mu, \sigma^2)$ 的均值 $\mu$ 进行检验，原假设 $H_0: \mu = 2000$；备择假设 $H_1: \mu \neq 2000$。现抽取一个容量 $n=9$ 的样本，已知样本均值 $\bar{x} = 1900$，样本标准差 $S = 490$。则在显著性水平 $\alpha = 0.05$ 下，是否接受原假设？
