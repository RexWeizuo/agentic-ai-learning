# 习题

## 第 119 页

第六章 二重积分  
1、比较下列二重积分的大小：  
$I_1 = \iint_D (x+y)^2 d\sigma$ 与 $I_2 = \iint_D (x+y)^3 d\sigma$，  
其中，$D$ 是由 $x$ 轴，$y$ 轴和直线 $x+y=1$ 所围成的区域。

2、比较下列二重积分的大小：  
$I_1 = \iint_D \sin^2(x+y) d\sigma$ 与 $I_2 = \iint_D (x+y)^2 d\sigma$，  
其中，$D$ 是平面上任一有界闭区域。

3、设函数 $f(x,y)$ 在区域 $D$ 内连续，又区域 $D_r = \{(x,y) | (x-x_0)^2 + (y-y_0)^2 \leqslant r^2\}$，$(x_0, y_0) \in D$，求极限 $\lim_{r \to 0^+} \frac{1}{\pi r^2} \iint_{D_r} f(x,y) dxdy$。

4、计算二重积分 $\iint_D (x^2+y^2) dxdy$，其中 $D = \{(x,y) | |x| \leqslant 1, |y| \leqslant 1\}$。

5、计算二重积分 $\iint_D (xy^2 + e^{x+2y}) dxdy$，其中 $D = \{(x,y) | -1 \leqslant x \leqslant 1, -1 \leqslant y \leqslant 1\}$。

6、计算二重积分 $\iint_D xy e^{xy^2} dxdy$，其中 $D = \{(x,y) | 0 \leqslant x \leqslant 1, 0 \leqslant y \leqslant 1\}$。

7、计算二重积分 $\iint_D x \cos(x+y) dxdy$，其中 $D$ 是顶点为 $(0,0), (\pi,0)$ 和 $(\pi,\pi)$ 的三角形闭区域。

## 第 120 页

8、计算二重积分 $\iint_D xy dxdy$，其中 $D$ 是由抛物线 $y^2=x$ 与直线 $y=x-2$ 所围成的闭区域。

9、交换下列二次积分的积分次序。  
(1) $\int_0^1 dy \int_y^{\sqrt{y}} f(x,y) dx$；  
(2) $\int_0^1 dx \int_0^{x^2} f(x,y) dy + \int_1^2 dx \int_0^{2-x} f(x,y) dy$；  
(3) $\int_{-2}^1 dy \int_{y^2}^{2-y} f(x,y) dx$。

10、计算二次积分 $\int_0^1 dy \int_{\sqrt{y}}^1 \sqrt{1-x^4} dx$。

11、计算二次积分 $\int_0^\pi dx \int_x^\pi \frac{\sin y}{y} dy$。

12、计算二次积分 $\int_0^1 dy \int_{3y}^3 e^{x^2} dx$。

13、计算二次积分 $\int_0^2 dx \int_x^2 2y^2 \sin(xy) dy$。

14、计算二重积分 $\iint_D (x^2 \tan x + y^3 + 4) dxdy$，其中 $D = \{(x,y) | x^2+y^2 \leqslant 4\}$。

15、计算二重积分 $\iint_D (|x|+|y|) dxdy$，其中 $D = \{(x,y) | |x|+|y| \leqslant 1\}$。

16、记平面区域 $D = \{(x,y) | 0 \leqslant x \leqslant 1, 0 \leqslant y \leqslant 1\}$，计算二重积分

$$
I = \iint_D \frac{a f(x) + b f(y)}{f(x) + f(y)} dxdy
$$

## 第 121 页

其中常数 $a, b > 0$，$y = f(x)$ 为定义在 $(-\infty, +\infty)$ 上的连续正值函数。

17. 将二重积分 $\iint f(x, y) d\sigma$ 化为极坐标形式下的二次积分，其中 $D$ 为：  
    (1) $\{ (x, y) \mid x^2 + y^2 \leqslant 2x \}$；  
    (2) $\{ (x, y) \mid 1 \leqslant x^2 + y^2 \leqslant 4 \}$；  
    (3) $\{ (x, y) \mid 0 \leqslant x \leqslant 1, 0 \leqslant y \leqslant 1-x \}$；  
    (4) $\{ (x, y) \mid x^2 + y^2 \leqslant 2(x+y) \}$；  
    (5) $\{ (x, y) \mid 2x \leqslant x^2 + y^2 \leqslant 4 \}$.
18. 计算二重积分 $\iint \sqrt{4 - x^2 - y^2} dx dy$，其中 $D = \{ (x, y) \mid x^2 + y^2 \leqslant 2x \}$.
19. 计算二重积分 $\iint \arctan \frac{y}{x} dx dy$，其中 $D = \{ (x, y) \mid 1 \leqslant x^2 + y^2 \leqslant 4, 0 \leqslant y \leqslant x \}$.
20. 计算二重积分 $\iint xy dx dy$，其中 $D = \{ (x, y) \mid x^2 + y^2 \geqslant 1, x^2 + y^2 \leqslant 2x, x \geqslant 0, y \geqslant 0 \}$.
