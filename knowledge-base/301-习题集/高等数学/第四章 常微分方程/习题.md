# 习题

## 第 80 页

第四章 常微分方程

1、求下列微分方程的通解。  
(1) $x^2 y' + y = 0$；  
(2) $x \sec y \, dx + (x+1) dy = 0$；  
(3) $(x^2+1) \frac{dy}{dx} = xy$；  
(4) $y y' + e^{y^2+3x} = 0$。

2、求解下列微分方程的初值问题。  
(1) $\frac{dy}{dx} = y^2+1, \quad y(1)=0$；  
(2) $\frac{dy}{dx} = \frac{xy+3x}{x^2+1}, \quad y(2)=2$。

3、求下列微分方程的通解。  
(1) $x \frac{dy}{dx} = y + x \sec \frac{y}{x}$；  
(2) $(x+y) dx + (y-x) dy = 0$；  
(3) $x y' - y - x e^{\frac{y}{x}} = 0$；  
(4) $(x + \sqrt{x^2+y^2}) y' = y, \quad x>0, \quad y>0$。

4、求下列微分方程的通解。  
(1) $y' - 3y = e^x$；  
(2) $y' + 4y = x$；  
(3) $y' - 2xy = x$；  
(4) $\frac{dy}{dx} + y \cos x = e^{-\sin x}$；

## 第 81 页

(5) $dx + (x+y^2) dy = 0$。

5、求解下列微分方程的初值问题。  
(1) $y' + y = x + e^x, \quad y(0)=0$；  
(2) $x y' - 3y = x^2, \quad x>0, \quad y(1)=1$；  
(3) $t \frac{dx}{dt} = -x + \sin t, \quad x(\pi)=1$。

6、求下列微分方程的通解。  
(1) $y'' = x + \cos x$；  
(2) $y'' = \frac{1}{1+x^2}$；  
(3) $4xy'' - y'' - 4y' = 0$；  
(4) $y y'' - (y')^2 = 0$；  
(5) $y'' = (y')^3 + y'$。

7、求下列微分方程的通解。  
(1) $y'' - 3y' + 2y = 0$；  
(2) $y'' - 2y' - 2y = 0$；  
(3) $y'' - 2y' = 0$；  
(4) $y'' + 6y' + 9y = 0$；  
(5) $y'' + 6y' + 10y = 0$；  
(6) $y''' - y'' - y' + y = 0$。
