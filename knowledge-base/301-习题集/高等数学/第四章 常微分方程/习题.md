# 习题

## 第 80 页

第四章 常微分方程

1、求下列微分方程的通解。  
(1) $x^2 y' + y = 0$；  
(2) $x \sec y \, dx + (x+1) dy = 0$；  
(3) $(x^2+1) \frac{dy}{dx} = xy$；  
(4) $y y' + e^{y^2+3x} = 0$。

> 💡 **1(4) 解法**：拆 $e^{y^2+3x} = e^{y^2} \cdot e^{3x}$ → $y e^{-y^2} dy = -e^{3x} dx$（可分离）。左边令 $u=y^2$ 积分。

> 💡 **1(2) 解法**：看似不像可分离，移项即现——
> $x \sec y \, dx = -(x+1) dy \Rightarrow \frac{x}{x+1} dx = -\cos y \, dy$
> 左边 $\frac{x}{x+1} = 1 - \frac{1}{x+1}$，积分得 $x - \ln|x+1| + \sin y = C$

2、求解下列微分方程的初值问题。  
(1) $\frac{dy}{dx} = y^2+1, \quad y(1)=0$；  
(2) $\frac{dy}{dx} = \frac{xy+3x}{x^2+1}, \quad y(2)=2$。

3、求下列微分方程的通解。  
(1) $x \frac{dy}{dx} = y + x \sec \frac{y}{x}$；  
(2) $(x+y) dx + (y-x) dy = 0$；  
(3) $x y' - y - x e^{\frac{y}{x}} = 0$；  
(4) $(x + \sqrt{x^2+y^2}) y' = y, \quad x>0, \quad y>0$。

> 💡 **3(4) 为什么用 dx/dy 不用 dy/dx**：$y' = \frac{y}{x+\sqrt{x^2+y^2}}$ 分子有 y、分母混 x,y → 直接设 $u=y/x$ 很丑。翻转成 $\frac{dx}{dy} = \frac{x}{y} + \sqrt{(\frac{x}{y})^2+1}$，每一项都是 $x/y$ 的函数 → 标准齐次型，设 $u=x/y$ 一步到位。**规则**：齐次方程哪边翻转更干净就用哪边。

4、求下列微分方程的通解。  
(1) $y' - 3y = e^x$；  
(2) $y' + 4y = x$；  
(3) $y' - 2xy = x$；  
(4) $\frac{dy}{dx} + y \cos x = e^{-\sin x}$；

## 第 81 页

(5) $dx + (x+y^2) dy = 0$。

> 💡 **4(5) 为什么用 dx/dy**：$\frac{dy}{dx} = -\frac{1}{x+y^2}$ 非线性的，没公式。翻转成 $\frac{dx}{dy} + x = -y^2$，标准一阶线性（x 是 y 的函数），$P(y)=1, Q(y)=-y^2$。**规则**：dy/dx 不好看就翻转，哪个方向变标准形就用哪个。

5、求解下列微分方程的初值问题。  
(1) $y' + y = x + e^x, \quad y(0)=0$；  
(2) $x y' - 3y = x^2, \quad x>0, \quad y(1)=1$；  
(3) $t \frac{dx}{dt} = -x + \sin t, \quad x(\pi)=1$。

6、求下列微分方程的通解。  
(1) $y'' = x + \cos x$；  
(2) $y'' = \frac{1}{1+x^2}$；  
(3) $4xy'' - y'' - 4y' = 0$；  
(4) $y y'' - (y')^2 = 0$；  
(5) $y'' = (y')^3 + y'$。

7、求下列微分方程的通解。  
(1) $y'' - 3y' + 2y = 0$；  
(2) $y'' - 2y' - 2y = 0$；  
(3) $y'' - 2y' = 0$；  
(4) $y'' + 6y' + 9y = 0$；  
(5) $y'' + 6y' + 10y = 0$；  
(6) $y''' - y'' - y' + y = 0$。
