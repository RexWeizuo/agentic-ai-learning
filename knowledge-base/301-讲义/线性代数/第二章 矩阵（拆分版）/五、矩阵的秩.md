# 五、矩阵的秩

### 4. 矩阵的秩

#### (1) 子式

在一 $m \times n$ 阶矩阵 $A$ 中，任取 $k$ 行 $k$ 列（$0 < k \leq \min\{m, n\}$）上的 $k^2$ 个元素，不改变其相对位置，得到一 $k$ 阶行列式，称为 $k$ 阶子式。

**例** $ \begin{pmatrix}  
1 & -1 & 0 & 0 \\  
0 & 2 & 1 & 0 \\  
0 & 1 & 2 & -1  
\end{pmatrix} $

对于一 $m \times n$ 阶矩阵，最高阶子式的阶数 = $\min\{m, n\}$。

若 $A$ 为 $n$ 阶方阵，$A$ 的最高阶子式为 $|A|$。

#### (2) 秩的定义

若矩阵 $A$ 存在 $r$ 阶子式 $\neq 0$，而所有 $r+1$ 阶子式（如果存在）全等于 0，则称矩阵 $A$ 的秩为 $r$，记作 $r(A) = r$。

① $r(A) = 0 \iff A = O$

② 对于 $A_{m \times n}$：

- 若 $r(A) = m$，$A$ 为行满秩矩阵
- 若 $r(A) = n$，$A$ 为列满秩矩阵

③ 若 $A$ 为 $n$ 阶方阵：

$$
\begin{cases}
r(A) = n \iff |A| \neq 0 \iff A \text{ 可逆} \\
r(A) < n \iff |A| = 0 \iff A \text{ 不可逆}
\end{cases}
$$

(4) 矩阵秩的重要结论

① $0 \leq r(A_{m \times n}) \leq \min\{m, n\}$

② $r(A) = r(A^T) = r(A^TA) = r(AA^T) = r(kA) \ (k \neq 0)$

③ 若 P 可逆，则 $r(PA) = r(AP) = r(A)$

④ $r(A + B) \leq r(A) + r(B)$

⑤ $\max\{r(A), r(B)\} \leq r(A, B) \leq r(A) + r(B)$

⑥ $r(AB) \leq \min\{r(A), r(B)\}$

⑦ 若 $A_{m \times n}B_{n \times s} = O$，则 $r(A) + r(B) \leq n$

⑧ 若 A 为 n 阶矩阵，$r(A^*) = \begin{cases} n, & r(A) = n, \\ 1, & r(A) = n-1, \\ 0, & r(A) < n-1. \end{cases}$
