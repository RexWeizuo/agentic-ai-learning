# 知识提要 常用工具

## 知识提要

一元函数的导数与微分计算相对于极限和不定积分计算而言，要简单不少，这块重点是需要大家掌握各类函数的求导规则，例如复合函数、隐函数、参数方程、反函数、极坐标、分段函数等等。

对于一元函数而言，可导与可微是等价的，例如函数 $y=f(x)$ 的导数可以表示为 $y'=\frac{dy}{dx}=f'(x)$，而 $y=f(x)$ 的微分形式表示为 $dy=f'(x)dx$。

因此，一元函数的导数与微分计算并无二致，求出来一个函数的导数，其微分形式自然也就有了，故本专题主要针对一元函数的导数计算进行讲解。

## （一）导数的定义

函数 $y=f(x)$ 在点 $x=x_0$ 处的导数记作 $f'(x_0)=\frac{dy}{dx}\bigg|_{x=x_0}=\lim_{\Delta x \to 0}\frac{f(x_0+\Delta x)-f(x_0)}{\Delta x}$，

函数 $y=f(x)$ 在点 $x=x_0$ 处可导，意味着在该点处左导数=右导数，即满足 $f'_+(x_0)=f'_-(x_0)$。

## （二）基本初等函数导数表

$(C)'=0$
$(x^n)'=nx^{n-1}$
$(\sin x)'=\cos x$
$(\cos x)'=-\sin x$
$(a^x)' = a^x \ln a$
$(e^x)' = e^x$
$(\log_a x)' = \frac{1}{x \ln a}$
$(\ln x)' = \frac{1}{x}$
$(\tan x)' = \sec^2 x$
$(\cot x)' = -\csc^2 x$
$(\sec x)' = \sec x \cdot \tan x$
$(\csc x)' = -\csc x \cdot \cot x$
$(\arcsin x)' = \frac{1}{\sqrt{1-x^2}}$
$(\arccos x)' = -\frac{1}{\sqrt{1-x^2}}$
$(\arctan x)' = \frac{1}{1+x^2}$
$(\text{arccot } x)' = -\frac{1}{1+x^2}$

## （三）导数与微分的四则运算

$(u \pm v)' = u' \pm v'$
$(ku)' = ku'$
$(uv)' = u'v + uv'$
$(\frac{u}{v})' = \frac{u'v - uv'}{v^2}$
$d(u \pm v) = du \pm dv$
$d(ku) = kdu$
$d(uv) = vdu + udv$
$d(\frac{u}{v}) = \frac{vdu - udv}{v^2}$
