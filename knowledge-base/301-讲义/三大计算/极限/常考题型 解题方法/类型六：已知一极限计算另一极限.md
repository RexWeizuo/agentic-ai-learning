这份文档聚焦于**已知一个含抽象函数** **$f(x)$** **的极限，求另一个含** **$f(x)$** **的极限**的类型。  
核心解法有两种策略：

1. **凑项法**：通过极限的四则运算法则，将已知极限与待求极限建立联系，然后提取出待求极限。
2. **待定系数 / 特殊函数法**：设待求极限为常数 $a$，反解出 $f(x)$ 的表达式代入已知条件求 $a$；或在选填题中直接取一个满足条件的特殊函数来算。

下面对【例 1.20】的三种方法逐一详解。

---

## 【例 1.20】

已知

$$
\lim_{x \to 0} \frac{e^x f(x) + \sin x}{x^2} = 1,
$$

求

$$
\lim_{x \to 0} \frac{f(x) + \sin x}{x^2}.
$$

### 分析

已知极限 $\frac{0}{0}$ 型，分子分母均趋于 0。未知函数 $f(x)$ 的具体形式，需要从已知极限中提取信息。

---

### 方法一：凑项法（借助极限四则运算法则）

我们希望从已知极限式中“剥离”出待求的 $\frac{f(x)+\sin x}{x^2}$。

已知极限：

$$
\lim_{x \to 0} \frac{e^x f(x) + \sin x}{x^2} = 1.
$$

将 $e^x f(x)$ 写成 $f(x) + (e^x-1)f(x)$，于是

$$
\frac{e^x f(x) + \sin x}{x^2} = \frac{f(x) + \sin x}{x^2} + \frac{(e^x-1)f(x)}{x^2}.
$$

令待求极限为 $A = \lim_{x \to 0} \frac{f(x)+\sin x}{x^2}$。

我们的目标是通过已知极限和 $f(x)$ 的其他信息求出 $A$。现在的问题是我们不知道 $\lim_{x \to 0} \frac{(e^x-1)f(x)}{x^2}$。

但我们还有可以利用的信息：从已知极限为 1 可知，分子 $e^x f(x) + \sin x \sim x^2$，且由于 $\sin x \sim x$，可见 $f(x)$ 的主项应能消掉 $e^x$ 中的常数项带来的低阶量。具体地，当 $x\to 0$ 时，$e^x = 1 + x + \frac{x^2}{2} + o(x^2)$，  
分母是 $x^2$，要使极限存在且非零，$f(x)$ 至少含有 $-x$ 项以抵消 $\sin x$ 带来的 $x$。

我们可以先通过已知极限求 $\lim_{x \to 0} \frac{f(x)}{x}$，或者更直接地把 $f(x)$ 渐近展开。

**更漂亮的凑项法**：  
由极限

$$
\lim_{x \to 0} \frac{e^x f(x) + \sin x}{x^2} = 1,
$$

得

$$
e^x f(x) + \sin x = x^2 + o(x^2).
$$

于是

$$
f(x) = e^{-x}\bigl( x^2 - \sin x + o(x^2) \bigr).
$$

代入待求极限：

$$
\frac{f(x)+\sin x}{x^2} = \frac{e^{-x}(x^2 - \sin x) + \sin x + o(x^2)}{x^2}.
$$

利用 $e^{-x} = 1 - x + \frac{x^2}{2} + o(x^2)$，$\sin x = x - \frac{x^3}{6} + o(x^3)$，计算：

$$
\begin{aligned}
e^{-x}(x^2 - \sin x) &= (1 - x + \frac{x^2}{2})(x^2 - x + \frac{x^3}{6}) + o(x^2) \\
&= (x^2 - x) - x(x^2 - x) + \frac{x^2}{2}(x^2 - x) + \cdots \quad \text{(只保留到 }x^2\text{)}
\end{aligned}
$$

更系统地：  
首先 $\sin x = x + o(x)$，所以 $x^2 - \sin x = -x + x^2 + o(x^2)$（注意符号，$x^2 - x$，高阶保留 $x^2$ 项）。乘 $e^{-x} = 1 - x + \frac{x^2}{2} + o(x^2)$：

$$
\begin{aligned}
(1 - x + \frac{x^2}{2})(-x + x^2) &= -x + x^2 + x^2 - x^3 + \frac{x^2}{2}(-x) + \cdots \\
&= -x + 2x^2 + o(x^2).
\end{aligned}
$$

所以

$$
f(x) = -x + 2x^2 + o(x^2).
$$

那么

$$
f(x) + \sin x = (-x + 2x^2) + (x - \frac{x^3}{6}) = 2x^2 + o(x^2),
$$

除以 $x^2$ 得 $2$。

**这个直接展开有点长，文档中的凑项法可能更简洁**：  
已知

$$
1 = \lim_{x \to 0} \frac{e^x f(x) + \sin x}{x^2}
    = \lim_{x \to 0} \left( \frac{f(x)+\sin x}{x^2} + \frac{(e^x-1)f(x)}{x^2} \right).
$$

我们需要求 $A = \lim \frac{f(x)+\sin x}{x^2}$，并求 $B = \lim \frac{(e^x-1)f(x)}{x^2}$。

因为 $\lim \frac{e^x-1}{x} = 1$，所以 $\frac{(e^x-1)f(x)}{x^2} = \frac{e^x-1}{x} \cdot \frac{f(x)}{x}$。  
若 $\lim \frac{f(x)}{x}$ 存在，则 $B = 1 \cdot \lim \frac{f(x)}{x}$。

现在找 $\lim \frac{f(x)}{x}$：由已知极限为 1，分子 $e^x f(x) + \sin x$ 必须与 $x^2$ 同阶，而 $\sin x \sim x$，这要求 $f(x)$ 提供 $-x$ 来抵消 $\sin x$。  
由已知极限式也可导出：

$$
1 = \lim_{x \to 0} \frac{e^x f(x) + \sin x}{x^2} 
    = \lim_{x \to 0} \frac{f(x) + x + o(x)}{x^2} \quad (\text{因为 } e^x = 1+x+\cdots,\ \sin x = x+\cdots).
$$

更精确：分子 $= f(x)(1+x+\frac{x^2}{2}+o(x^2)) + x - \frac{x^3}{6}+o(x^3)$  
$= f(x) + x f(x) + \frac{x^2}{2}f(x) + x + o(x^2)$ （保留到 $x^2$ 项，注意 $x f(x)$ 可能贡献 $x^2$）。

要使得比值趋于 1，则常数项和 $x$ 的一次项系数必须为 0：

- 常数项：$f(0) = 0$
- 一次项：$f(0) + 1 = 0$? 实际上 $f(x) = a_1 x + a_2 x^2 + \cdots$，代入：  
  $f(x) + x f(x) + \frac{x^2}{2} f(x) + x = (a_1 x + a_2 x^2) + x(a_1 x) + \cdots + x = (a_1+1)x + (a_2 + a_1)x^2 + \cdots$  
  要求一次项系数 $a_1+1=0 \Rightarrow a_1 = -1$，二次项系数为 $(a_2 + a_1)$ 应等于 1（因为分母 $x^2$，比值极限就是二次项系数）。所以 $a_2 -1 = 1 \Rightarrow a_2 = 2$。于是 $f(x) = -x + 2x^2 + o(x^2)$。那么 $\frac{f(x)}{x} = -1 + 2x \to -1$，则 $B = -1$。

由方程 $1 = A + B = A - 1$ 得 $A = 2$。

**凑项法的关键**：将已知极限拆成待求极限 $A$ 加上另一可计算的极限 $B$，通过已知值求得 $A$。

### 方法二：设待求极限为常数 $a$，反解 $f(x)$

设

$$
\lim_{x \to 0} \frac{f(x) + \sin x}{x^2} = a.
$$

则

$$
f(x) = a x^2 - \sin x + o(x^2).
$$

代入已知条件：

$$
\begin{aligned}
\lim_{x \to 0} \frac{e^x (a x^2 - \sin x + o(x^2)) + \sin x}{x^2} \\
= \lim_{x \to 0} \frac{e^x a x^2 - e^x \sin x + \sin x}{x^2} \\
= \lim_{x \to 0} \left( a e^x + \frac{\sin x (1 - e^x)}{x^2} \right).
\end{aligned}
$$

因为 $1 - e^x = -x - \frac{x^2}{2} + o(x^2)$，$\sin x = x + o(x^2)$，所以

$$
\frac{\sin x (1 - e^x)}{x^2} = \frac{(x + o(x^2))(-x - \frac{x^2}{2} + o(x^2))}{x^2} = \frac{-x^2 + o(x^2)}{x^2} = -1 + o(1).
$$

而 $a e^x \to a$，所以原式趋于 $a - 1$。已知该极限为 1，故 $a - 1 = 1$，即 $a = 2$。

此方法干净利落！

---

### 方法三：特殊函数法（仅适用于选填题）

直接构造一个满足已知条件的函数 $f(x)$。  
从已知条件可以猜出 $f(x)$ 的渐近形式 $f(x) = -x + 2x^2 + \cdots$，不妨直接取 $f(x) = -x + 2x^2$，验证：

$$
\frac{e^x(-x+2x^2) + \sin x}{x^2} = \frac{(1+x+x^2/2)(-x+2x^2) + x - x^3/6}{x^2} + o(1).
$$

计算分子：$(1+x)(-x+2x^2) = -x + 2x^2 - x^2 + 2x^3 = -x + x^2 + 2x^3$；加上 $\frac{x^2}{2}(-x) = -\frac{x^3}{2}$，得 $-x + x^2 + \frac{3}{2}x^3$；再加 $\sin x = x - \frac{x^3}{6}$，和 $= x^2 + (\frac{3}{2}-\frac{1}{6})x^3 = x^2 + \frac{4}{3}x^3$。除以 $x^2$ 得 $1 + \frac{4}{3}x \to 1$。满足条件。然后代入待求式：

$$
\frac{f(x)+\sin x}{x^2} = \frac{-x+2x^2 + x}{x^2} = \frac{2x^2}{x^2} = 2.
$$

得到相同答案。此法虽不严谨，但对付客观题很高效。

---

## 小结

- **凑项法**是利用极限的线性运算，把已知极限拆成待求极限和可求极限的线性组合。
- **待定系数法**直接从待求极限设出未知常数，把 $f(x)$ 用泰勒展开表示，代回已知极限解方程，逻辑直接。
- **特殊函数法**（选填技巧）利用极限只依赖于局部展开的性质，构造简单多项式函数验证并计算。

这类题目最终的答案往往是靠展开系数比较得出的，因此对函数的泰勒展开熟练度有较高要求。
