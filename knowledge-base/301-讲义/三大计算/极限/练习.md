## 1. 极限 $\displaystyle \lim_{x \to 0} \frac{\arcsin \frac{x}{\sqrt{1-x^2}}}{\sin x + \cos x - 1}$

**类型一：求** **$\frac{0}{0}$** **型未定式的极限**

**解析**  
当 $x\to0$ 时，分子中 $\frac{x}{\sqrt{1-x^2}}\sim x$，$\arcsin x \sim x$，故分子 $\sim x$。  
分母：$\sin x + \cos x - 1$，泰勒展开：  
$\sin x = x - \frac{x^3}{6}+o(x^3)$,  
$\cos x = 1 - \frac{x^2}{2} + o(x^2)$，

$$
\sin x + \cos x - 1 = \left(x - \frac{x^3}{6}\right) + \left(1 - \frac{x^2}{2}\right) - 1 
= x - \frac{x^2}{2} - \frac{x^3}{6} + o(x^2) = x + o(x)
$$

因此分子 $\sim x$，分母 $\sim x$，极限为 $1$。  
更严谨：$\arcsin \frac{x}{\sqrt{1-x^2}} = \arctan x$（因为令 $x=\sin\theta$，则 $\arcsin(\tan\theta)$? 实际上恒等式：$\arcsin \frac{x}{\sqrt{1-x^2}} = \arctan x$ 在 $|x|<1$ 成立）。于是分子就是 $\arctan x \sim x$。分母 $\sim x$，所以极限为 $1$。

**答案**：$1$

---

## 2. 极限 $\displaystyle \lim_{x \to 0} \frac{1 - \cos \sqrt{\tan x - \sin x}}{\sqrt[3]{1+x^3} - \sqrt[3]{1-x^3}}$

**类型一：求** **$\frac{0}{0}$** **型未定式的极限**

**解析**  
先处理分母：$\sqrt[3]{1+x^3} - \sqrt[3]{1-x^3} \sim \frac{1}{3}x^3 - (-\frac{1}{3}x^3) = \frac{2}{3}x^3$（利用 $(1+u)^\alpha -1 \sim \alpha u$）。  
分子：$\tan x - \sin x = \tan x(1-\cos x) \sim x \cdot \frac{x^2}{2} = \frac{x^3}{2}$，故 $\sqrt{\tan x - \sin x} \sim \sqrt{\frac{x^3}{2}}$。  
于是 $1 - \cos \bigl(\sqrt{\tan x - \sin x}\bigr) \sim \frac{1}{2}\bigl(\sqrt{\tan x - \sin x}\bigr)^2 = \frac{1}{2}(\tan x - \sin x) \sim \frac{1}{2} \cdot \frac{x^3}{2} = \frac{x^3}{4}$。  
所以原式 $\sim \frac{x^3/4}{(2/3)x^3} = \frac{1/4}{2/3} = \frac{3}{8}$。

**答案**：$\displaystyle \frac{3}{8}$

## 3. 极限 $\displaystyle \lim_{x \to +\infty} \frac{\ln(x + \sqrt{x^2+1}) - \ln(x + \sqrt{x^2-1})}{e^{\frac{1}{x}} - \ln\left(\frac{x+1}{x}\right) - 1}$

**类型一：求** **$\frac{0}{0}$** **型未定式的极限**（本质是 $0/0$，通过倒代换或展开处理）

**解析**  
首先观察分子：两个对数差。当 $x\to+\infty$，$\sqrt{x^2\pm1}=x\sqrt{1\pm\frac1{x^2}} = x\bigl(1\pm\frac1{2x^2} + o(\frac1{x^2})\bigr) = x \pm \frac1{2x} + o(1/x)$。  
所以 $x+\sqrt{x^2+1} \approx 2x + \frac1{2x}$，$x+\sqrt{x^2-1} \approx 2x - \frac1{2x}$。  
分子：$\ln(2x+\frac1{2x}) - \ln(2x-\frac1{2x}) = \ln\frac{2x+\frac1{2x}}{2x-\frac1{2x}} = \ln\frac{1+\frac1{4x^2}}{1-\frac1{4x^2}}$ 用 $\ln\frac{1+u}{1-u} \sim 2u$（当 $u\to0$），这里 $u = \frac1{4x^2}$，所以分子 $\sim 2\cdot \frac1{4x^2} = \frac1{2x^2}$。  
分母：$e^{1/x} - \ln(1+\frac1x) - 1$。展开：$e^{1/x}=1+\frac1x+\frac1{2x^2}+o(1/x^2)$，$\ln(1+1/x)=\frac1x - \frac1{2x^2}+o(1/x^2)$。  
所以分母 $= \bigl(1+\frac1x+\frac1{2x^2}\bigr) - \bigl(\frac1x - \frac1{2x^2}\bigr) -1 + o(1/x^2) = \frac1{2x^2} + \frac1{2x^2} + o(1/x^2) = \frac1{x^2} + o(1/x^2)$。  
因此原式 $\sim \frac{1/(2x^2)}{1/x^2} = \frac12$。

**答案**：$\displaystyle \frac{1}{2}$

## 4. 极限 $\displaystyle \lim_{x \to 0} \frac{e^{x^2} - e^{2-2\cos x}}{(x+x^2)(\sin x - \tan x)}$

**类型一：求** **$\frac{0}{0}$** **型未定式的极限**（利用了 $e^\square - e^\triangle$ 技巧和等价替换）

**解析**  
先处理分母：$(x+x^2) \sim x$（保留主部），$\sin x - \tan x = \sin x - \frac{\sin x}{\cos x} = \sin x \frac{\cos x-1}{\cos x} \sim x\cdot\frac{-x^2/2}{1} = -\frac{x^3}{2}$。所以分母 $\sim x \cdot (-\frac{x^3}{2}) = -\frac{x^4}{2}$。  
分子：$e^{x^2} - e^{2-2\cos x}$。利用 $e^A - e^B = e^B(e^{A-B}-1)$。$B = x^2$ 和 $A=2-2\cos x$？比较大小：当 $x\to0$，$x^2$ 和 $2-2\cos x = x^2 - \frac{x^4}{12}+...$ 都是 $x^2$ 阶。为提取公因子，可提 $e^{x^2}$ 或 $e^{2-2\cos x}$。不妨提 $e^{x^2}$：$e^{2-2\cos x} - e^{x^2} = e^{x^2}(e^{2-2\cos x - x^2}-1)$。指数差：$2-2\cos x - x^2 = 2(1-\cos x) - x^2 = 2\bigl(\frac{x^2}{2} - \frac{x^4}{24}+o(x^4)\bigr) - x^2 = (x^2 - \frac{x^4}{12}) - x^2 = -\frac{x^4}{12}+o(x^4)$。  
于是 $e^{2-2\cos x - x^2} - 1 \sim -\frac{x^4}{12}$，乘以 $e^{x^2}\to 1$，分子 $\sim -\frac{x^4}{12}$。  
原式 $\sim \frac{-x^4/12}{-x^4/2} = \frac{1}{6}$。

**答案**：$\displaystyle \frac{1}{6}$

## 5. 极限 $\displaystyle \lim_{x \to 0} \frac{1 - \cos x \cdot \sqrt{\cos 2x}}{x^2}$

**类型一：求** **$\frac{0}{0}$** **型未定式的极限**

**解析**  
分子有理化？不必。直接泰勒展开。  
$\cos x = 1 - \frac{x^2}{2} + \frac{x^4}{24} + o(x^4)$。  
$\cos 2x = 1 - \frac{(2x)^2}{2} + \frac{(2x)^4}{24} + o(x^4) = 1 - 2x^2 + \frac{2}{3}x^4 + o(x^4)$。  
$\sqrt{\cos 2x} = 1 - x^2 + \frac{1}{2}\cdot\frac{2}{3}x^4 - \frac{1}{8}(-2x^2)^2$? 可直接用 $(1+u)^{1/2}=1+\frac12 u - \frac18 u^2+...$，其中 $u = -2x^2 + \frac23 x^4 +...$  
$\sqrt{\cos 2x} = 1 + \frac12(-2x^2 + \frac23 x^4) - \frac18(-2x^2)^2 + o(x^4) = 1 - x^2 + \frac13 x^4 - \frac12 x^4 + o(x^4) = 1 - x^2 - \frac16 x^4 + o(x^4)$。  
乘积：$\cos x \sqrt{\cos 2x} = \bigl(1 - \frac{x^2}{2} + \frac{x^4}{24}\bigr)\bigl(1 - x^2 - \frac16 x^4\bigr) + o(x^4)$  
$= 1 - x^2 - \frac16 x^4 - \frac12 x^2 + \frac12 x^4 + \frac{x^4}{24} + o(x^4)$ （注意：$- \frac{x^2}{2}\cdot (-x^2) = +\frac12 x^4$）  
$= 1 - \frac32 x^2 + \bigl(-\frac16 + \frac12 + \frac1{24}\bigr)x^4 + o(x^4) = 1 - \frac32 x^2 + \bigl(\frac{-4+12+1}{24}\bigr)x^4 = 1 - \frac32 x^2 + \frac{9}{24}x^4 = 1 - \frac32 x^2 + \frac38 x^4 + o(x^4)$。  
所以 $1 - \cos x\sqrt{\cos 2x} = \frac32 x^2 - \frac38 x^4 + o(x^4)$。  
除以 $x^2$ 得 $\frac32 - \frac38 x^2 \to \frac32$。

**答案**：$\displaystyle \frac{3}{2}$

## 6. 极限 $\displaystyle \lim_{x \to \infty} \frac{(x+a)^{x+a}(x+b)^{x+b}}{(x+a+b)^{2x+a+b}}$

**类型三：求幂指型（**​**$1^\infty$** **等）未定式的极限**（实际上是幂指函数取对数化为 $0/0$ 或 $\infty/\infty$，或者通过拆项取极限）

**解析**  
这类极限通常取对数后处理。令原式为 $L$。取对数：

$$
\ln L = \lim_{x\to\infty} \Big[ (x+a)\ln(x+a) + (x+b)\ln(x+b) - (2x+a+b)\ln(x+a+b) \Big].
$$

将各项对 $x$ 展开，提取主部。我们利用 $\ln(x+c) = \ln x + \ln(1+\frac{c}{x}) = \ln x + \frac{c}{x} - \frac{c^2}{2x^2} + \cdots$。  
计算：  
$(x+a)\ln(x+a) = (x+a)\left(\ln x + \frac{a}{x} - \frac{a^2}{2x^2} + \cdots\right) = x\ln x + a\ln x + a + \frac{a^2}{x} - \frac{a^2}{2x} - \frac{a^3}{2x^2} + \cdots$。不够系统，我们可以用更精确的泰勒直到常数项（因为 $x\to\infty$，有 $x\ln x$ 量级）。一种常见技巧是先提出 $x\ln x$ 项，但三项相消后可能会有常数项。可以直接把三项写为：  
$(x+a)\ln(x+a) = (x+a)\ln x + (x+a)\ln(1+a/x) = x\ln x + a\ln x + (x+a)(a/x - a^2/(2x^2) + a^3/(3x^3)...)$。  
同理处理其他项。最终会发现 $\ln L$ 趋于某个常数。具体：

$$
\begin{aligned}
&(x+a)\ln(x+a)+(x+b)\ln(x+b)-(2x+a+b)\ln(x+a+b)\\
&= x\ln\frac{(x+a)(x+b)}{(x+a+b)^2} + a\ln\frac{x+a}{x+a+b} + b\ln\frac{x+b}{x+a+b} + \text{可能的高阶项}
\end{aligned}
$$

更快的做法：原式改写为

$$
\frac{(x+a)^{x+a}(x+b)^{x+b}}{(x+a+b)^{2x+a+b}} = \left( \frac{x+a}{x+a+b} \right)^{x+a} \left( \frac{x+b}{x+a+b} \right)^{x+b}.
$$

令 $p = \frac{x+a}{x+a+b} = 1 - \frac{b}{x+a+b}$, $q = 1 - \frac{a}{x+a+b}$。当 $x\to\infty$，$p,q\to1$，指数 $\to\infty$，是 $1^\infty$ 型。用公式 $\lim (1+u)^v = e^{\lim v u}$（当 $u\to0$）。  
第一因子：$\left(1 - \frac{b}{x+a+b}\right)^{x+a}$，指数与 $u$ 的乘积为 $(x+a)\cdot \left(-\frac{b}{x+a+b}\right) \to -b$。  
第二因子：$(x+b)\cdot \left(-\frac{a}{x+a+b}\right) \to -a$。  
因此极限为 $e^{-b} \cdot e^{-a} = e^{-(a+b)}$。

**答案**：$\displaystyle e^{-(a+b)}$

## 7. 极限 $\displaystyle \lim_{x \to +\infty} \frac{\left(1+\frac{1}{x}\right)^{x^2}}{e^x}$

**类型三：求幂指型（**​**$1^\infty$** **等）未定式的极限**

**解析**  
标准处理：取对数，或利用 $\left(1+\frac1x\right)^{x^2} = e^{x^2\ln(1+1/x)}$。指数展开：  
$\ln(1+1/x) = \frac1x - \frac1{2x^2} + \frac1{3x^3} - \cdots$，故 $x^2\ln(1+1/x) = x - \frac12 + \frac1{3x} - \cdots$。  
所以 $\left(1+\frac1x\right)^{x^2} = e^{x - 1/2 + o(1)}$。  
于是原式 $= \frac{e^{x - 1/2 + o(1)}}{e^x} = e^{-1/2 + o(1)} \to e^{-1/2}$。

**答案**：$\displaystyle \frac{1}{\sqrt{e}}$ 或 $e^{-1/2}$

## 8. 极限 $\displaystyle \lim_{x \to 1} (1-x^2) \tan\left(\frac{\pi}{2}x\right)$

**类型二：求** **$0\cdot\infty$** **型未定式的极限**

**解析**  
当 $x\to1$，$1-x^2 \to 0$，$\tan(\frac{\pi}{2}x) \to \infty$（因为 $\frac{\pi}{2}x \to \frac{\pi}{2}$ 从左侧，正切趋于 $+\infty$；若 $x>1$ 则趋于 $-\infty$，需要讨论左右极限？通常 $x\to1$ 意味着考虑双侧，但 $\tan$ 在 $x=1$ 处无穷且变号，极限不存在？但题目可能默认 $x\to1^-$？按照高等数学常规，通常是求极限若存在必须左右相等，但这里可能题目隐含 $x\to1$ 时通过等价变换得出结果。事实上，令 $t=1-x$，当 $x\to1$ 时 $t\to0$。  
$1-x^2 = 1-(1-t)^2 = 2t - t^2 \sim 2t$。  
$\tan(\frac{\pi}{2}x) = \tan(\frac{\pi}{2}(1-t)) = \tan(\frac{\pi}{2} - \frac{\pi}{2}t) = \cot(\frac{\pi}{2}t) \sim \frac{1}{\frac{\pi}{2}t} = \frac{2}{\pi t}$。  
所以乘积 $\sim (2t) \cdot \frac{2}{\pi t} = \frac{4}{\pi}$。注意 $t\to0$ 可正可负，但 $\cot$ 是奇函数，$\cot(\frac{\pi}{2}t) = \frac{1}{\tan(\frac{\pi}{2}t)}$，若 $t\to0^+$，$\tan>0$，若 $t\to0^-$，$\tan<0$，而 $1-x^2 = 2t-t^2$ 与 $t$ 同号，故乘积恒正，左右极限均为 $4/\pi$，极限存在。

**答案**：$\displaystyle \frac{4}{\pi}$

## 11. 极限 $\displaystyle \lim_{x \to +\infty}\left(\frac{\pi}{2}-\arctan x\right)^{\frac{1}{\ln x}}$

**类型三：求幂指型（**​**$0^0$**​ **）未定式的极限**

**解析**  
当 $x\to+\infty$，$\frac{\pi}{2}-\arctan x \to 0$，指数 $\frac{1}{\ln x}\to 0$，故为 $0^0$ 型。取对数：

$$
\ln L = \lim_{x\to+\infty} \frac{\ln(\frac{\pi}{2}-\arctan x)}{\ln x}.
$$

利用 $\frac{\pi}{2}-\arctan x \sim \frac1x$（因为 $\arctan x = \frac{\pi}{2} - \arctan\frac1x$，所以 $\frac{\pi}{2}-\arctan x = \arctan\frac1x \sim \frac1x$）。于是分子 $\ln(\frac1x) = -\ln x$，所以 $\ln L = \lim \frac{-\ln x}{\ln x} = -1$，故 $L = e^{-1}$。

**答案**：$\displaystyle \frac{1}{e}$

## 12. 极限 $\displaystyle \lim_{x \to \infty}\left(x \arctan \frac{1}{x}\right)^{x^2}$

**类型三：求幂指型（**​**$1^\infty$** **等）未定式的极限**

**解析**  
$x\to\infty$ 时，$\arctan(1/x) \sim 1/x$，故底数 $x \arctan(1/x) \to 1$，指数 $x^2\to\infty$，为 $1^\infty$ 型。令 $t=1/x$，当 $x\to\infty$ 时 $t\to0$。原式变为 $\displaystyle \left(\frac{\arctan t}{t}\right)^{1/t^2}$。取对数：

$$
\ln L = \lim_{t\to0} \frac{ \ln(\frac{\arctan t}{t}) }{t^2}.
$$

展开 $\arctan t = t - \frac{t^3}{3} + \frac{t^5}{5} - \cdots$，所以 $\frac{\arctan t}{t} = 1 - \frac{t^2}{3} + \frac{t^4}{5} - \cdots$。  
$\ln(1+u) = u - \frac{u^2}{2} + \cdots$，其中 $u = -\frac{t^2}{3} + \frac{t^4}{5} + ...$  
所以 $\ln\left(\frac{\arctan t}{t}\right) = \left(-\frac{t^2}{3} + \frac{t^4}{5}\right) - \frac12\left(-\frac{t^2}{3}\right)^2 + o(t^4) = -\frac{t^2}{3} + \frac{t^4}{5} - \frac{t^4}{18} + o(t^4) = -\frac{t^2}{3} + \left(\frac{1}{5}-\frac{1}{18}\right)t^4 = -\frac{t^2}{3} + \frac{13}{90}t^4 + o(t^4)$。  
除以 $t^2$：$\frac{\ln(\cdots)}{t^2} = -\frac13 + \frac{13}{90}t^2 + o(t^2) \to -\frac13$。  
所以 $L = e^{-1/3}$。

**答案**：$\displaystyle e^{-1/3}$

## 13. 极限 $\displaystyle \lim_{x \to +\infty}\left(x+\sqrt{1+x^2}\right)^{\frac{1}{x}}$

**类型三：求幂指型（**​**$\infty^0$**​ **）未定式的极限**

**解析**  
$x\to+\infty$，底数 $x+\sqrt{1+x^2} \sim 2x \to \infty$，指数 $1/x \to 0$，为 $\infty^0$ 型。取对数：

$$
\ln L = \lim_{x\to+\infty} \frac{\ln(x+\sqrt{1+x^2})}{x}.
$$

$\ln(x+\sqrt{1+x^2}) = \ln x + \ln(1+\sqrt{1+1/x^2})$，$x\to\infty$ 时 $\sqrt{1+1/x^2}=1+1/(2x^2)+...$，所以括号内 $\to 2$，故 $\ln(x+\sqrt{1+x^2}) = \ln(2x + o(x)) = \ln x + \ln 2 + o(1)$。因此 $\frac{\ln x + \ln 2}{x} \to 0$，所以 $\ln L = 0$，$L=1$。  
更严谨：直接用等价无穷大：$x+\sqrt{1+x^2} = x + x\sqrt{1+1/x^2} = x(1+1+1/(2x^2)+...) = 2x + o(1)$，对数 $\sim \ln(2x)$，除以 $x$ 趋于 $0$。

**答案**：$1$

## 14. 极限 $\displaystyle \lim_{x \to 0} \frac{\int_{\sin x}^{x} \sqrt{3+t^{2}} dt}{x\left(e^{x^{2}}-1\right)}$

**类型四：求含变限积分的未定式的极限**

**解析**  
分母：$x(e^{x^2}-1) \sim x \cdot x^2 = x^3$。  
分子：变限积分 $\int_{\sin x}^{x} \sqrt{3+t^2} dt$，当 $x\to0$ 时上下限都趋于 $0$，积分上下限差很小。可利用积分中值定理或直接求导（洛必达）。  
使用洛必达：原式为 $\frac00$ 型。分子求导：$\frac{d}{dx}\int_{\sin x}^{x} f(t)dt = f(x) - f(\sin x)\cos x$，其中 $f(t)=\sqrt{3+t^2}$。  
所以导数 $= \sqrt{3+x^2} - \sqrt{3+\sin^2 x}\cdot \cos x$。  
分母求导：$(x(e^{x^2}-1))' = (e^{x^2}-1) + x\cdot 2x e^{x^2} = e^{x^2}-1 + 2x^2 e^{x^2} \sim x^2 + 2x^2 = 3x^2$（因为 $e^{x^2}-1\sim x^2$）。  
现在极限变为 $\lim_{x\to0} \frac{\sqrt{3+x^2} - \sqrt{3+\sin^2 x}\cos x}{3x^2}$。  
在 $x=0$ 处，$\sqrt{3+x^2} \approx \sqrt{3}(1+\frac{x^2}{6})$，$\sin x \sim x$，$\cos x = 1 - \frac{x^2}{2}$。  
则 $\sqrt{3+\sin^2 x}\cos x \approx \sqrt{3}(1+\frac{x^2}{6})(1-\frac{x^2}{2}) \approx \sqrt{3}(1 - \frac{x^2}{3})$。  
分子 $\approx \sqrt{3}(1+\frac{x^2}{6}) - \sqrt{3}(1-\frac{x^2}{3}) = \sqrt{3}(\frac{x^2}{6} + \frac{x^2}{3}) = \sqrt{3} \cdot \frac{x^2}{2}$。  
除以 $3x^2$ 得 $\frac{\sqrt{3}/2}{3} = \frac{\sqrt{3}}{6}$。  
所以极限为 $\frac{\sqrt{3}}{6}$。

**答案**：$\displaystyle \frac{\sqrt{3}}{6}$

## 15. 若极限 $\displaystyle \lim_{x \to 0} \frac{\sin x}{e^{x}-a}(\cos x-b)=5$ ，则 $a=\_\_\_\_ , b=\_\_\_\_$

**类型五：由极限确定未知参数**

**解析**  
当 $x\to0$ 时，分母 $e^x-a \to 1-a$，分子 $\sin x \to 0$，$\cos x - b \to 1-b$。如果极限为有限非零常数 $5$，则分母不能趋于 $0$（否则可能 $\frac00$ 型需要进一步讨论，但如果 $a=1$ 分母趋于 $0$，分子也有 $1-b$ 因子，必须使分子也趋于 $0$ 即 $b=1$，然后再看）。分情况：

- 若 $a\neq1$，则分母趋向 $1-a \neq 0$，分子趋向 $0\cdot(1-b)=0$，整个极限为 $0$，不等于 $5$，不可能。
- 因此必须有 $a=1$，此时分母 $e^x-1 \sim x$，为 $\frac00$ 型。原式变为 $\lim \frac{\sin x}{x} \cdot \frac{x}{e^x-1}(\cos x-b)$。而 $\frac{\sin x}{x}\to1$，$\frac{x}{e^x-1}\to1$，故极限等于 $\lim (\cos x - b) = 1 - b$。令 $1-b = 5$，得 $b = -4$。

所以 $a=1, b=-4$。

**答案**：$a=1,\ b=-4$

## 16. 若极限 $\displaystyle \lim_{x \to \infty}\left(a x^{2}+x^{4}+b x^{5} \int_{0}^{\frac{1}{x}} e^{-t^{2}} d t\right)$ 存在且等于 $A$ ，则 $A$ 的值为\_\_\_\_\_\_.

**类型五：由极限确定未知参数**

**解析**  
先分析积分：当 $x\to\infty$，$1/x \to 0$。$\int_{0}^{1/x} e^{-t^2} dt$。展开 $e^{-t^2}=1 - t^2 + \frac{t^4}{2} - \cdots$，积分得 $\left[ t - \frac{t^3}{3} + \frac{t^5}{10} - \cdots \right]_0^{1/x} = \frac1x - \frac1{3x^3} + \frac1{10x^5} - \cdots$。  
乘以 $b x^5$：$b x^5 \int_0^{1/x} e^{-t^2} dt = b x^5 \left( \frac1x - \frac1{3x^3} + \frac1{10x^5} + O(1/x^7) \right) = b x^4 - \frac{b}{3} x^2 + \frac{b}{10} + o(1)$。  
所以整个表达式为：

$$
a x^2 + x^4 + \left( b x^4 - \frac{b}{3} x^2 + \frac{b}{10} + o(1) \right) = (1+b)x^4 + \left(a - \frac{b}{3}\right)x^2 + \frac{b}{10} + o(1).
$$

要使极限 $(x\to\infty)$ 存在且等于常数 $A$，必须 $x^4$ 和 $x^2$ 的系数为零：  
$1+b=0 \Rightarrow b=-1$；  
$a - \frac{b}{3}=0 \Rightarrow a = \frac{-1}{3}$。  
此时表达式剩下常数项 $\frac{b}{10} = -\frac1{10}$，所以 $A = -\frac1{10}$。

**答案**：$\displaystyle A = -\frac{1}{10}$

## 17. 设 $\displaystyle \lim_{x \to 0} \frac{\sin 2 x+x f(x)}{x^{3}}=1$, 则 $\displaystyle \lim_{x \to 0} \frac{2 \cos x+f(x)}{x^{2}}=$

**类型六：已知一极限计算另一极限**

**解析**  
已知 $\frac{\sin 2x + x f(x)}{x^3} \to 1$。$\sin 2x = 2x - \frac{(2x)^3}{6} + o(x^3) = 2x - \frac{4}{3}x^3 + o(x^3)$。所以分子 $= 2x - \frac{4}{3}x^3 + x f(x) + o(x^3)$。要使其除以 $x^3$ 极限为 $1$，必须消掉 $2x$ 项，故 $f(x) = -2 + \text{高次项}$。设 $f(x) = -2 + a x^2 + o(x^2)$（因为分母是 $x^3$，$x f(x)$ 项提供 $x^3$ 需要 $f(x)$ 的二次项）。则 $x f(x) = -2x + a x^3 + o(x^3)$。分子 $= (2x - 2x) + (-\frac{4}{3} + a)x^3 + o(x^3)$。极限为 $1$，所以 $a - \frac{4}{3} = 1 \Rightarrow a = \frac{7}{3}$。于是 $f(x) = -2 + \frac{7}{3}x^2 + o(x^2)$。

现求 $\lim_{x\to0} \frac{2\cos x + f(x)}{x^2}$。$\cos x = 1 - \frac{x^2}{2} + o(x^2)$，所以 $2\cos x = 2 - x^2 + o(x^2)$。  
分子 $= (2 - x^2) + (-2 + \frac{7}{3}x^2) + o(x^2) = \left( -1 + \frac{7}{3} \right)x^2 + o(x^2) = \frac{4}{3}x^2 + o(x^2)$。  
除以 $x^2$ 得 $\frac{4}{3}$。

**答案**：$\displaystyle \frac{4}{3}$

## 18. 已知 $\displaystyle \lim_{x \to 0} \frac{2-2 \cos x+x f(x)}{x^{3}}=0$, 则 $\displaystyle \lim_{x \to 0} \frac{x+f(\sin x)}{x^{2}}=$

**类型六：已知一极限计算另一极限**

**解析**  
$2-2\cos x = 2(1-\cos x) = x^2 - \frac{x^4}{12} + \cdots$，所以在 $x^3$ 分母下，$2-2\cos x$ 贡献的是 $x^2$ 量级，需要 $x f(x)$ 消掉 $x^2$ 项。设 $f(x) = b x + c x^2 + d x^3 + \cdots$，则 $x f(x) = b x^2 + c x^3 + \cdots$。  
分子：$(x^2 + o(x^3)) + (b x^2 + c x^3 + o(x^3)) = (1+b)x^2 + c x^3 + o(x^3)$。因为极限为 $0$（除以 $x^3$ 后极限为 $0$），所以 $1+b=0$ （否则 $x^2$ 项除以 $x^3$ 会无穷大）$\Rightarrow b=-1$。此时分子剩下 $c x^3 + o(x^3)$，除以 $x^3$ 极限为 $c$，因此 $c=0$。所以我们有 $f(x) = -x + o(x^2)$ （更高阶不影响）。

计算 $\lim_{x\to0} \frac{x + f(\sin x)}{x^2}$。$\sin x = x - \frac{x^3}{6} + \cdots$。$f(\sin x) = -\sin x + o(\sin x)^2$？但需要 $f$ 在 $0$ 附近的展开。由 $f(x) = -x + o(x)$ 知 $f(\sin x) = -\sin x + o(\sin x) = - (x - \frac{x^3}{6}) + o(x)??$ 这里因 $o(\sin x)$ 相对于 $x$ 是高阶，但未知是不是二阶。实际上，我们已知 $f(x) = -x + o(x)$，这提供的精度不够求 $\frac{x+f(\sin x)}{x^2}$，需要知道 $f(x)$ 的二阶展开。重新审视条件：极限为 $0$ 且分子为 $x^3$ 时，我们只需要消去 $x^2$ 项，并不能确定 $f(x)$ 的二次项系数。但我们可以用极限表示式反推。由 $\frac{2-2\cos x + x f(x)}{x^3} = 0$，可得 $x f(x) = -2+2\cos x + o(x^3) = -x^2 + \frac{x^4}{12} + o(x^3)$。所以 $f(x) = -x + \frac{x^3}{12} + o(x^3)$。因此 $f(x)$ 的展开为 $f(x) = -x + 0\cdot x^2 + \frac{x^3}{12} + o(x^3)$。则 $f(\sin x) = -\sin x + \frac{\sin^3 x}{12} + o(x^3) = -(x - \frac{x^3}{6}) + \frac{x^3}{12} + o(x^3) = -x + \frac{x^3}{6} + \frac{x^3}{12} + o(x^3) = -x + \frac{x^3}{4} + o(x^3)$。  
于是 $x + f(\sin x) = x - x + \frac{x^3}{4} + o(x^3) = \frac{x^3}{4} + o(x^3)$。  
除以 $x^2$ 得 $\frac{x}{4} + o(x) \to 0$。极限为 $0$。

**答案**：$0$

## 19. 设 $f(x)$ 一阶可导且 $f'(0) \neq 0, f(0)=1$, 则 $\displaystyle \lim_{x \to 0} \frac{f\left(e^{2 x}-1\right)-1}{\ln f(x)}=$

**类型七：利用导数计算极限**

**解析**  
利用导数定义。当 $x\to0$，$e^{2x}-1 \sim 2x \to 0$，所以 $f(e^{2x}-1) - 1 = f(0+ (e^{2x}-1)) - f(0)$。  
分子 $\sim f'(0) (e^{2x}-1) \sim f'(0) \cdot 2x$。  
同理分母：$\ln f(x) = \ln(1 + (f(x)-1)) \sim f(x)-1$（因为 $f(x)\to1$）。而 $f(x)-1 \sim f'(0)x$。所以分母 $\sim f'(0)x$。  
因此原式 $\sim \frac{2 f'(0) x}{f'(0) x} = 2$。

**答案**：$2$

## 20. 设 $f(x)$ 二阶导连续, 且 $\displaystyle \lim_{x \to 0} \frac{1-f(x)}{x^{2}}=\frac{1}{2}$, 则 $\displaystyle \lim_{x \to+\infty}\left[f\left(\frac{1}{\sqrt{x}}\right)\right]^{x}=$

**类型七：利用导数计算极限**（或也可归为类型三幂指型，但条件给出导数信息，本质用泰勒展开）

**解析**  
由条件 $\frac{1-f(x)}{x^2} \to \frac12$ 得 $f(0)=1$, $f'(0)=0$, $f''(0) = -1$。泰勒展开：$f(x) = 1 - \frac{1}{2}x^2 + o(x^2)$。  
待求极限：令 $t = 1/\sqrt{x}$，则 $x = 1/t^2$，$x\to+\infty \Rightarrow t\to0^+$。  
原式 $= \lim_{t\to0^+} [f(t)]^{1/t^2}$。取对数：$\lim_{t\to0} \frac{\ln f(t)}{t^2}$。  
$f(t) = 1 - \frac12 t^2 + o(t^2)$，故 $\ln f(t) = -\frac12 t^2 + o(t^2)$。除以 $t^2$ 得 $-\frac12$。  
所以原极限 $= e^{-1/2} = \frac{1}{\sqrt{e}}$。

**答案**：$\displaystyle \frac{1}{\sqrt{e}}$
