# 知识提要 常用工具

知识提要

积分学是微积分的重要部分，占据了整个高数数学的半壁江山，而一元函数积分学是积分学的基础.咱们后面的二重积分、三重积分、曲线曲面积分的概念、性质与计算其实与一元积分部分非常相似，同时其计算往往最终要转化为定积分后再计算！

一元函数积分学包括不定积分、变限积分、定积分以及反常积分这四个部分，无论是相关的定义、性质或是计算，都是考研数学的考察重点，一定要学透了！

接下去，我们会带大家针对积分的计算进行训练，解决大家在积分计算这块的痛点！

常用工具

（一）基本积分表

$$
\int \frac{1}{x} dx = \ln|x| + C
$$

$$
\int 0 dx = C
$$

$$
\int a^x dx = \frac{1}{\ln a} a^x + C
$$

$$
\int \sin x dx = -\cos x + C
$$

$$
\int \sec^2 x dx = \tan x + C
$$

$$
\int \tan x \cdot \sec x dx = \sec x + C
$$

$$
\int \sec x dx = \ln|\sec x + \tan x| + C
$$

$$
\int x^n dx = \frac{1}{n+1} x^{n+1} (n \neq -1)
$$

$$
\int e^x dx = e^x + C
$$

$$
\int \cos x dx = \sin x + C
$$

$$
\int \csc^2 x dx = -\cot x + C
$$

$$
\int \csc x \cdot \cot x dx = -\csc x + C
$$

$$
\int \csc x dx = \ln|\csc x - \cot x| + C
$$

$$
\int \tan x dx = -\ln|\cos x| + C
$$

$$
\int \cot x dx = \ln|\sin x| + C
$$

$$
\int \frac{1}{\sqrt{1-x^2}} dx = \arcsin x + C
$$

$$
\int \frac{1}{\sqrt{a^2-x^2}} dx = \arcsin \frac{x}{a} + C
$$

$$
\int \frac{1}{1+x^2} dx = \arctan x + C
$$

$$
\int \frac{1}{a^2+x^2} dx = \frac{1}{a} \arctan \frac{x}{a} + C
$$

$$
\int \frac{1}{x^2-a^2} dx = \frac{1}{2a} \ln \left| \frac{x-a}{x+a} \right| + C
$$

$$
\int \frac{1}{\sqrt{x^2 \pm a^2}} dx = \ln \left| \sqrt{x^2 \pm a^2} + x \right| + C
$$

(二) 积分法

不定积分法

- 第一换元法(凑微分)

  - ① 适用类型：相对简单的类型，被积函数能扔到微元
  - ② 注意添和拆的技巧，多
  - ③ 小技巧：如e^x分式，
- 第二换元法

  - 三角换元

    - ① 适用类型：解
    - ② 注意三角换元
  - 倒代换 —— 适用类型：分式分母
  - 整体换元 —— 整体代换的部分，
- 分部积分

  - ① 适用类型：两类函数相乘/相除形式
  - ② 基本原则：反对幂三指
  - ③ 能凑先凑，往往需要多次分部积分计
