# 类型一：利用积分法的原函数计算

咱们计算不定积分的“武器库”就那几种，但具体某道题采用哪种方法以及你能不能把它算对，只有一种办法，那就是亲手算，多练习，计算能力的提升是没有捷径的！
同时，我们在计算不定积分时，往往会在一道题里用到1~3种不同的积分法，并且很难一下子看出来合适的积分方法. 不要慌，多尝试，方法就这么些！
