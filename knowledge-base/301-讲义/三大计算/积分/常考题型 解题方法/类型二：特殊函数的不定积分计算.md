## (1) 有理分式的不定积分计算

【解题方法】
由于有理分式可以拆分为多项式与真分式之和，而真分式可分解为多个简单分式之和，因此有理分式的不定积分可通过上述拆分方式转化为多个简单分式后再进行积分。
对于有理分式的不定积分，关键的两步，一是拆分规则，二是待定系数法计算系数。
有理分式拆分为多项式与真分式的过程，可利用长除法直接计算，这里不多做赘述。
真分式分解为多个简单分式的规则如下：
① $\frac{1}{(x+a)(x+b)} = \frac{A}{x+a} + \frac{B}{x+b}$
② </span>$\frac{1}{(x+a)(x+b)^2} = \frac{A}{x+a} + \frac{B}{x+b} + \frac{C}{(x+b)^2}$
③ </span>$\frac{1}{(x+a)(x^2+bx+c)} = \frac{A}{x+a} + \frac{Bx+C}{x^2+bx+c}$​ ($x^2+bx+c$<span data-type="text" style="color: var(--b3-font-color11);"> 已无法继续因式分解)
④ </span>$\frac{1}{(x+a)(x^2+bx+c)^2} = \frac{A}{x+a} + \frac{Bx+C}{x^2+bx+c} + \frac{Dx+E}{(x^2+bx+c)^2}$
对于分解式中的待定系数 $A, B, C, D$ 等等，需要利用待定系数法进行计算。
此类问题的计算量比较大，务必要小心再小心！

## (2) 三角函数对应的有理式的不定积分计算

【解题方法】
对三角函数的分式进行积分，两种处理方式：分子-&gt; 分母 + 分母导数
① 分式的分子、分母均为关于 $\sin x, \cos x$​ 的线性组合，通过拆分的方式求原函数
例如，计算 $\int \frac{a_1 \sin x + b_1 \cos x}{a_2 \sin x + b_2 \cos x} dx$，可将分子拆分成待定系数的形式，
$a_1 \sin x + b_1 \cos x = A(a_2 \sin x + b_2 \cos x) + B(a_2 \sin x + b_2 \cos x)'$，根据 $\begin{cases} A \cdot a_2 - B \cdot b_2 = a_1 \\ A \cdot b_2 + B \cdot a_2 = b_1 \end{cases}$，可解出 $A, B$ 的值。
最后，可得 $\int \frac{a_1 \sin x + b_1 \cos x}{a_2 \sin x + b_2 \cos x} dx = A \int dx + B \int \frac{(a_2 \sin x + b_2 \cos x)'}{a_2 \sin x + b_2 \cos x} dx = Ax + B \ln |a_2 \sin x + b_2 \cos x| + C$。
② 分式的分母为关于 $\sin x, \cos x$​ 的线性组合但分子不是，通过万能代换的方式求原函数
万能代换法在考研中几乎很少用到，但建议大家还是了解下。万能代换法适用于三角函数的有理分式，能够通过换元的方式将其化为有理分式的积分。
万能代换的换元方式固定，令 $t = \tan \frac{x}{2}$，则 $x = 2 \arctan t$，$dx = \frac{2}{1+t^2} dt$，
由于 $\tan x = \frac{2 \tan \frac{x}{2}}{1 - \left( \tan \frac{x}{2} \right)^2}$，故 $\tan x = \frac{2t}{1-t^2}$，根据辅助三角形可知，$\sin x = \frac{2t}{1+t^2}, \cos x = \frac{1-t^2}{1+t^2}$，
将原三角函数有理分式中三角函数与微元分别代换即可！

(3) 分段函数的不定积分计算
【解题方法】
由于分段函数在不同区间内有不同的函数表达式，因此分段函数的不定积分计算需采用分段积分法.
注意点：由于题目给的 $f(x)$ 连续，因此要保证求出来的原函数也要连续.
计算方法：题目要求的是不定积分 $\int f(x)dx$ ， $f(x)$ 为分段函数，
第一步，在不同区间上分别积分计算出相应的原函数 $F(x)$ （此时不写常数 $C$）；
第二步，计算原函数在分段点处左、右极限，给原函数分段点左侧或右侧补上差值，保证 $F(x)$ 连续；
第三步，此时 $F(x)+C$ 即为分段函数的原函数了！
当被积函数包含绝对值、最值时，往往也需要用上述的分段积分法进行计算.
