# 解析

## 一、填空题

### (1)

求极限 $\displaystyle\lim_{x \to 0} \frac{\tan x - \sin x}{\bigl(1 - \sqrt[3]{1 + x^2}\bigr)\bigl(\sqrt{1 + \sin x} - 1\bigr)}$。

**解：**  当 $x \to 0$ 时，利用等价无穷小与泰勒展开。

$$
\tan x = x + \frac{x^3}{3} + o(x^3),\quad \sin x = x - \frac{x^3}{6} + o(x^3)
$$

$$
\tan x - \sin x = \frac{x^3}{2} + o(x^3) \sim \frac{1}{2}x^3.
$$

分母第一部分：

$$
1 - \sqrt[3]{1 + x^2} = -\bigl((1+x^2)^{1/3} - 1\bigr) \sim -\frac{1}{3}x^2.
$$

分母第二部分：

$$
\sqrt{1 + \sin x} - 1 \sim \frac{1}{2}\sin x \sim \frac{1}{2}x.
$$

相乘得：

$$
\Bigl(1 - \sqrt[3]{1 + x^2}\Bigr)\Bigl(\sqrt{1 + \sin x} - 1\Bigr) \sim \left(-\frac{1}{3}x^2\right)\!\left(\frac{1}{2}x\right) = -\frac{1}{6}x^3.
$$

因此

$$
\lim_{x \to 0} \frac{\frac{1}{2}x^3}{-\frac{1}{6}x^3} = -3.
$$

**答案：**  $-3$

### (2)

求极限 $\displaystyle\lim_{x \to +\infty} x^2 \ln \frac{\arctan(x+1)}{\arctan x}$.

**解：**  设 $g(x) = \ln\arctan x$，则

$$
x^2 \ln\frac{\arctan(x+1)}{\arctan x} = x^2\bigl(g(x+1)-g(x)\bigr).
$$

由拉格朗日中值定理，存在 $\xi\in (x,x+1)$ 使得

$$
g(x+1)-g(x) = g'(\xi) = \frac{1}{\arctan\xi}\cdot\frac{1}{1+\xi^2}.
$$

当 $x\to+\infty$ 时 $\xi\to+\infty$，$\arctan\xi\to\frac{\pi}{2}$，$\frac{1}{1+\xi^2}\sim\frac{1}{\xi^2}\sim\frac{1}{x^2}$，故

$$
g'(\xi) \sim \frac{2}{\pi}\cdot\frac{1}{x^2}.
$$

于是

$$
x^2\bigl(g(x+1)-g(x)\bigr) \to \frac{2}{\pi}.
$$

**答案：**  $\dfrac{2}{\pi}$

### (3)

设 $\displaystyle\lim_{x \to 0} \left[\cos x + \frac{f(x)}{x}\right]^{\frac{1}{x^2}} = \frac{1}{e}$，求 $\displaystyle\lim_{x \to 0} \frac{f(x)}{x^3}$.

**解：**  这是 $1^\infty$ 型极限，利用公式 $\lim u^v = \exp\bigl(\lim v(u-1)\bigr)$（当 $u\to1$）。  
原极限 $= e^{-1}$，故

$$
\lim_{x\to0} \frac{\cos x + \frac{f(x)}{x} - 1}{x^2} = -1.
$$

而 $\cos x - 1 \sim -\dfrac{x^2}{2}$，所以

$$
\lim_{x\to0} \frac{-\frac{x^2}{2} + \frac{f(x)}{x} + o(x^2)}{x^2} = -1
\;\Longrightarrow\; -\frac{1}{2} + \lim_{x\to0}\frac{f(x)}{x^3} = -1.
$$

因此

$$
\lim_{x\to0}\frac{f(x)}{x^3} = -\frac{1}{2}.
$$

**答案：**  $-\dfrac{1}{2}$

### (4)

设曲线 $y=f(x)$ 在原点与曲线 $y=\sin x$ 相切，求 $\displaystyle\lim _{n \rightarrow \infty} \sqrt{n} \cdot \sqrt{f\!\left(\frac{2}{n}\right)}$.

**解：**  相切即 $f(0)=\sin0=0$，$f'(0)=\cos0=1$。当 $n\to\infty$ 时 $\frac{2}{n}\to0$，由局部线性化：

$$
f\!\left(\frac{2}{n}\right) = f(0) + f'(0)\frac{2}{n} + o\!\left(\frac{1}{n}\right) = \frac{2}{n} + o\!\left(\frac{1}{n}\right).
$$

因此

$$
\sqrt{n}\cdot\sqrt{f\!\left(\frac{2}{n}\right)} \sim \sqrt{n}\cdot\sqrt{\frac{2}{n}} = \sqrt{2}.
$$

故极限为 $\sqrt{2}$.

**答案：**  $\sqrt{2}$

### (5)

设 $y = y(x)$ 由方程 $y - x e^{y} = 1 - ex$ 确定，求 $\displaystyle\lim_{x \to 0} \frac{3y - 2 - \cos x}{\ln(1 + x^{2})}$.

**解：**  先求 $y(0), y'(0), y''(0)$。  
代入 $x=0$：$y(0) - 0 = 1 \Rightarrow y(0)=1$。  
方程两边对 $x$ 求导：

$$
y' - e^{y} - x e^{y} y' = -e.
$$

代入 $x=0, y=1$：$y'(0) - e = -e \Rightarrow y'(0)=0$。  
再求导：

$$
y'' - e^{y}y' - \bigl(e^{y}y' + x e^{y}(y')^2 + x e^{y}y''\bigr) = 0,
$$

即 $y'' - 2e^{y}y' - x e^{y}(y')^2 - x e^{y}y'' = 0$。  
代入 $x=0, y=1, y'=0$ 得 $y''(0)=0$。

当 $x\to0$ 时，分母 $\ln(1+x^2)\sim x^2$。  
分子：$3y-2-\cos x = 3(y-1) + (1-\cos x)$。  
$y-1 = y'(0)x + \frac{y''(0)}{2}x^2 + o(x^2) = o(x^2)$，  
$1-\cos x = \frac{x^2}{2} + o(x^2)$。  
分子 $= \frac{x^2}{2} + o(x^2)$，分母 $\sim x^2$，故极限为 $\frac{1}{2}$.

**答案：**  $\dfrac{1}{2}$

### (6)

设 $y = x^{x}\;(x>1)$ 的反函数为 $x = f(y)$，求 $f''(4)$.

**解：**  $y = x^x$ 当 $x=2$ 时 $y=4$，故 $f(4)=2$。  
先求 $f'(y) = \frac{dx}{dy} = 1\Big/ \frac{dy}{dx}$。  
$\ln y = x\ln x\Rightarrow \frac{y'}{y} = \ln x + 1 \Rightarrow y' = x^x(\ln x + 1)$。  
在 $x=2$ 处 $y' = 4(\ln2 + 1)$。  
故 $f'(4) = \dfrac{1}{4(\ln2+1)}$。

又 $f''(y) = \frac{d}{dy}f'(y) = \frac{d}{dx}\!\bigl[\frac{1}{x^x(\ln x+1)}\bigr] \Big/ \frac{dy}{dx}$。  
设 $g(x) = \dfrac{1}{x^x(\ln x+1)}$，则  
$\ln g = -x\ln x - \ln(\ln x+1)$，  
$\frac{g'}{g} = -(\ln x+1) - \frac{1}{x(\ln x+1)}$。  
在 $x=2$ 处：  
$g(2) = \dfrac{1}{4(\ln2+1)}$，  
$g'(2) = g(2)\!\left[-(\ln2+1) - \frac{1}{2(\ln2+1)}\right]$。  
于是

$$
f''(4) = \frac{g'(2)}{y'(2)}
= \frac{g(2)}{4(\ln2+1)}\!\left[-(\ln2+1) - \frac{1}{2(\ln2+1)}\right]
= \frac{1}{16(\ln2+1)^2}\cdot\frac{-2(\ln2+1)^2 - 1}{2(\ln2+1)}
= -\frac{2(\ln2+1)^2 + 1}{32(\ln2+1)^3}.
$$

**答案：**  $-\dfrac{2(\ln2+1)^2 + 1}{32(\ln2+1)^3}$

### (7)

已知 $\begin{cases} x = \sqrt{t^{2}+1}, \\ y = \ln\bigl(t + \sqrt{t^{2}+1}\bigr), \end{cases}$ 求 $\displaystyle\left.\frac{d^{2}y}{dx^{2}}\right|_{t=1}$.

**解：**  求导：  
\(\frac{dx}{dt} = \frac{t}{\sqrt{t<sup>{2}+1}},\quad\frac{dy}{dt} = \frac{1}{t+\sqrt{t</sup>{2}+1}}\!\left(1+\frac{t}{\sqrt{t<sup>{2}+1}}\right) = \frac{1}{\sqrt{t</sup>{2}+1}}.\)  
于是

$$
\frac{dy}{dx} = \frac{dy/dt}{dx/dt} = \frac{1}{\sqrt{t^{2}+1}} \Big/ \frac{t}{\sqrt{t^{2}+1}} = \frac{1}{t}.
$$

$$
\frac{d^{2}y}{dx^{2}} = \frac{d}{dt}\!\left(\frac{1}{t}\right) \Big/ \frac{dx}{dt}
= -\frac{1}{t^{2}} \Big/ \frac{t}{\sqrt{t^{2}+1}} = -\frac{\sqrt{t^{2}+1}}{t^{3}}.
$$

代入 $t=1$ 得 $\left.\frac{d^{2}y}{dx^{2}}\right|_{t=1} = -\sqrt{2}.$

**答案：**  $-\sqrt{2}$

### (8)

设 $f(x) = \arctan x - \dfrac{x}{1 + a x^{2}}$，且 $f'''(0) = 1$，求 $a$.

**解：**  用麦克劳林展开。

$$
\arctan x = x - \frac{x^{3}}{3} + \frac{x^{5}}{5} - \cdots
$$

$$
\frac{x}{1 + a x^{2}} = x\bigl(1 - a x^{2} + a^{2} x^{4} - \cdots\bigr) = x - a x^{3} + a^{2} x^{5} - \cdots
$$

则

$$
f(x) = \left(x - \frac{x^{3}}{3}\right) - \bigl(x - a x^{3}\bigr) + O(x^{5}) = \Bigl(a - \frac{1}{3}\Bigr)x^{3} + O(x^{5}).
$$

泰勒展开中 $x^{3}$ 的系数为 $\frac{f'''(0)}{3!} = a - \frac{1}{3}$。  
已知 $f'''(0) = 1$，故

$$
\frac{1}{6} = a - \frac{1}{3} \;\Longrightarrow\; a = \frac{1}{6} + \frac{1}{3} = \frac{1}{2}.
$$

**答案：**  $\dfrac{1}{2}$

### (9)

设 $f(x) = \begin{cases} x e^{-x}, & -1 \le x < 0, \\ x \sin x, & 0 \le x \le 1. \end{cases}$ 记 $F(x) = \int f(x)\,dx$，求 $F(x)$.

**解：**  分段求不定积分。  
当 $-1 \le x < 0$ 时：

$$
\int x e^{-x}\,dx = -\int x\,d(e^{-x}) = -x e^{-x} + \int e^{-x}\,dx = -e^{-x}(x+1) + C_{1}.
$$

当 $0 \le x \le 1$ 时：

$$
\int x \sin x\,dx = -\int x\,d(\cos x) = -x\cos x + \int \cos x\,dx = -x\cos x + \sin x + C_{2}.
$$

由于原函数必须连续，考虑 $x=0$ 处的左、右极限：  
左极限 $F(0-) = -1 + C_{1}$，右极限 $F(0+) = C_{2}$。令其相等得 $C_{2} = C_{1} - 1$。  
令 $C = C_{1}$，则

$$
F(x) = \begin{cases}
-e^{-x}(x+1) + C, & -1 \le x < 0, \\
-x\cos x + \sin x + C - 1, & 0 \le x \le 1.
\end{cases}
$$

（注：若不加任意常数，可取一组特定的原函数，例如令 $C=1$ 使表达式更简洁，但保留常数是通解。）

**答案：**  $F(x) = \begin{cases} -e^{-x}(x+1) + C_{1}, & -1 \le x < 0, \\ -x\cos x + \sin x + C_{2}, & 0 \le x \le 1 \end{cases}$ 且满足 $C_{2}=C_{1}-1$（或等价形式）

### (10)

求极限 $\displaystyle\lim_{n \to \infty} \sum_{k=1}^{n} \frac{k}{n^{2}} \ln\!\left(1 + \frac{2k}{n}\right)$.

**解：**  写成积分和式：

$$
\frac{1}{n} \sum_{k=1}^{n} \frac{k}{n} \ln\!\left(1 + 2\cdot\frac{k}{n}\right).
$$

这是函数 $x\ln(1+2x)$ 在 $[0,1]$ 上的定积分：

$$
I = \int_{0}^{1} x \ln(1+2x)\,dx.
$$

令 $t = 1+2x$，则 $x = \frac{t-1}{2},\; dx = \frac{dt}{2}$，积分限 $x=0\to1$ 对应 $t=1\to3$.

$$
I = \int_{1}^{3} \frac{t-1}{2}\,\ln t \cdot \frac{dt}{2}
= \frac{1}{4} \int_{1}^{3} (t-1)\ln t\,dt.
$$

计算：

$$
\int t\ln t\,dt = \frac{t^{2}}{2}\ln t - \frac{t^{2}}{4},\qquad
\int \ln t\,dt = t\ln t - t.
$$

$$
\int_{1}^{3}(t-1)\ln t\,dt = \left[\frac{t^{2}}{2}\ln t - \frac{t^{2}}{4} - t\ln t + t\right]_{1}^{3}
= \left(\frac{9}{2}\ln3 - 2 - 3\ln3 + 3\right) - \left(0 - \frac{1}{4} - 0 + 1\right)
= \frac{3}{2}\ln3.
$$

故 $I = \frac{1}{4}\cdot\frac{3}{2}\ln3 = \frac{3}{8}\ln3$.

**答案：**  $\dfrac{3}{8}\ln 3$

### (11)

设 $f(x) = \displaystyle\int_{1}^{x} \frac{1}{\sqrt{1 + t^{4}}}\,dt$，求 $\displaystyle\int_{0}^{1} x^{2} f(x)\,dx$.

**解：**  用分部积分法。  
令 $u = f(x),\; dv = x^{2}dx$，则  
$du = f'(x)dx = \frac{1}{\sqrt{1+x^{4}}}dx,\; v = \frac{x^{3}}{3}$.

$$
\int_{0}^{1} x^{2}f(x)\,dx = \left.\frac{x^{3}}{3}f(x)\right|_{0}^{1} - \int_{0}^{1} \frac{x^{3}}{3}\cdot\frac{1}{\sqrt{1+x^{4}}}\,dx.
$$

$f(1) = \int_{1}^{1}\cdots = 0$，$0^{3}\cdot f(0)=0$，边界项为零。  
因此

$$
\text{原式} = -\frac{1}{3}\int_{0}^{1} \frac{x^{3}}{\sqrt{1+x^{4}}}\,dx.
$$

令 $u = 1+x^{4}$，则 $du = 4x^{3}dx$，当 $x=0$ 时 $u=1$，$x=1$ 时 $u=2$：

$$
-\frac{1}{3}\cdot\frac{1}{4}\int_{1}^{2} u^{-1/2}\,du = -\frac{1}{12}\Bigl[2u^{1/2}\Bigr]_{1}^{2} = -\frac{1}{6}(\sqrt{2} - 1) = \frac{1 - \sqrt{2}}{6}.
$$

**答案：**  $\dfrac{1 - \sqrt{2}}{6}$

### (12)

反常积分 $\displaystyle\int_{0}^{+\infty} \frac{x}{(1 + x)^{3}}\,dx$.

**解：**  直接求原函数。令 $t = 1+x$，$x = t-1$，$dx = dt$，积分限 $0 \to +\infty$ 变为 $1 \to +\infty$：

$$
\int_{1}^{+\infty} \frac{t-1}{t^{3}}\,dt = \int_{1}^{+\infty} \bigl(t^{-2} - t^{-3}\bigr)dt
= \left[-t^{-1} + \frac{1}{2}t^{-2}\right]_{1}^{+\infty}
= 0 - (-1 + \frac{1}{2}) = \frac{1}{2}.
$$

**答案：**  $\dfrac{1}{2}$

## 二、解答题

### (13)（10分）计算不定积分 $\displaystyle\int \frac{\sin x}{\sin ^{3} x+\cos ^{3} x}\,dx$.

**解：**  分子分母同除以 $\cos^{3}x$：

$$
\int \frac{\sin x}{\sin^{3}x+\cos^{3}x}\,dx
= \int \frac{\tan x \cdot \sec^{2}x}{\tan^{3}x + 1}\,dx.
$$

令 $u = \tan x$，则 $du = \sec^{2}x\,dx$，积分化为

$$
\int \frac{u}{u^{3}+1}\,du.
$$

因式分解：$u^{3}+1 = (u+1)(u^{2}-u+1)$。部分分式：

$$
\frac{u}{u^{3}+1} = \frac{A}{u+1} + \frac{Bu+C}{u^{2}-u+1}.
$$

通分比较分子：

$$
u = A(u^{2}-u+1) + (Bu+C)(u+1) = (A+B)u^{2} + (-A+B+C)u + (A+C).
$$

得方程组：

$$
A+B=0,\quad -A+B+C=1,\quad A+C=0.
$$

解得 $A = -\frac{1}{3}$，$B = \frac{1}{3}$，$C = \frac{1}{3}$。  
于是

$$
\int \frac{u}{u^{3}+1}\,du = \frac{1}{3}\left( -\int\frac{du}{u+1} + \int\frac{u+1}{u^{2}-u+1}\,du \right).
$$

处理第二个积分：配平方 $u^{2}-u+1 = \left(u-\frac{1}{2}\right)^{2} + \frac{3}{4}$，且 $u+1 = \left(u-\frac{1}{2}\right) + \frac{3}{2}$。

$$
\int\frac{u+1}{u^{2}-u+1}\,du
= \int\frac{u-\frac{1}{2}}{u^{2}-u+1}\,du + \frac{3}{2}\int\frac{du}{\left(u-\frac{1}{2}\right)^{2} + \frac{3}{4}}
$$

$$
= \frac{1}{2}\ln(u^{2}-u+1) + \frac{3}{2}\cdot\frac{1}{\sqrt{3}/2}\arctan\frac{u-\frac{1}{2}}{\sqrt{3}/2}
= \frac{1}{2}\ln(u^{2}-u+1) + \sqrt{3}\arctan\frac{2u-1}{\sqrt{3}}.
$$

综上，

$$
\int \frac{u}{u^{3}+1}\,du = \frac{1}{3}\left[ -\ln|u+1| + \frac{1}{2}\ln(u^{2}-u+1) + \sqrt{3}\arctan\frac{2u-1}{\sqrt{3}} \right] + C.
$$

化简对数部分：

$$
-\ln|u+1| + \frac{1}{2}\ln(u^{2}-u+1) = \frac{1}{2}\ln\frac{u^{2}-u+1}{(u+1)^{2}}.
$$

故原积分

$$
= \frac{1}{6}\ln\frac{\tan^{2}x - \tan x + 1}{(\tan x+1)^{2}} + \frac{\sqrt{3}}{3}\arctan\frac{2\tan x-1}{\sqrt{3}} + C.
$$

### (14)（10分）已知 $f(x)=\begin{cases} a x^{2}+b x+c, & x<0, \\ \ln (x+1), & x \ge 0 \end{cases}$ 在 $x=0$ 处二阶可导，求参数 $a, b, c$.

**解：**  二阶可导蕴含 $f(x)$ 在 $x=0$ 连续、一阶可导且一阶导数连续、二阶导数存在。  
连续性：

$$
f(0) = \ln 1 = 0,\quad \lim_{x\to0^{-}}f(x) = c \;\Longrightarrow\; c = 0.
$$

一阶可导：

$$
f'(0^{+}) = \left.\frac{1}{x+1}\right|_{x=0} = 1,\quad f'(0^{-}) = 2a\cdot0 + b = b \;\Longrightarrow\; b = 1.
$$

二阶可导：

$$
f''(0^{+}) = -\frac{1}{(x+1)^{2}}\Big|_{x=0} = -1,\quad f''(0^{-}) = 2a \;\Longrightarrow\; 2a = -1 \;\Longrightarrow\; a = -\frac{1}{2}.
$$

**答案：**  $a = -\dfrac{1}{2},\; b = 1,\; c = 0$.

### (15)（10分）

设 $f(x)$ 连续，且 $\displaystyle\lim_{x \to 0} \frac{x f(x) - \ln(1+x)}{x^{2}} = 2$，  
$F(x) = \displaystyle\int_{0}^{x} t f(x-t)\,dt$，当 $x \to 0$ 时 $F(x) - \frac{1}{2}x^{2} \sim b x^{k}$ ($b\neq 0, k\in\mathbb{N}^{*}$)。

#### (1) 求 $k, b$ 及 $f(0)$；

#### (2) 证明 $f(x)$ 在 $x=0$ 处可导并求 $f'(0)$.

**解：**

 **(1)**  由已知极限：

$$
\frac{x f(x) - \ln(1+x)}{x^{2}}
= \frac{x(f(x)-1) + x - \ln(1+x)}{x^{2}} = \frac{f(x)-1}{x} + \frac{x - \ln(1+x)}{x^{2}}.
$$

利用 $\displaystyle\lim_{x\to0}\frac{x - \ln(1+x)}{x^{2}} = \frac{1}{2}$（泰勒或洛必达），得

$$
\lim_{x\to0}\frac{f(x)-1}{x} + \frac{1}{2} = 2
\;\Longrightarrow\; \lim_{x\to0}\frac{f(x)-1}{x} = \frac{3}{2}.
$$

因此 $f(x)-1 \sim \frac{3}{2}x$，推出 $f(0)=1$ 且后续将知 $f'(0)=\frac{3}{2}$。

现求 $F(x)$。作代换 $u = x-t$，则 $t = x-u$，$dt = -du$，

$$
F(x) = \int_{0}^{x} t f(x-t)\,dt = \int_{x}^{0} (x-u)f(u)(-du) = \int_{0}^{x} (x-u)f(u)\,du
= x\int_{0}^{x} f(u)\,du - \int_{0}^{x} u f(u)\,du.
$$

由 (1) 知 $f(0)=1$，且由 (2) 将证 $f$ 在 $0$ 处可导且 $f'(0)=\frac{3}{2}$，故有局部展开  
$f(u) = 1 + \frac{3}{2}u + o(u)$。计算积分：

$$
\int_{0}^{x} f(u)\,du = x + \frac{3}{4}x^{2} + o(x^{2}),
$$

$$
\int_{0}^{x} u f(u)\,du = \int_{0}^{x} \bigl(u + \tfrac{3}{2}u^{2} + o(u^{2})\bigr)du = \frac{1}{2}x^{2} + o(x^{2}).
$$

于是

$$
F(x) = x\Bigl(x + \frac{3}{4}x^{2} + o(x^{2})\Bigr) - \Bigl(\frac{1}{2}x^{2} + o(x^{2})\Bigr)
= x^{2} + \frac{3}{4}x^{3} - \frac{1}{2}x^{2} + o(x^{3}) = \frac{1}{2}x^{2} + \frac{3}{4}x^{3} + o(x^{3}).
$$

所以

$$
F(x) - \frac{1}{2}x^{2} = \frac{3}{4}x^{3} + o(x^{3}) \sim \frac{3}{4}x^{3}.
$$

故 $k = 3$，$b = \frac{3}{4}$。  
（也可用泰勒：$F'(x)=\int_{0}^{x}f(u)du$, $F''(x)=f(x)$, $F'''(x)=f'(x)$，在 $0$ 处 $F(0)=F'(0)=0$, $F''(0)=f(0)=1$, $F'''(0)=f'(0)=3/2$，得到相同展开。）

综上，$f(0)=1$，$k=3$，$b=\dfrac{3}{4}$。

 **(2)**  由 (1) 推导中已有

$$
\lim_{x\to0}\frac{x f(x) - \ln(1+x)}{x^{2}} = 2
\;\Longrightarrow\; \lim_{x\to0}\frac{f(x)-1}{x} = \frac{3}{2}.
$$

该极限存在说明 $f(x)$ 在 $x=0$ 处可导，且导数值为 $\frac{3}{2}$。

**答案：**  (1) $f(0)=1,\; k=3,\; b=\frac{3}{4}$; (2) 可导，$f'(0)=\frac{3}{2}$.

### (16)（10分）

设连续函数 $f(x)$ 满足 $\displaystyle f(x) = \frac{x^{2}}{1+x^{2}} + \frac{1}{1+x^{2}}\int_{0}^{1} f(x)\,dx + \int_{-1}^{1} f(x)\,dx$，计算 $\displaystyle\int_{0}^{2} f(x-1)\,dx$.

**解：**  令常数

$$
A = \int_{0}^{1} f(x)\,dx,\qquad B = \int_{-1}^{1} f(x)\,dx,
$$

则方程化为

$$
f(x) = \frac{x^{2}}{1+x^{2}} + \frac{A}{1+x^{2}} + B
= 1 - \frac{1}{1+x^{2}} + \frac{A}{1+x^{2}} + B
= 1 + B + \frac{A-1}{1+x^{2}}.
$$

记 $C = 1+B$，$D = A-1$，则 $f(x) = C + \dfrac{D}{1+x^{2}}$（偶函数）。

由定义 $A = \int_{0}^{1} \bigl(C + \frac{D}{1+x^{2}}\bigr)dx = C + D\arctan x\big|_{0}^{1} = C + \frac{\pi}{4}D$.  
因 $f$ 为偶函数，$B = \int_{-1}^{1} f(x)\,dx = 2\int_{0}^{1} f(x)\,dx = 2A$.

用 $A$ 表示 $C, D$：

$$
C = 1 + B = 1 + 2A,\qquad D = A - 1.
$$

代入 $A$ 的表达式：

$$
A = (1+2A) + \frac{\pi}{4}(A-1) \;\Longrightarrow\;
A = 1 + 2A + \frac{\pi}{4}A - \frac{\pi}{4}.
$$

移项：

$$
A - 2A - \frac{\pi}{4}A = 1 - \frac{\pi}{4}
\;\Longrightarrow\; -A\Bigl(1 + \frac{\pi}{4}\Bigr) = 1 - \frac{\pi}{4}
\;\Longrightarrow\; A = \frac{\frac{\pi}{4} - 1}{1 + \frac{\pi}{4}} = \frac{\pi - 4}{\pi + 4}.
$$

于是 $B = 2A = \dfrac{2(\pi - 4)}{\pi + 4}$。

最后计算所给积分：令 $u = x-1$，则

$$
\int_{0}^{2} f(x-1)\,dx = \int_{-1}^{1} f(u)\,du = B = \frac{2(\pi - 4)}{\pi + 4}.
$$

**答案：**  $\dfrac{2(\pi - 4)}{\pi + 4}$

## 总体框架梳理与核心方法

### 一、极限部分（题 1~5, 10）

反复出现的方法：

- **等价无穷小**：$\sin x \sim x$, $(1+u)^\alpha-1\sim\alpha u$, $\ln(1+u)\sim u$ 等。
- **泰勒展开**：求极限的终极手段，尤其当等价无穷小“抵消”或不足以定阶时（如第 1、3、8 题）。
- **中值定理**：处理两个函数值的差 $\to$ 乘以系数（如第 2 题）。
- **幂指函数**：统一用 $e^{\lim v(u-1)}$。
- **数列极限和式**：强行凑成 $\frac1n\sum g(k/n)$ 的形式，对应定积分 $\int_0^1 g(x)dx$。

### 二、导数与微分部分（题 4~8, 14, 15）

常见技术：

- **单点导数 / 高阶导数**：用定义、泰勒展开或逐次求导。
- **隐函数求导**：方程两边对 $x$ 直接求导。
- **参数方程求导**：一阶、二阶标准公式。
- **反函数求导**：$\frac{dx}{dy}=1/\frac{dy}{dx}$ 及其二阶推广。
- **分段函数可导性**：连续 → 左右导数相等（一阶、二阶）。
- **麦克劳林展开**：通过展开定参数（第 8 题），或由极限反推可导性与导数值（第 15 题）。

### 三、积分部分（题 9, 11~13, 16）

核心操作：

- **不定积分基本方法**：分部积分 $\int udv = uv - \int vdu$（题 9, 11），凑微分。
- **有理函数积分**：部分分式分解（题 13）。
- **三角有理函数的齐次化**：同除 $\cos^kx$ 化出 $\tan x$（题 13）。
- **定积分 / 反常积分**：换元 + 牛顿-莱布尼茨公式（题 12）。
- **积分上限函数的处理**：求导或分部积分（题 11, 15）。
- **积分方程**：将定积分视作常数求解（题 16），得到函数后再进行新的积分计算。

## 延展与提高建议

1. **泰勒展开的灵活运用**  
   很多题（1、3、8、15）如果熟练到能随手写出 $\tan x,\sin x,\ln(1+x)$ 的展开，解题速度会有质的飞跃。建议系统记忆常用函数的麦克劳林级数。
2. **极限中的“反求参数”思维**  
   第 3 题和第 15 题都体现了“已知极限 ⇔ 反求函数的局部信息”这一模式，这在考研和竞赛中是高频考点。
3. **积分上限函数的“解构”**   
   第 11 和 15 题本质上考察的是“被积函数含积分上限函数”时，如何通过分部积分或求导转化为常规积分。理解这一点，能应对更复杂的变限积分问题。
4. **积分方程与对称性化简**  
   第 16 题中，发现 $f(x)$ 为偶函数后，$B=2A$ 直接减少了一个未知量。在含有对称区间的积分方程中一定要优先利用对称性。
5. **计算工具的自检**  
   复杂的参数或反函数二阶导数的结果很容易出错，答题后可以用数值验算（如 $x=2.01$ 差分近似）快速检查符号和量级。

这套试卷完整覆盖了极限、微分、积分的核心计算能力，吃透它，三大计算的基本功就基本扎实了。
