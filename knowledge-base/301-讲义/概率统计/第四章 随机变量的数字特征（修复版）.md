### 二、期望 Expected Value / Mean

#### (一) 离散型随机变量的数学期望

**1、有限型**

若分布律为 $P\{X=x_i\} = p_i \quad (i=1, 2, \cdots, n)$

则 $EX = \sum_{i=1}^{n} x_i p_i$

**2、无限型**

若分布律为 $P\{X=x_i\} = p_i \quad (i=1, 2, 3, \cdots)$

则 $EX = \sum_{i=1}^{\infty} x_i p_i$

3、离散型随机变量函数的数学期望

(1) 一维情形  
若分布律为 $P\{X=x_i\} = p_i \quad (i=1, 2, 3, \cdots)$ , $Y=g(X)$  
则

$$
EY = E[g(X)] = \sum_{i=1}^{\infty} g(x_i) p_i
$$

(2) 二维情形  
若 $(X, Y)$ 的联合分布律为 $P\{X=x_i, Y=y_j\} = p_{ij} \quad (i, j = 1, 2, \cdots)$ , $Z=g(X, Y)$ .  
则

$$
EZ = \sum_{i=1}^{\infty} \sum_{j=1}^{\infty} g(x_i, y_j) p_{ij}
$$

(二) 连续型随机变量的数学期望

1. 定义. 若 $X$ 的概率密度为 $f(x)$ , 则

$$
EX = \int_{-\infty}^{+\infty} x f(x) dx
$$

2. 连续型随机变量函数的数学期望  
   (1) 一维情形  
   若 $X$ 的概率密度为 $f(x)$ , $Y=g(X)$ , 则

$$
EY = E[g(X)] = \int_{-\infty}^{+\infty} g(x) \cdot f(x) dx
$$

(2) 二维情形  
设 $(X, Y)$ 的联合概率密度为 $f(x, y)$ , $Z=g(X, Y)$ , 则

$$
EZ = \int_{-\infty}^{+\infty} \int_{-\infty}^{+\infty} g(x, y) \cdot f(x, y) dx dy
$$

三、方差 Deviation / Variation $\quad DX$

1、定义 $\quad \quad \quad \quad \quad \quad \quad DX = E [ X - EX ] ^ { 2 }$

注： $\quad \quad \quad D X = E \left[ X - E X \right] ^ { 2 } = E \left[ X ^ { 2 } - 2 X E X + ( E X ) ^ { 2 } \right]$  
$\quad \quad \quad = E ( X ^ { 2 } ) - 2 ( E X ) ^ { 2 } + ( E X ) ^ { 2 }$  
$\quad \quad \quad = E ( X ^ { 2 } ) - ( E X ) ^ { 2 } \quad$ 这个公式使用频率更高

2、性质  
$\textcircled { 1 } \quad D ( C ) = 0 \quad ( C \text { 为常数 } )$  
$\textcircled { 2 } \quad D ( X + C ) = D X$  
$\textcircled { 3 } \quad D ( C X ) = C ^ { 2 } D X$  
$\textcircled { 4 } \quad$ 若 $X 、 Y$ 相互独立 ( 或不相关 ), 则 $D ( k _ { 1 } X + k _ { 2 } Y ) = k _ { 1 } ^ { 2 } D X + k _ { 2 } ^ { 2 } D Y$  
$\textcircled { 5 } \quad D ( X \pm Y ) = D X + D Y \pm 2 C o v ( X , Y ) .$

**五、协方差 Covariance Cov(X,Y)**

**1、定义**

$$
Cov(X, Y) = E[(X - EX)(Y - EY)] = E(XY) - EX \cdot EY
$$

**2、性质**  
① $Cov(X, X) = DX$  
② $Cov(X, Y) = Cov(Y, X)$  
③ $Cov(X, a) = 0 \quad (a为常数)$  
④ $Cov(aX, Y) = a Cov(X, Y)$  
$\Rightarrow Cov(aX+b, cY+d) = ac Cov(X, Y)$  
⑤ 若 $X、Y$ 相互独立（或不相关），则 $Cov(X, Y) = 0$

$$

六、相关系数 Correlation $\rho_{XY}$  
1、定义

$$
\rho_{XY} = \frac{\text{Cov}(X, Y)}{\sqrt{DX} \sqrt{DY}} \quad (DX, DY \neq 0)
$$

2、性质  
① $-1 \leqslant \rho_{XY} \leqslant 1$  
② $\rho_{XY}$ 描述的是 $X, Y$ 的线性相关性：  
若 $\rho_{XY} = -1$，则 $X, Y$ 完全负相关；  
若 $\rho_{XY} = 1$，则 $X, Y$ 完全正相关；  
若 $\rho_{XY} = 0$，则 $X, Y$ 不相关。
