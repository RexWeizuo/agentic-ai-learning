# 二、方差

三、方差 Deviation / Variation $\quad DX$

1、定义 $\quad \quad \quad \quad \quad \quad \quad DX = E [ X - EX ] ^ { 2 }$

注： $\quad \quad \quad D X = E \left[ X - E X \right] ^ { 2 } = E \left[ X ^ { 2 } - 2 X E X + ( E X ) ^ { 2 } \right]$  
$\quad \quad \quad = E ( X ^ { 2 } ) - 2 ( E X ) ^ { 2 } + ( E X ) ^ { 2 }$  
$\quad \quad \quad = E ( X ^ { 2 } ) - ( E X ) ^ { 2 } \quad$ 这个公式使用频率更高

2、性质  
$\textcircled { 1 } \quad D ( C ) = 0 \quad ( C \text { 为常数 } )$  
$\textcircled { 2 } \quad D ( X + C ) = D X$  
$\textcircled { 3 } \quad D ( C X ) = C ^ { 2 } D X$  
$\textcircled { 4 } \quad$ 若 $X 、 Y$ 相互独立 ( 或不相关 ), 则 $D ( k _ { 1 } X + k _ { 2 } Y ) = k _ { 1 } ^ { 2 } D X + k _ { 2 } ^ { 2 } D Y$  
$\textcircled { 5 } \quad D ( X \pm Y ) = D X + D Y \pm 2 C o v ( X , Y ) .$
