# 二、中心极限定理

## 三、中心极限定理

设随机变量序列 $X_1, X_2, \cdots$ 独立同分布，且数学期望与方差存在，即 $E(X_i) = \mu, D(X_i) = \sigma^2 \neq 0 \quad (i=1, 2, \cdots)$，则

$$
\lim_{n \to \infty} P\left\{ \frac{\sum_{i=1}^n X_i - n\mu}{\sqrt{n}\sigma} < x \right\} = \frac{1}{\sqrt{2\pi}} \int_{-\infty}^x e^{-\frac{t^2}{2}} dt
$$
