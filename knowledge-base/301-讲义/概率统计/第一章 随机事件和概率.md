# 第一章 随机事件和概率

第一章 随机事件和概率

一、随机事件与样本空间

1、随机试验  
① 可重复性  
② 可能结果不止一个，且所有结果可知  
③ 事先不知道哪个结果会出现  
例．抛硬币，掷骰子 $\cdots$

2、样本空间  
随机试验所有结果组成的集合称为样本空间 $\Omega$，每一个元素称为样本点 $\omega$．  
例．掷骰子所有结果为

$$
\Omega = \{1, 2, 3, 4, 5, 6\}
$$

其中圆圈内的4代表 $\omega$．

3、随机事件  
样本空间的子集称为随机事件．$A, B, C \cdots$  
① 单点集为基本事件．  
例．掷到1点． $A = \{1\}$  
② 多点集为复合事件．  
例．掷到偶数点数． $B = \{2, 4, 6\}$  
③ 两类特殊事件 $\begin{cases} \text{必然事件 } \Omega \\ \text{不可能事件 } \emptyset \end{cases}$．

(二) 随机事件的运算律

(1) 交换律：$A + B = B + A$；$AB = BA$  
(2) 结合律：$(A + B) + C = A + (B + C)$；$(AB)C = A(BC)$  
(3) 分配律：$A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$

$$
A \cap (B \cup C) = (A \cap B) \cup (A \cap C)
$$

(4) 德·摩根律：$\overline{A \cup B} = \overline{A} \cap \overline{B}$；$\overline{A \cap B} = \overline{A} \cup \overline{B}$

三、事件的概率  
(一) 概率的公理化定义  
对于随机试验中的每一事件 $A$，均赋予一个实数 $P(A)$，称其为 $A$ 的概率。集合函数 $P(A)$ 满足以下条件：  
① 非负性：$P(A) \ge 0$；  
② 规范性：$P(\Omega) = 1, P(\phi) = 0$；  
③ 可列可加性：若 $A_1, A_2, \dots, A_n, \dots$ 两两互斥，  
则 $P(A_1 + A_2 + \dots + A_n + \dots) = P(A_1) + P(A_2) + \dots + P(A_n) + \dots$

注：概率的公理化定义界定了概率这个概念所必须满足的一般性质，它没有也不可能解决在特定场合下如何确定概率的问题，这需要大量艰苦的工作。  
它的意义在于为普遍而严格的数理化概率理论奠定了基础，就好比在柯西给出了极限的精确定义后，我们很快建立了宏伟的微积分大厦，有了科尔莫洛夫的公理化定义作为地基，概率论高楼也很快拔地而起。

(二) 概率的性质  
(1) 若 $A \subset B$，则 $P(A) \le P(B)$  
(2) 加法公式：$P(A+B) = P(A) + P(B) - P(AB)$  
(3) 减法公式：$P(A-B) = P(A) - P(AB)$  
(4) $P(\bar{A}) = 1 - P(A)$  
注：$P(A+B+C) = P(A) + P(B) + P(C) - P(AB) - P(AC) - P(BC) + P(ABC)$

 **(四) 几何概型**

(1) 如果一个随机试验的样本空间 $\Omega$ 充满某个区域，其度量(长度、面积或体积等)大小可用 $S_\Omega$ 表示.  
(2) 任意一点落在度量相同的子区域内是等可能的.  
(3) 若事件 $A$ 为 $\Omega$ 中的某个子区域，其度量大小可用 $S_A$ 表示.  
则事件 $A$ 的概率为

$$
P(A) = \frac{S_A}{S_\Omega}
$$

**四、条件概率与独立性**

 **(一) 条件概率**

(1) 定义：设有两个事件 $A, B$，且 $P(B) > 0$，则“在给定 $B$ 发生的条件下 $A$ 发生的概率”记为 $P(A|B)$，定义为

$$
P(A|B) = \frac{P(AB)}{P(B)}
$$

(2) 性质  
① $0 \le P(A|B) \le 1$  
② $P(\Omega|A) = 1, \quad P(\phi|A) = 0$  
③ $P(\bar{B}|A) = 1 - P(B|A)$  
④ $P\{(A_1+A_2)|B\} = P(A_1|B) + P(A_2|B) - P(A_1A_2|B)$

 **(二) 事件的独立性**

若 $P(A) = P(A|B)$，即意味着 $B$ 的发生丝毫不会影响 $A$ 发生的概率，我们称 $A, B$ 两事件独立. 因为 $P(A|B) = P(AB)/P(B)$，我们可以得到 $P(AB) = P(A)P(B) \quad (P(B) > 0)$.

(1) 定义. 设 $A, B$ 为两个事件，若有

$$
P(AB) = P(A)P(B)
$$

则称 $A, B$ 独立.

注：上式比 $P(A) = P(A|B)$ 更好，因为它不受 $P(B)=0$ 影响.

(2) 独立的性质

若 $A, B$ 相互独立，则 $A$ 与 $\bar{B}$，$\bar{A}$ 与 $B$，$\bar{A}$ 与 $\bar{B}$ 也独立.

解释：左手掷骰子，右手抛硬币，显然掷到 1 点不会影响抛到正面，同样的掷不到 1 点也不会影响硬币的正反.

即 $A, B$ 独立意味着 $A$ 发生与否和 $B$ 发生与否毫无关系.

**例**. 已知 $A, B$ 独立，证明：$A, \bar{B}$ 独立.

$$
P(A\bar{B}) = P(A) - P(AB)
$$

$$
= P(A) - P(A)P(B)
$$

$$
= P(A)[1 - P(B)]
$$

$$
= P(A)P(\bar{B})
$$

(3) 三个事件的独立性  
① $A, B, C$ 两两独立

$$

P(A)P(B) = P(AB) \\

五、全概率公式、贝叶斯公式  
(1) 全概率公式  
$A_1, A_2, \cdots, A_n$ 是完备事件组，$P(A_i) > 0, \, i=1, 2, \cdots, n$，则

$$
P(B) = \sum_{i=1}^{n} P(A_i)P(B|A_i)
$$

(2) 贝叶斯公式  
$A_1, A_2, \cdots, A_n$ 是完备事件组，$P(A_i) > 0, \, i=1, 2, \cdots, n$，则

$$
P(A_i|B) = \frac{P(A_i)P(B|A_i)}{\sum_{i=1}^{n} P(A_i)P(B|A_i)} \quad i=1, 2, \cdots, n
$$
