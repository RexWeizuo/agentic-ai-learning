# 一、随机事件与运算

一、随机事件与样本空间

1、随机试验  
① 可重复性  
② 可能结果不止一个，且所有结果可知  
③ 事先不知道哪个结果会出现  
例．抛硬币，掷骰子 $\cdots$

2、样本空间  
随机试验所有结果组成的集合称为样本空间 $\Omega$，每一个元素称为样本点 $\omega$．  
例．掷骰子所有结果为

$$
\Omega = \{1, 2, 3, 4, 5, 6\}
$$

其中圆圈内的4代表 $\omega$．

3、随机事件  
样本空间的子集称为随机事件．$A, B, C \cdots$  
① 单点集为基本事件．  
例．掷到1点． $A = \{1\}$  
② 多点集为复合事件．  
例．掷到偶数点数． $B = \{2, 4, 6\}$  
③ 两类特殊事件 $\begin{cases} \text{必然事件 } \Omega \\ \text{不可能事件 } \emptyset \end{cases}$．

(二) 随机事件的运算律

(1) 交换律：$A + B = B + A$；$AB = BA$  
(2) 结合律：$(A + B) + C = A + (B + C)$；$(AB)C = A(BC)$  
(3) 分配律：$A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$

$$
A \cap (B \cup C) = (A \cap B) \cup (A \cap C)
$$

(4) 德·摩根律：$\overline{A \cup B} = \overline{A} \cap \overline{B}$；$\overline{A \cap B} = \overline{A} \cup \overline{B}$

例1、试表示下列事件  
(1) $A, B, C$ 都发生或者都不发生；  
(2) $A, B, C$ 中至多有两个发生．

例2、设 $A, B$ 和 $C$ 为任意的三事件，则下列选项正确的是（ ）．  
(A) 若 $A + C = B + C$，则 $A = B$  
(B) 若 $A - C = B - C$，则 $A = B$  
(C) 若 $AC = BC$，则 $A = B$  
(D) 若 $AB = \emptyset$，且 $\overline{A}\,\overline{B} = \emptyset$，则 $\overline{A} = B$
