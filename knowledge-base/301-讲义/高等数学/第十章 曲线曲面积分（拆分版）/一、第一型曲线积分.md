# 一、第一型曲线积分

**一、第一型曲线积分**

  **(一) 预备知识：弧长公式**

(图示：弧微分 $ds$ 的几何意义，放大后为一个直角三角形，斜边为 $ds$，两直角边分别为 $dx$ 和 $dy$)

弧微分 $ds = \sqrt{(dx)^2 + (dy)^2}$

① $y = y(x)$

$$
ds = \sqrt{1 + \left(\frac{dy}{dx}\right)^2} \, dx = \sqrt{1 + [y'(x)]^2} \, dx \quad (dx > 0)
$$

则 $s = \int_{a}^{b} \sqrt{1 + [y'(x)]^2} \, dx$，$b > a$。

② $\begin{cases} x = x(t), \\ y = y(t), \end{cases} \quad t \in [\alpha, \beta]$

$$
ds = \sqrt{[x'(t)dt]^2 + [y'(t)dt]^2} = \sqrt{[x'(t)]^2 + [y'(t)]^2} \, dt \quad (dt > 0)
$$

则 $s = \int_{\alpha}^{\beta} \sqrt{[x'(t)]^2 + [y'(t)]^2} \, dt$，$\beta > \alpha$。

③ 极坐标下 $r = r(\theta)$

$$
\begin{cases} x = r(\theta)\cos\theta, \\ y = r(\theta)\sin\theta, \end{cases} \quad \theta \in [\alpha, \beta].
$$

则 $dx = d(r\cos\theta) = (r'\cos\theta - r\sin\theta)d\theta$,

$dy = d(r\sin\theta) = (r'\sin\theta + r\cos\theta)d\theta$,

$(dx)^2 + (dy)^2 = [r^2 + (r')^2](d\theta)^2$,

则 $ds = \sqrt{(dx)^2 + (dy)^2} = \sqrt{[r(\theta)]^2 + [r'(\theta)]^2} \, d\theta$,

故 $s = \int_{\alpha}^{\beta} \sqrt{[r(\theta)]^2 + [r'(\theta)]^2} \, d\theta$.

  **(二) 如何求一段曲线弧状的不均匀质线的质量？(线密度** **$f(x,y)$**​  **)**

$dm = f(x, y) ds$

  **(三) 定义**

$\int_C f(x, y) ds$

  **(四) 性质**

1. **线性**  
   $\int_C [kf(x,y) + lg(x,y)] ds = k\int_C f(x,y) ds + l\int_C g(x,y) ds$
2. **区间可加性** $C = C_1 + C_2$  
   $\int_C f(x,y) ds = \int_{C_1} f(x,y) ds + \int_{C_2} f(x,y) ds$

  **(五) 计算法**

1. $C: y = y(x), x \in [a, b]$.  
   $\int_C f(x, y) ds = \int_a^b f(x, y(x)) \cdot \sqrt{1 + (y')^2} dx$
2. $C: \begin{cases} x = x(t), \\ y = y(t), \end{cases} t \in [\alpha, \beta]$  
   $\int_C f(x, y) ds = \int_\alpha^\beta f(x(t), y(t)) \cdot \sqrt{[x'(t)]^2 + [y'(t)]^2} dt$

若曲面为 $z = z(x, y)$，则可将其看作 $F(x, y, z) = z(x, y) - z = 0$，  
则 $\nabla F = \nabla [z(x, y) - z] = (z_x, z_y, -1)$。  
当然 $(-z_x, -z_y, 1)$ 也是法向量，  
实际选取时要看朝“上”还是朝“下”。  
（在计算曲面积分时很重要）

**例1、**   求曲面 $x^2 + \cos(xy) + yz + x = 0$ 在点 $(0, 1, -1)$ 处的切平面方程。

**例2、**   求曲面 $z = x^2 + \frac{1}{4}y^2 - 1$ 在点 $(-1, -2, 1)$ 处的切平面方程和法线方程。
