# 四、格林公式

**五、格林 (Green) 公式**

  **(一) 连通区域及边界方向**

单连通  
复连通

  **(二) 格林公式**

设 $P(x,y)$, $Q(x,y)$ 在区域 $D$ 上有连续的偏导数，$D$ 的边界为分段光滑的曲线 $C$，则

$$
\oint_{C^+} Pdx + Qdy = \iint_{D} \left( \frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y} \right) dxdy
$$

注：Green 公式对单、复连通区域都适用.

例1、计算 $I = \oint_{C^+} (y-x) dx + (3x-y) dy$，其中曲线 $C$ 为 $x^2 + y^2 - 2x - 8y + 8 = 0$.

### 第 240 页

**例4.**   设曲线积分 $\int_{L} [f(x) - e^x] \sin y dx - f(x) \cos y dy$ 与路径无关, 其中 $f(x)$ 具有一阶连续导数, 且 $f(0)=0$, 则 $f(x)$ 等于 ( )

(A) $\frac{e^{-x}-e^{x}}{2}$  
(B) $\frac{e^{x}-e^{-x}}{2}$  
(C) $\frac{e^{x}+e^{-x}}{2}-1$  
(D) $1-\frac{e^{x}+e^{-x}}{2}$

解: 由题意知, $P = [f(x) - e^x] \sin y$, $Q = -f(x) \cos y$.  
因为积分与路径无关, 所以 $\frac{\partial P}{\partial y} = \frac{\partial Q}{\partial x}$.  
即 $\frac{\partial}{\partial y}([f(x) - e^x] \sin y) = \frac{\partial}{\partial x}(-f(x) \cos y)$  
$[f(x) - e^x] \cos y = -f'(x) \cos y$  
$f(x) - e^x = -f'(x)$  
$f'(x) + f(x) = e^x$

这是一个一阶线性微分方程, 其通解为:  
$f(x) = e^{-\int dx} (\int e^x \cdot e^{\int dx} dx + C) = e^{-x} (\int e^x \cdot e^x dx + C) = e^{-x} (\int e^{2x} dx + C) = e^{-x} (\frac{1}{2}e^{2x} + C) = \frac{1}{2}e^x + Ce^{-x}$

又因为 $f(0)=0$, 所以 $0 = \frac{1}{2} + C \Rightarrow C = -\frac{1}{2}$.  
故 $f(x) = \frac{1}{2}e^x - \frac{1}{2}e^{-x} = \frac{e^x - e^{-x}}{2}$.

答案选 (B).

  **(四) 二元函数的全微分求积**

若 $\frac{\partial Q}{\partial x} = \frac{\partial P}{\partial y}$, 则 $Pdx + Qdy$ 是某个函数 $u(x,y)$ 的全微分.  
求 $u(x,y)$ 的方法:

1. **偏积分法**  
   $u(x,y) = \int P(x,y) dx + \varphi(y)$  
   然后对 $y$ 求偏导, 令其等于 $Q(x,y)$, 解出 $\varphi'(y)$, 再积分得到 $\varphi(y)$.
2. **线积分法 (特殊路径法)**   
   $u(x,y) = \int_{(x_0,y_0)}^{(x,y)} Pdx + Qdy$  
   通常取 $(x_0,y_0)$ 为 $(0,0)$, 并选择平行于坐标轴的折线路径.  
   $u(x,y) = \int_{x_0}^{x} P(x,y_0) dx + \int_{y_0}^{y} Q(x,y) dy$  
   或  
   $u(x,y) = \int_{y_0}^{y} Q(x_0,y) dy + \int_{x_0}^{x} P(x,y) dx$

### 第 242 页

**3. 凑微分法**  
将 $Pdx + Qdy$ 分组, 凑成常见函数的全微分.

常用全微分公式:

- $d(xy) = ydx + xdy$
- $d(\frac{x}{y}) = \frac{ydx - xdy}{y^2}$
- $d(\frac{y}{x}) = \frac{xdy - ydx}{x^2}$
- $d(\arctan\frac{y}{x}) = \frac{xdy - ydx}{x^2+y^2}$
- $d(\ln(x^2+y^2)) = \frac{2xdx + 2ydy}{x^2+y^2}$
- $d(\frac{1}{2}\ln(x^2+y^2)) = \frac{xdx + ydy}{x^2+y^2}$

**例.**   求 $(2x \cos y + y^2 \cos x)dx + (2y \sin x - x^2 \sin y)dy$ 的原函数.  
$u = (2x \cos y dx - x^2 \sin y dy) + (y^2 \cos x dx + 2y \sin x dy)$  
$= d(x^2 \cos y) + d(y^2 \sin x)$  
$= d(x^2 \cos y + y^2 \sin x)$  
所以 $u(x,y) = x^2 \cos y + y^2 \sin x + C$.

**例5.**   已知 $\frac{(x+ay)dx + ydy}{(x+y)^2}$ 为某函数的全微分, 则 $a=$ ( )  
(A) -1  
(B) 0  
(C) 1  
(D) 2

解: $P = \frac{x+ay}{(x+y)^2}$, $Q = \frac{y}{(x+y)^2}$.  
$\frac{\partial P}{\partial y} = \frac{a(x+y)^2 - (x+ay) \cdot 2(x+y)}{(x+y)^4} = \frac{a(x+y) - 2(x+ay)}{(x+y)^3} = \frac{ax+ay-2x-2ay}{(x+y)^3} = \frac{(a-2)x - ay}{(x+y)^3}$  
$\frac{\partial Q}{\partial x} = \frac{0 \cdot (x+y)^2 - y \cdot 2(x+y)}{(x+y)^4} = \frac{-2y(x+y)}{(x+y)^4} = \frac{-2y}{(x+y)^3}$  
令 $\frac{\partial P}{\partial y} = \frac{\partial Q}{\partial x}$, 得:  
$(a-2)x - ay = -2y$  
$(a-2)x + (2-a)y = 0$  
要使上式恒成立, 必须 $a-2=0$, 即 $a=2$.

答案选 (D).

：全微分方程

  **(四) 全微分方程**

- **形式**:

$$
du(x,y) = P(x,y)dx + Q(x,y)dy = 0
$$

- **判定标准**:

$$
\frac{\partial P(x,y)}{\partial y} = \frac{\partial Q(x,y)}{\partial x}
$$

- **通解**:

$$
u(x,y) = C
$$

- **解法**:

$$
\int_{(x_0,y_0)}^{(x,y)} P(x,y)dx + Q(x,y)dy = C
$$

$$
\int_{AC} = \int_{AB} + \int_{BC}
$$

    -   $\overline{AB}$:

$$
\begin{cases} x = t, \\ y = y_0, \end{cases} \quad t: x_0 \to x
$$

$$
\int_{AB} Pdx + Qdy = \int_{x_0}^{x} P(t, y_0)dt
$$

    -   $\overline{BC}$:

$$
\begin{cases} x = x, \\ y = t, \end{cases} \quad t: y_0 \to y
$$

$$
\int_{BC} Pdx + Qdy = \int_{y_0}^{y} Q(x, t)dt
$$

    -   即:

$$
\int_{(x_0,y_0)}^{(x,y)} = \int_{x_0}^{x} P(t, y_0)dt + \int_{y_0}^{y} Q(x, t)dt = C
$$

- **例**: 求解 $(5x^4 + 3xy^2 - y^3)dx + (3x^2y - 3xy^2 + y^2)dy = 0$.

：高斯(Gauss)公式

**六. 高斯 (Gauss) 公式**

  **(一) 散度 (divergence)**

- **定义**:

$$
div\vec{F} \stackrel{def}{=} \nabla \cdot \vec{F} = \frac{\partial P}{\partial x} + \frac{\partial Q}{\partial y} + \frac{\partial R}{\partial z}
$$

- **意义**: 通量的体密度或“源”强度。

  **(二) Gauss 公式**

设 $P, Q, R$ 在空间有界闭区域 $\Omega$ 有连续偏导数，$\Omega$ 的边界 $\Sigma$ 是分片光滑的闭曲面，取外侧为正向，则

$$
\oiint_{\Sigma^+} Pdydz + Qdxdz + Rdxdy = \iiint_{\Omega} (\frac{\partial P}{\partial x} + \frac{\partial Q}{\partial y} + \frac{\partial R}{\partial z}) dV
$$

- **例1**: 计算 $I = \oiint_{\Sigma} x dydz + y dxdz + z dxdy$，其中 $\Sigma$ 为椭球面 $\frac{x^2}{a^2} + \frac{y^2}{b^2} + \frac{z^2}{c^2} = 1$ 的外侧。
