# 二、一阶微分方程

**二、一阶微分方程**

  **(一) 变量可分离型**

形式：$\frac{dy}{dx} = \varphi(x) \cdot \psi(y)$

解法：化为 $\frac{1}{\psi(y)} dy = \varphi(x) dx$

两边积分：$\int \frac{1}{\psi(y)} dy = \int \varphi(x) dx$

例1：$y' = x \sqrt{1 - y^2}$

---第1页---

例2：$\frac{dy}{dx} = 3y$

  **(二) 齐次型**

形式：$\frac{dy}{dx} = g\left(\frac{y}{x}\right)$

解法：设 $u = \frac{y}{x}$，则 $y = ux$

$\frac{dy}{dx} = \frac{du}{dx} \cdot x + u$

$\Rightarrow \frac{du}{dx} \cdot x + u = g(u)$

$\Rightarrow \frac{du}{dx} = \frac{g(u) - u}{x}$ （变量可分离型）

---第2页---

例1：$xy' = y + xe^{\frac{y}{x}}$

例2：$y dx - (x + y \tan\frac{x}{y}) dy = 0$

---第3页---

  **(三) 一阶线性微分方程**

形式：$y' + P(x)y = Q(x)$

解法：$y = e^{-\int P(x) dx} \left( \int Q(x) \cdot e^{\int P(x) dx} dx + C \right)$

例：$x \ln x \, dy + (y - \ln x) dx = 0$

**绝对值问题**

1. $y' + \frac{1}{x} y = \frac{1}{x}$
2. $y' + \frac{1}{2x} y = \frac{1}{x}$

### (三) 线性微分方程解的结构

#### 通解形式：

$$
y = y^* + k_1 y_1 + k_2 y_2 + \cdots + k_n y_n
$$

其中：

- $y^*$ 为非齐次特解
- $y_1, y_2, \ldots, y_n$ 为齐次的 $n$ 个线性无关特解
