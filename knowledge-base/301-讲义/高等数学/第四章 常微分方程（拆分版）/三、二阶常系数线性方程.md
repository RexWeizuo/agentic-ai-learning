# 三、二阶常系数线性方程

#### 1. 齐次

##### 形式：

$$
y'' + py' + qy = 0
$$

##### 特征方程：

$$
r^2 + pr + q = 0 \implies \text{特征根：} r_1, r_2
$$

|特征根情况|通解形式|
| ------------| ----------|
|相异实根 $r_1, r_2$|$k_1 e^{r_1 x} + k_2 e^{r_2 x}$|
|二重根 $r$|$k_1 e^{rx} + k_2 x e^{rx}$|
|共轭复根 $a \pm bi$|$k_1 e^{ax} \cos(bx) + k_2 e^{ax} \sin(bx)$|

#### 例：

(1) $y'' + 5y' + 6y = 0$

### 3. 叠加原理

$$
y'' + py' + qy = f_1(x) + f_2(x)
$$

若 $y_1^*$, $y_2^*$ 分别是方程

$$
y'' + py' + qy = f_1(x) \quad \text{和} \quad y'' + py' + qy = f_2(x)
$$

的特解，则 $y_1^* + y_2^*$ 是原方程的特解。

**例.**   求下列方程的通解：

(1) $y'' - 3y' + 2y = xe^x + \cos x$

(2) $y'' + y = x + \sin x$
