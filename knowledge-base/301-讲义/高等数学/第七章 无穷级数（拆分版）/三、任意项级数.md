# 三、任意项级数

四、任意项级数判敛法

(一) 交错级数

1. 定义

$\sum_{n=1}^{\infty} (-1)^n u_n / \sum_{n=1}^{\infty} (-1)^{n-1} u_n \quad , \quad u_n > 0$

2. 交错级数的 Leibniz 判敛法（充分条件）

定理：若 $\sum_{n=1}^{\infty} (-1)^{n-1} u_n$, $u_n > 0$ 满足

① $\lim_{n \to \infty} u_n = 0$；② $\{u_n\}$ 单调递减。

则级数收敛。

证明：

$S_{2n} = (u_1 - u_2) + (u_3 - u_4) + \cdots + (u_{2n-1} - u_{2n})$

$S_{2n+2} = S_{2n} + (u_{2n+1} - u_{2n+2})$

因为 $u_n \downarrow$，所以 $S_{2n+2} > S_{2n}$，即 $\{S_{2n}\} \uparrow$

$S_{2n} = u_1 - (u_2 - u_3) - (u_4 - u_5) - \cdots - (u_{2n-2} - u_{2n-1}) - u_{2n} < u_1$

由单调有界原则可知 $\lim_{n \to \infty} S_{2n}$ 存在，令 $\lim_{n \to \infty} S_{2n} = S$.

又 $\lim_{n \to \infty} S_{2n-1} = \lim_{n \to \infty} (S_{2n} - u_{2n}) = \lim_{n \to \infty} S_{2n} - \lim_{n \to \infty} u_{2n} = S - 0 = S$

故 $\lim_{n \to \infty} S_n = \sum_{n=1}^{\infty} (-1)^{n-1} u_n = S$.

注：Leibniz条件是充分条件，不是必要条件.

例如 $\sum_{n=2}^{\infty} \frac{(-1)^n}{n + (-1)^n}$ 是收敛的交错级数，而 $\frac{1}{n + (-1)^n}$ ($n \geq 2$) 并没有单调性.

例、判敛

(1) $\sum_{n=1}^{\infty} (-1)^n \frac{1}{n^p}$ ($p > 0$).

(2) $\sum_{n=1}^{\infty} (-1)^{n+1} \frac{1}{\sqrt[n]{n}}$.

(二) 绝对收敛和条件收敛

定义：对任意项级数 $\sum_{n=1}^{\infty} u_n$,

① $\sum_{n=1}^{\infty} |u_n|$ 收敛，称原级数绝对收敛；

② $\sum_{n=1}^{\infty} |u_n|$ 发散，$\sum_{n=1}^{\infty} u_n$ 收敛，称原级数条件收敛.

定理：$\sum_{n=1}^{\infty} |u_n|$ 收敛 $\implies \sum_{n=1}^{\infty} u_n$ 收敛.

证明：

构造 $v_n = \frac{|u_n| + u_n}{2}$, $w_n = \frac{|u_n| - u_n}{2}$

显然 $v_n \geq 0$, 且 $v_n \leq |u_n|$

故 $\sum_{n=1}^{\infty} |u_n|$ 收敛 $\implies \sum_{n=1}^{\infty} v_n$ 收敛.

同理有 $\sum_{n=1}^{\infty} |u_n|$ 收敛 $\implies \sum_{n=1}^{\infty} w_n$ 收敛.

综上 $\sum_{n=1}^{\infty} |u_n|$ 收敛 $\implies \sum_{n=1}^{\infty} (v_n - w_n) = \sum_{n=1}^{\infty} u_n$ 收敛.

例、判敛

(1) $\sum_{n=1}^{\infty} \frac{\sin n}{n^2}$

(2) $\sum_{n=1}^{\infty} (-1)^n \frac{\sqrt{n+1}}{n}$

五、函数项级数

(一) 基本概念

1. $\sum_{n=0}^{\infty} u_n(x)$ 称为函数项级数.
2. 若 $\sum_{n=0}^{\infty} u_n(x_0)$ 收敛，则称 $x = x_0$ 为 $\sum_{n=0}^{\infty} u_n(x)$ 的一个收敛点，反之为发散点.
3. 全体收敛点的集合称为收敛域.
4. 在收敛域上，记 $S(x) = \sum_{n=0}^{\infty} u_n(x)$ 为和函数.

例. 求 $\sum_{n=1}^{\infty} x^{n-1}$ 的收敛域与和函数.
