# 三、平面与直线

（一）平面

如何确定一个平面？

1、过一个点，加一个法向量

[图片：平面 $\pi$ 上一点 $M_0$ 和法向量 $\vec{n}$ 的示意图]

$M_0(x_0, y_0, z_0)$, $M(x, y, z) \in \pi$

$\overrightarrow{M_0M} = (x - x_0, y - y_0, z - z_0)$

$\vec{n} = (A, B, C)$

$\overrightarrow{M_0M} \perp \vec{n}$

(I) $A(x - x_0) + B(y - y_0) + C(z - z_0) = 0$ （点法式）

化简得

(II) $Ax + By + Cz + D = 0$ （一般式）

2、过一个点，平行于两个不共线的向量

[图片：平面由点 $M_0$ 和两个向量 $\vec{\alpha}$, $\vec{\beta}$ 确定的示意图]

$\vec{\alpha} = (a_1, a_2, a_3)$

$\vec{\beta} = (b_1, b_2, b_3)$

$\overrightarrow{M_0M} = (x - x_0, y - y_0, z - z_0)$

$\vec{\alpha}, \vec{\beta}, \overrightarrow{M_0M}$ 共面 $\implies [\vec{\alpha}, \vec{\beta}, \overrightarrow{M_0M}] = 0$

(III) $\begin{vmatrix} a_1 & a_2 & a_3 \\ b_1 & b_2 & b_3 \\ x-x_0 & y-y_0 & z-z_0 \end{vmatrix} = 0$ （标准方程）

3、平面在 $x, y, z$ 轴截距分别为 $a, b, c \quad (abc \neq 0)$

(IV) $\frac{x}{a} + \frac{y}{b} + \frac{z}{c} = 1$ （截距式）

[图片：一个三维坐标系，显示了一个平面与三个坐标轴相交于点 a, b, c，形成一个四面体]

例. 求下列平面方程

(1) 过点 $A(1,1,1)$, $B(2,2,2)$ 且垂直于平面 $x+y-z=4$ 的平面；

(2) 过 $x$ 轴和点 $A(1,3,-2)$ 的平面。

(二) 直线

如何确定一条直线？

1、过一个点，方向确定

[图片：一条穿过点 $M_0(x_0, y_0, z_0)$ 和 $M(x, y, z)$ 的直线，并标出了方向向量 $\vec{s} = (m, n, p)$]

由 $\overrightarrow{M_0M} // \vec{s}$ 可知

(I) $\frac{x - x_0}{m} = \frac{y - y_0}{n} = \frac{z - z_0}{p}$ （标准方程）

(II) $\begin{cases} x = x_0 + mt \\ y = y_0 + nt \\ z = z_0 + pt \end{cases}$ （参数方程）

例. 求直线方程：过点 $(-3, 2, -5)$，与 $x, y, z$ 轴正方向夹角分别为 $\frac{\pi}{3}, \frac{2}{3}\pi, \frac{\pi}{4}$。

2、两平面交线

(III) $\begin{cases} A_1x + B_1y + C_1z + D_1 = 0 \\ A_2x + B_2y + C_2z + D_2 = 0 \end{cases}$ （一般方程）

[图片：两个平面 $\Pi_1$ 和 $\Pi_2$ 相交于一条直线 $L$]

(三) 平面束方程

过两平面交线的平面集合称为平面束方程

$a(A_1x + B_1y + C_1z + D_1) + b(A_2x + B_2y + C_2z + D_2) = 0$

例. 求过平面 $2x-y-2z+1=0$ 与平面 $x+y+4z-2=0$ 的交线，且垂直于平面 $x-3y+z-2=0$ 的平面。
