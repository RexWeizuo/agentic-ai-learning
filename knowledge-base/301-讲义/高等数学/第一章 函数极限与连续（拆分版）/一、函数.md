# 一、函数

一、函数  
1、定义：数集之间的对应关系  
记作：y = f(x), x ∈ D  
因变量 对应法则 自变量 定义域

2、两要素  
① 定义域D (Domain).  
② 对应法则f (function)：“一对一”或“多对一”.

例、y = f(x) = 3x + 2, x ∈ R.  
此函数的功能是将实数域中任一数x变成3x+2.  
(“变换”的思想至关重要).

例、狄利克雷(Dirichlet)函数.  
D(x) = { 1, x∈有理数,  
       { 0, x∈无理数.

例1. 画出取整函数 [x] 在 [-2,2] 上的图像.

例2. 已知 $f(x + 1/x) = (x+x^3)/(1+x^4)$, 求$f(x)$的表达式.

例3. 设函数f(x)满足  
$2f(x) + x^2f(1/x) = (x^2+2x)/\sqrt{1+x^2}$, $x > 0$.  
求f(x)表达式.

3、函数特性  
∃: Exist 存在  
∀: Arbitrary 任意  
(1) 单调性  
def. ∀x₁, x₂ ∈ D, 且 x₁ < x₂.  
① 若 f(x₁) < f(x₂), 则 f(x) 单调递增;  
② 若 f(x₁) > f(x₂), 则 f(x) 单调递减.  
注: 若 f(x₁) ≤ f(x₂), 则 f(x) “单调不减”;  
    若 f(x₁) ≥ f(x₂), 则 f(x) “单调不增”.

(2) 奇偶性 (D为对称区间).  
def. ① 若 ∀x ∈ D, 有 f(x) = f(-x), 则称 y = f(x) 为偶函数.  
例. y = x², |x|, cosx ...  
[图片：一个关于y轴对称的抛物线形状的图]  
② 若 ∀x ∈ D, 有 f(x) = -f(-x), 则称 y = f(x) 为奇函数.  
例. y = x, x³, sinx ...  
[图片：一个关于原点对称的曲线图]  
注: (1) 奇·奇 = 偶;  
        奇·偶 = 奇;  
        偶·偶 = 偶.  
    (2) f(x) + f(-x) 必为偶;  
        f(x) - f(-x) 必为奇;  
        f(x) = 1/2[f(x)+f(-x)] + 1/2[f(x)-f(-x)].

(3) 有界性  
def. ∃m, M ∈ R, 对 ∀x ∈ D, 都有 m ≤ f(x) ≤ M, 则称 f(x) 有界.  
(直观来说, 就是f(x)可被“夹板”夹住.)  
[图片：一条在两条水平虚线之间波动的曲线]  
注: 界不唯一.

(4) 周期性  
def. 若 f(x) = f(x+T), 则称 f(x) 为周期函数. T > 0.  
思考: ① T唯一吗?  
      ② 一定存在最小正周期吗?

例、若f(x)为定义在(-∞,+∞)上的偶函数, 且图形关于x=2对称, 则f(x)一定为周期函数, 其周期 T = ____.

**4、函数运算**

(1) 四则运算. $f+g$, $f-g$, $f \cdot g$, $f/g$.

(2) 复合运算. $y = f[g(x)]$.  
例如 $y = 2^{\tan3x}$ 由 $y = 2^u$, $u = \tan v$, $v = 3x$ 复合而成.

例. 设 $f(x) = \begin{cases} 1, & |x| \le 1, \\ 0, & |x| > 1, \end{cases}$ 求 $f[f(x)]$.

**复合函数奇偶性** $\quad$ **不同则偶，相同则同**

(1) 若 $y = f(t)$, $t = g(x)$ 奇偶性不同，则 $y = f[g(x)]$ 为偶函数；  
(2) 若 $y = f(t)$, $t = g(x)$ 奇偶性相同，则 $y = f[g(x)]$ 奇偶性与 $f(x)$ 相同.

**5、反函数**

一一对应函数 $y=f(x)$ 在其值域上可定义反函数.

$$
y = f(x) \iff x = f^{-1}(y)
$$

例如 $y=x^3$ 的反函数为 $x=y^{1/3}$.

引申：$y=f(x)$ 与 $y=f^{-1}(x)$ 的图像关于 $y=x$ 对称.

**6、隐函数**

定义：由方程 $F(x,y)=0$ 确定的函数.

例如 $x^2+y^2=9$ 在 $\begin{cases} y \ge 0 \text{时确定 } y = \sqrt{9-x^2} \\ y < 0 \text{时确定 } y = -\sqrt{9-x^2} \end{cases}$

函数其实就是对应关系，像 $y=f(x)=3x$ 这种显而易见的对应关系称为显函数，而“隐藏”在方程里的对应关系就是隐函数，有时可以根据 $F(x,y)=0$ 写出显函数，有时则不可以.  
例如 $2x-y = 2\arctan(y-x)$.

**7、参数方程表示的函数**

$$
\begin{cases} x = x(t) \\ y = y(t) \end{cases}
$$

**8、极坐标表示的函数**  
$r = r(\theta)$  
（此处有一幅极坐标示意图）

极坐标与直角坐标转换关系：$\begin{cases} x = r\cos\theta \\ y = r\sin\theta \end{cases}$

注：7、8例子见导学段讲义.
