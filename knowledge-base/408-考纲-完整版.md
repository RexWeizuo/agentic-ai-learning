# 一、数据结构（45 分）

### 1. 基本概念
- 数据结构的基本概念、算法的基本概念

### 2. 线性表
- 线性表的基本概念、顺序存储、链式存储、应用

### 3. 栈、队列和数组
- 栈和队列的基本概念、顺序/链式存储结构
- 多维数组的存储、特殊矩阵的压缩存储
- 栈、队列和数组的应用

### 4. 树和二叉树（★重点）
- 树的基本概念
- 二叉树的定义、主要特性、存储结构、遍历
- 线索二叉树的基本概念和构造
- 树和森林的存储、与二叉树的转换、遍历
- 哈夫曼树和编码、并查集、堆

### 5. 图
- 图的基本概念
- 存储：邻接矩阵、邻接表、邻接多重表、十字链表
- 遍历：DFS、BFS
- 应用：最小生成树、最短路径、拓扑排序、关键路径

### 6. 查找（★重点）
- 顺序查找、分块查找、折半查找
- 树型查找：二叉搜索树、平衡二叉树、红黑树
- B 树及其基本操作、B+ 树的基本概念
- 散列(Hash)表
- 字符串模式匹配
- 查找算法的分析及应用

### 7. 排序
- 插入排序（直接/折半）、冒泡排序
- 简单选择排序、希尔排序、快速排序
- 堆排序、二路归并排序、基数排序
- 外部排序
- 排序算法的分析与应用

# 二、计算机组成原理（45 分）

### 1. 计算机系统概述
- 计算机系统层次结构、硬件/软件基本组成
- "存储程序"工作方式
- 性能指标：吞吐量、响应时间、CPI、MIPS、FLOPS

### 2. 数据的表示和运算
- 进位计数制及转换
- 定点数的编码表示
- 基本运算部件（加法器、ALU）
- 补码加/减运算、标志位生成
- 乘/除法运算原理
- 整数（无符号/带符号）表示和运算
- IEEE754 浮点数标准、浮点数加减运算

### 3. 存储器层次结构（★重点）
- 存储器分类、层次化结构
- SRAM / DRAM / Flash 存储器
- 主存储器（DRAM 芯片、多模块存储器）
- 外部存储器（磁盘、SSD）
- Cache：基本原理、映射方式（直接/全相联/组相联）、替换算法、写策略
- 虚拟存储器：页式（页表、TLB）、段式、段页式

### 4. 指令系统
- 指令格式、寻址方式
- 数据对齐和大/小端存放
- CISC vs RISC
- 高级语言与机器代码的对应

### 5. 中央处理器 CPU（★重点）
- CPU 功能和基本结构、指令执行过程
- 数据通路、控制器（硬布线/微程序）
- 异常和中断机制
- 指令流水线：结构冒险、数据冒险、控制冒险
- 超标量和动态流水线
- 多处理器：SISD/SIMD/MIMD、硬件多线程、多核处理器、SMP

### 6. 总线和 I/O 系统
- 总线概念、组成、性能指标、事务和定时
- I/O 接口功能和结构
- I/O 方式：程序查询、程序中断、DMA

# 三、操作系统（35 分）

### 1. 操作系统基础
- 基本概念、发展历程
- CPU 运行模式（内核/用户模式）、中断/异常、系统调用
- 程序链接与装入、内存映像与地址空间
- 操作系统结构：分层、模块化、宏内核、微内核、外核
- 操作系统引导、虚拟机

### 2. 进程管理（★重点）
- 进程与线程：概念、状态转换、线程实现（内核级/用户级）
- 进程间通信：共享内存、消息传递、管道、信号
- CPU 调度：调度算法（FCFS/SJF/RR/优先级/多级反馈队列）、上下文切换
- 同步与互斥：软件/硬件方法、锁、信号量、条件变量
- 经典同步问题：生产者-消费者、读者-写者、哲学家进餐
- 死锁：预防、避免（银行家算法）、检测和解除

### 3. 内存管理（★重点）
- 逻辑/物理地址空间、地址变换、内存保护
- 连续分配、页式管理、段式管理、段页式管理
- 虚拟内存：请求页式管理、页置换算法（OPT/FIFO/LRU/CLOCK/改进CLOCK）
- 内存映射文件、抖动、工作集
- 写时复制（Copy-on-Write）

### 4. 文件管理
- 文件概念、inode、文件操作
- 文件逻辑/物理结构
- 目录：树形目录、硬链接/软链接
- 文件系统全局结构、空闲空间管理
- 虚拟文件系统、文件系统挂载

### 5. I/O 管理
- 设备分类、I/O 控制方式（轮询/中断/DMA）
- I/O 软件层次结构
- 缓冲区管理、SPOOLing
- 磁盘调度（FCFS/SSTF/SCAN/C-SCAN）、固态硬盘

# 四、计算机网络（25 分）

### 1. 计算机网络概述
- 网络定义、组成、分类、性能指标
- 分层结构、协议/接口/服务
- ISO/OSI 参考模型和 TCP/IP 参考模型

### 2. 物理层
- 信道、信号、带宽、码元、波特、速率
- 奈奎斯特定理与香农定理
- 编码与调制
- 电路交换/报文交换/分组交换、数据报与虚电路
- 传输介质、物理层设备

### 3. 数据链路层
- 组帧、差错控制
- 流量控制与可靠传输：停止-等待、GBN、选择重传
- 介质访问控制：信道划分、随机访问（ALOHA/CSMA/CSMA-CD/CSMA-CA）
- 局域网：以太网 IEEE802.3、WLAN IEEE 802.11、VLAN
- 广域网、PPP 协议
- 以太网交换机

### 4. 网络层（★重点）
- 异构网络互连、路由与转发、SDN
- 路由算法：距离-向量、链路状态、层次路由
- IPv4：分组、地址、NAT、子网划分、CIDR、ARP、DHCP、ICMP
- IPv6：主要特点、地址
- 路由协议：RIP、OSPF、BGP
- IP 组播、移动 IP
- 路由器组成与路由表

### 5. 传输层（★重点）
- 传输层功能、端口、无连接/面向连接
- UDP：数据段、校验
- TCP：段格式、三次握手/四次挥手、可靠传输、流量控制、拥塞控制

### 6. 应用层
- C/S 模型、P2P 模型
- DNS：层次域名空间、域名服务器、解析过程
- FTP、电子邮件（SMTP/POP3/MIME）
- WWW：HTTP 协议
